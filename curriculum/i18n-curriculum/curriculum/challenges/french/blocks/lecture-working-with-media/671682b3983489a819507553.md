# --interactive--

Dans une leçon précédente, tu as découvert l'élément `iframe`. Dans cette leçon, tu vas apprendre à mieux l'utiliser. Cet élément signifie inline frame. C'est un élément en ligne utilisé pour intégrer directement d'autre contenu HTML dans la page HTML. Ce contenu HTML peut être une vidéo, une carte, un autre élément HTML ou même d'autres pages web.

Voici un exemple d'intégration d'un cours freeCodeCamp populaire depuis YouTube. Pour voir une autre vidéo, active l'éditeur interactif et change la valeur de `src` avec la vidéo de ton choix.

:::interactive_editor

```html
<iframe
  width="400"
  height="400"
  src="https://www.youtube.com/embed/PkZNo7MFNFg?si=-UBVIUNM3csdeiWF"
  title="Learn JavaScript - Full Course for Beginners (YouTube video)"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
  referrerpolicy="strict-origin-when-cross-origin"
  allowfullscreen
></iframe>
```

:::

L'attribut `src` indique l'URL de la page que tu veux intégrer. L'attribut `width` indique la largeur de l'`iframe`. L'attribut `height` indique la hauteur de l'`iframe`. L'attribut `allowfullscreen` permet à l'utilisateur d'afficher l'`iframe` en mode plein écran. Il est aussi recommandé d'ajouter un attribut `title` à l'`iframe`, car c'est important pour l'accessibilité.

L'attribut `allow`, lui, te permet de définir ce qu'une iframe peut ou ne peut pas faire. On appelle cela une liste d'autorisations. Dans l'exemple ci-dessus, ajouter `clipboard-write` lui permet d'écrire des éléments dans ton presse-papiers. Les éléments d'une liste d'autorisations peuvent être séparés par des points-virgules ou des espaces, et les deux peuvent être utilisés ensemble.

Note que la vidéo peut venir de n'importe où. Elle n'a pas besoin de venir de services vidéo comme YouTube ou Vimeo.

N'oublie pas que tu peux aussi intégrer une carte, une autre page web ou du HTML direct dans l'élément `iframe`. Voici un exemple de carte intégrée.

Essaie d'interagir avec la carte en zoomant et dézoomant.

:::interactive_editor

```html
<h1>A Map from Openstreetmap.org Embedded with the iframe Element</h1>

<iframe
  width="425"
  height="350"
  src="https://www.openstreetmap.org/export/embed.html?bbox=3.006134033203125%2C6.150112578753815%2C3.6357879638671875%2C6.749850810550778&amp;layer=mapnik"
  title="Map of Lagos area, Nigeria"
  style="border: 1px solid black"
>
</iframe>
<br />
<small>
  <a href="https://www.openstreetmap.org/#map=11/6.4501/3.3210">
    View Larger Map
  </a>
</small>
```

:::

Si tu veux intégrer du HTML direct dans l'élément `iframe`, tu dois utiliser l'attribut `srcdoc` au lieu de `src`.
