# --interactive--

Quand tu construis des sites web multilingues ou que tu manipules des attributs de données personnalisés, tu as souvent besoin de styliser des éléments selon la langue qu'ils contiennent ou selon des valeurs de données précises.

Les attributs `lang` et `data-lang` sont couramment utilisés pour ces objectifs, et le sélecteur d'attribut en CSS te permet d'appliquer efficacement des styles selon ces attributs.

L'attribut `lang` est utilisé en HTML pour préciser la langue du contenu à l'intérieur d'un élément. Tu peux vouloir styliser des éléments différemment selon la langue dans laquelle ils sont écrits, surtout sur un site web multilingue.

Voici un exemple d'utilisation du sélecteur d'attribut pour cibler des éléments avec un attribut `lang` précis :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">This is an English paragraph.</p>
<p lang="fr">Ceci est un paragraphe en français.</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

Cette règle CSS applique un style italique à tout élément paragraphe dont l'attribut `lang` est défini sur anglais (`en`). Cela peut être utile pour mettre en évidence du texte anglais dans un document qui inclut plusieurs langues.

De la même façon, tu peux utiliser le sélecteur d'attribut pour cibler des éléments avec un attribut `data-lang`.

Les attributs de données personnalisés comme `data-lang` sont couramment utilisés pour stocker des informations supplémentaires dans des éléments, par exemple pour préciser la langue utilisée dans une section précise de texte.

Voici comment utiliser CSS pour cibler et styliser des éléments selon l'attribut `data-lang` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p data-lang="fr">Ceci est un paragraphe en français.</p>
<p data-lang="en">This is a paragraph in English.</p>
```

```css
p[data-lang="fr"] {
  color: blue;
}
```

:::

Dans ce cas, tout élément `p` avec un attribut `data-lang` défini sur français (`fr`) verra la couleur de son texte changée en bleu. Cela te permet d'identifier et de styliser rapidement des sections de contenu selon les informations de langue stockées dans l'attribut `data-lang`.

Les sélecteurs d'attribut comme ceux-ci fournissent une façon puissante d'appliquer un style conditionnel selon les métadonnées intégrées dans ton HTML, ce qui rend tes pages web plus dynamiques et sensibles au contexte.
