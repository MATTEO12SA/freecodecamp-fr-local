# --interactive--

Quand tu travailles avec des listes ordonnées en HTML, l'attribut `type` te permet de préciser le style de numérotation utilisé, comme une numérotation numérique, alphabétique ou en chiffres romains.

L'attribut `type` peut être défini sur un élément `ol` (liste ordonnée) pour contrôler le format de numérotation :

- `1` pour les listes numériques (comme 1, 2, 3, ...),

- `A` pour les listes alphabétiques en majuscules (comme A, B, C, ...),

- `a` pour les listes alphabétiques en minuscules (comme a, b, c, ...),

- `I` pour les chiffres romains en majuscules (comme I, II, III, ...),

- `i` pour les chiffres romains en minuscules (comme i, ii, iii, ...).

Voici comment utiliser le sélecteur d'attribut pour cibler une liste ordonnée avec un attribut `type` précis :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="A"] {
  color: purple;
  font-weight: bold;
}
```

:::

Dans cet exemple, la règle CSS applique un texte violet et une graisse de police en gras à toute liste ordonnée qui utilise une numérotation alphabétique en majuscules, ce qui est précisé par `type="A"`.

De même, tu peux cibler les listes ordonnées qui utilisent des chiffres romains en minuscules en sélectionnant celles avec `type="i"` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="i">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="i"] {
  color: green;
}
```

:::

Cette règle change la couleur du texte en vert pour toute liste ordonnée qui utilise des chiffres romains en minuscules.

La possibilité de styliser les listes ordonnées selon leur attribut `type` te donne plus de contrôle sur l'apparence des listes dans ton document, ce qui permet de personnaliser leur rendu selon la structure du contenu.
