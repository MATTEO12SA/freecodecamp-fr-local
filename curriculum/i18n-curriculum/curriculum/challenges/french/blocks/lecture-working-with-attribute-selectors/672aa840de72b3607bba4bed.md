# --interactive--

Le sélecteur d'attribut en CSS est un outil puissant qui te permet de cibler des éléments HTML en fonction de leurs attributs. Cela signifie que tu peux appliquer des styles aux éléments qui ont des attributs précis, ou même certaines valeurs pour ces attributs.

Il est particulièrement utile quand tu veux styliser des éléments de manière dynamique, ou quand les noms de classes seuls ne donnent pas assez de précision.

Par exemple, tu peux utiliser le sélecteur d'attribut pour cibler tous les liens qui ont un attribut `href`. C'est particulièrement pratique quand tu veux appliquer un style uniforme à tous les liens d'une page.

Voici comment utiliser le sélecteur d'attribut pour cibler les liens avec l'attribut `href` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Example link with an href attribute</a>
<a>Example link without an href attribute</a>
```

```css
a {
  display: block;
}

a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

Cette règle CSS applique une couleur bleue et un soulignement à tout lien qui inclut un attribut `href`, ce qui garantit que tous les liens cliquables sont stylisés de manière cohérente.

Mais les sélecteurs d'attribut peuvent être plus précis. Supposons que tu veuilles cibler uniquement les liens qui ont un attribut `title`. L'attribut `title` fournit souvent des informations supplémentaires sur le lien, et tu peux vouloir styliser ces liens différemment pour indiquer qu'ils disposent d'informations supplémentaires. Voici comment faire :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

Cette règle applique un texte en gras et retire le soulignement à tout lien avec un attribut `title`. C'est une bonne façon de différencier visuellement ces liens des autres liens de la page.

Tu peux aussi combiner des sélecteurs d'attribut pour être encore plus précis. Par exemple, si tu veux styliser les liens qui ont à la fois les attributs `href` et `title`, tu peux faire ceci :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[href][title] {
  display:block;
  color: green;
}
```

:::

Dans ce cas, seuls les liens qui contiennent à la fois les attributs `href` et `title` seront stylisés avec un texte vert. Ce niveau de contrôle est ce qui rend les sélecteurs d'attribut si puissants en CSS.

Un autre exemple de sélecteur d'attribut consiste à faire correspondre une seule valeur dans une liste de valeurs séparées par des espaces dans un attribut.

Voici un exemple d'élément d'ancre avec plusieurs classes :

```html
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

Pour cibler cet élément d'ancre précis, tu peux utiliser le sélecteur suivant :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

La syntaxe `[attr~=value]` est utilisée ici pour cibler tous les éléments d'ancre dont l'attribut `class` contient le mot `"primary"`.

Si tu dois cibler un élément dont la valeur d'attribut commence par une valeur précise, tu peux utiliser la syntaxe `[attr^=value]`. Voici un exemple :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

Dans cet exemple, le sélecteur `a[href^="https://"]` cible tous les éléments d'ancre dont la valeur de l'attribut `href` commence par `"https://"`.

Pour cibler des éléments dont la valeur d'attribut se termine par une valeur précise, tu peux utiliser la syntaxe `[attr$=value]`. Voici un exemple :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Visit Example Site</a>
```

```css
a[href$=".com"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

Dans cet exemple, le sélecteur `a[href$=".com"]` cible tous les éléments d'ancre dont la valeur de l'attribut `href` se termine par `.com`.

Utiliser les sélecteurs d'attribut améliore non seulement le style de ta page web, mais aussi l'accessibilité, en rendant les éléments interactifs comme les liens plus faciles à distinguer selon leurs attributs.
