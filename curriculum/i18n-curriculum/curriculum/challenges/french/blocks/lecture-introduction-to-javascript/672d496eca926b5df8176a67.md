# --interactive--

En JavaScript, un type de donnée correspond au genre de valeur que tu stockes, comme un nombre ou un morceau de texte.

Une variable est un conteneur nommé qui stocke une valeur d'un type de donnée précis, ce qui te permet de la référencer et de la manipuler dans tout ton code.

Tu te souviens peut-être d'avoir travaillé avec des variables comme celles-ci en cours de mathématiques :

```md
x = 2
y = 4

x + y
```

Tu pouvais créer des variables comme `x` et `y`, puis les réutiliser dans ton programme et faire des opérations mathématiques comme une addition. C'est un concept similaire en programmation. Tu peux créer tes propres noms de variables et leur affecter des valeurs. Ces valeurs auront différents types de données.

Les types de données aident le programme à comprendre le genre de données avec lesquelles il travaille, qu'il s'agisse d'un nombre, de texte ou d'autre chose.

JavaScript possède plusieurs types de données de base que tu utiliseras dans tes programmes. Nous explorerons chaque type de donnée plus en détail dans les prochaines leçons. Pour l'instant, voici une brève introduction aux différents types de données en JavaScript.

Le premier type de donnée que nous allons voir est le type `Number`.

Un `Number` représente à la fois les entiers et les nombres à virgule flottante. Exemples d'entiers : `7`, `19` et `90`.

**REMARQUE** : `console.log()` est une fonction qui affiche des informations dans la console, une partie de ton navigateur web utilisée pour déboguer du code. Tu en apprendras plus sur `console.log()` dans les prochaines leçons. Les symboles `//` servent aussi à ajouter des commentaires dans ton code. Les commentaires sont des notes pour toi-même ou pour d'autres programmeurs, et elles sont ignorées quand le code s'exécute.

Active l'éditeur interactif et essaie de modifier certains entiers pour voir la console se mettre à jour.

:::interactive_editor

```js
// Examples of integers
console.log(3);
console.log(5);
console.log(-67);
```

:::

Un nombre à virgule flottante est un nombre avec une partie décimale. Exemples de nombres à virgule flottante : `3.14` et `5.2`.

:::interactive_editor

```js
// Examples of floating point numbers
console.log(3.14);
console.log(7.2);
console.log(-14.5);
```

:::

Le type de donnée suivant est `String`.

Une `String` est une suite de caractères, ou de texte, entourée de guillemets. Voici un exemple de chaîne avec des guillemets doubles :

:::interactive_editor

```js
console.log("I love to code!");
```

:::

Voici un exemple avec des guillemets simples :

:::interactive_editor

```js
console.log('I love to code!');
```

:::

Tu utiliseras souvent les chaînes pour représenter des noms, des libellés, des messages d'alerte, etc.

Un autre type de donnée utilisé en JavaScript est le type `Boolean`.

Un `Boolean` représente l'une de deux valeurs : `true` ou `false`. Par exemple, un programme peut vérifier si un utilisateur est connecté (`true`) ou non (`false`) et modifier la page en fonction de cela. Si l'utilisateur est connecté, tu veux probablement lui afficher le tableau de bord. Sinon, tu voudras lui afficher la page de connexion.

Les deux types de données suivants utilisés en JavaScript sont `undefined` et `null`.

`undefined` signifie qu'une variable a été déclarée mais qu'aucune valeur ne lui a encore été donnée. Tu en apprendras plus à ce sujet dans la prochaine leçon.

`null` signifie que la variable a été volontairement définie sur « rien » et ne contient aucune valeur. Nous explorerons plus tard son fonctionnement.

Les trois derniers types de données sont plus complexes par nature. Ce sont les objets, les symboles et `BigInt`.

Un `Object` est une collection de paires clé-valeur.

```js
{
  name: "Alice",
  age: 30
};
```

Les objets sont pratiques pour regrouper des informations liées. Tu apprendras à travailler avec les objets dans un module futur.

Un `Symbol` est un type spécial de valeur en JavaScript qui est toujours unique et ne peut pas être modifié. Il est souvent utilisé pour créer des étiquettes ou des identifiants uniques pour des propriétés :

```js
Symbol('mySymbol');
```

`BigInt` sert aux très grands nombres qui dépassent la limite du type `Number` :

```js
1234567890123456789012345678901234567890n;
```

Dans cet exemple, nous créons un `BigInt` en ajoutant `n` à la fin d'un très grand nombre.

`Symbol` et `BigInt` sont deux types moins couramment utilisés, mais il est quand même important de les connaître.

Comprendre ces types de données t'aide à manipuler et utiliser différents genres de données dans tes programmes, car chaque type a ses propres caractéristiques et comportements.
