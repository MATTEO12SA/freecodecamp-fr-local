# --interactive--

JavaScript est un langage de programmation puissant qui ajoute de l'interactivité et des comportements dynamiques aux sites web.

HTML et CSS sont des langages de balisage utilisés pour structurer le contenu et styliser les éléments d'une page. JavaScript va plus loin en permettant des fonctionnalités plus complexes, comme gérer les saisies utilisateur, animer des éléments et même construire des applications web complètes.

Par exemple, quand tu cliques sur un bouton, envoies un formulaire ou survoles un menu, JavaScript détermine le comportement de la page.

Voici un exemple de la façon dont ces trois langages fonctionnent ensemble :

:::interactive_editor

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

:::

Dans cet exemple, HTML sert à définir le contenu : un titre (un élément `h1`) et un bouton (l'élément `button`).

CSS sert à appliquer des styles au titre, par exemple rendre le texte vert. JavaScript sert à afficher un message d'alerte quand le bouton est cliqué.

Pour résumer, HTML fournit la structure, CSS ajoute le style et JavaScript active le comportement, ce qui crée une expérience web complète et interactive.
