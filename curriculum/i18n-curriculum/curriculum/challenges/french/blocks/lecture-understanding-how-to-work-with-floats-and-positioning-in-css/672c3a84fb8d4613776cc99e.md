# --interactive--

En CSS, le positionnement nous permet de contrôler la façon dont les éléments sont disposés sur une page. Deux types courants de positionnement sont le positionnement statique et le positionnement relatif. Par défaut, les éléments sont positionnés de manière statique. Cela signifie qu'ils suivent le flux normal du document, les uns après les autres, de haut en bas et de gauche à droite.

Le positionnement statique est le comportement par défaut de tous les éléments et ne nécessite aucune déclaration CSS particulière. Tu ne remarqueras rien de différent en utilisant le positionnement statique, car il garde simplement les éléments à l'endroit où ils apparaissent naturellement dans le document. Voici un exemple de paragraphe qui utilise le positionnement statique par défaut :

:::interactive_editor

```html
<p>This paragraph is statically positioned.</p>
```

:::

Lorsqu'il est affiché dans le navigateur, ce paragraphe apparaît à sa position naturelle, en suivant le flux normal du document. Aucun style particulier n'est appliqué pour le déplacer, et il reste à l'endroit où il apparaîtrait naturellement dans la mise en page.

Le positionnement relatif, lui, permet de décaler un élément par rapport à sa position normale sans perturber le flux du document. Tu peux le voir comme un déplacement de l'élément depuis sa position statique par défaut en lui donnant de nouvelles coordonnées. Voici comment appliquer un positionnement relatif :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="relative">This paragraph is positioned relatively.</p>
```

```css
body {
  border: solid 1px black;
}

.relative {
  position: relative;
  top: 30px;
  left: 30px;
}
```

:::

Dans cet exemple, le paragraphe apparaît `30px` plus bas et `30px` plus à droite que sa position d'origine. Cependant, l'espace qu'il aurait occupé dans le flux normal reste réservé. Le positionnement relatif est très utile quand tu veux déplacer légèrement un élément sans affecter le reste de la mise en page.
