# --interactive--

La propriété `z-index` en CSS sert à contrôler l'ordre d'empilement vertical des éléments positionnés qui se chevauchent sur la page. Quand plusieurs éléments sont empilés les uns sur les autres, la valeur de `z-index` détermine quel élément apparaît au-dessus. Plus la valeur de `z-index` est élevée, plus l'élément est proche de l'utilisateur, tandis que les valeurs plus faibles placent l'élément plus loin dans la pile.

Cependant, `z-index` ne fonctionne que sur les éléments qui sont positionnés, ce qui signifie que l'élément doit avoir une valeur de `position` autre que `static`, comme `relative`, `absolute` ou `fixed`. La valeur par défaut de `z-index` est `auto`, ce qui place l'élément dans l'ordre d'empilement par défaut. Voyons un exemple avec trois boîtes :

```html
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

Maintenant, nous pouvons appliquer quelques styles pour positionner les boîtes afin qu'elles se chevauchent comme ceci :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box1 {
  position: absolute;
  z-index: 1;
  background: lightcoral;
  top: 20px;
  left: 20px;
  width: 100px;
  height: 100px;
}

.box2 {
  position: absolute;
  z-index: 3;
  background: gold;
  top: 40px;
  left: 40px;
  width: 100px;
  height: 100px;
}

.box3 {
  position: absolute;
  z-index: 2;
  background: lightgreen;
  top: 60px;
  left: 60px;
  width: 100px;
  height: 100px;
}
```

:::

Pour le conteneur, le positionnement sera défini sur `relative`, et toutes les boîtes imbriquées à l'intérieur seront définies en positionnement `absolute`. Chaque boîte a une valeur différente pour `z-index`, ce qui fait que les boîtes sont superposées les unes au-dessus des autres.

Tu peux voir `z-index` comme une façon de créer des couches sur une page web, et les éléments avec des valeurs de `z-index` plus élevées seront placés au-dessus de ceux qui ont des valeurs plus faibles. C'est particulièrement utile pour contrôler le comportement des éléments qui se chevauchent dans des mises en page complexes, comme des modales, des pop-ups ou des infobulles.
