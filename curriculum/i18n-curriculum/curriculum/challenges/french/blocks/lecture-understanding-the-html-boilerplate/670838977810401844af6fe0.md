# --description--

Découvrons l'élément `link` et la façon de l'utiliser pour lier des feuilles de style externes.

L'élément `link` sert à lier des ressources externes comme des feuilles de style et des icônes de site. Voici la syntaxe de base pour utiliser l'élément `link` avec un fichier CSS externe :

```html
<link rel="stylesheet" href="./styles.css" />
```

L'attribut `rel` sert à indiquer la relation entre la ressource liée et le document HTML. Dans cette situation, nous devons préciser que cette ressource liée est une `stylesheet`.

Il est considéré comme une bonne pratique de séparer ton HTML et ton CSS dans des fichiers différents. Les développeurs utilisent l'élément `link` pour leur fichier CSS externe au lieu de tout écrire directement dans le document HTML.

L'attribut `href` sert à indiquer l'emplacement de l'URL de la ressource externe.

Le `dot` suivi d'une barre oblique dans l'exemple indique à l'ordinateur de chercher le fichier `styles.css` dans le dossier, ou répertoire, actuel.

L'élément `link` doit être placé dans l'élément `head`, comme dans l'exemple suivant :

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Examples of the link element</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

Dans une base de code professionnelle, tu verras souvent plusieurs éléments `link` dans le `head`, qui lient différentes feuilles de style, polices et icônes. Voici un exemple qui utilise l'élément `link` pour lier une police Google Font externe appelée *Playwright Cuba* :

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Playwrite+CU:wght@100..400&display=swap"
  rel="stylesheet"
/>
```

Google Fonts est un ensemble de polices personnalisées gratuites et open source que tu peux utiliser dans n'importe quel projet. Tu peux choisir les polices que tu veux utiliser, et Google te fournit le code HTML et CSS nécessaire. Dans cet exemple, la valeur `preconnect` de l'attribut `rel` indique au navigateur de créer une connexion anticipée vers la valeur indiquée dans l'attribut `href`. Cela sert à accélérer le chargement de ces ressources externes.

Un autre cas d'utilisation courant de l'élément `link` consiste à lier des icônes. Voici un exemple de lien vers un favicon :

```html
<link rel="icon" href="favicon.ico" />
```

Un favicon, abréviation de favorite icon, est une petite icône généralement affichée dans l'onglet du navigateur à côté du titre du site. Beaucoup de sites utilisent un favicon pour afficher l'icône de leur marque.
