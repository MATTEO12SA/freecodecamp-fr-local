# --description--

UTF-8, ou UCS Transformation Format 8, est un encodage de caractères standardisé très utilisé sur le Web. L'encodage de caractères est la méthode que les ordinateurs utilisent pour stocker les caractères sous forme de données. En pratique, tout le texte d'une page web est une suite de caractères stockés sous forme d'un ou plusieurs octets. En informatique, un octet est une unité de données composée de 8 bits, ou chiffres binaires. UTF-8 prend en charge tous les caractères de l'ensemble Unicode, ce qui inclut les caractères et symboles de tous les systèmes d'écriture, langues et symboles techniques. Voici un exemple d'utilisation de l'élément `meta` avec l'attribut `charset` pour définir l'encodage des caractères sur `UTF-8` :

```html
<meta charset="UTF-8" />
```

Définir l'encodage des caractères sur UTF-8 garantit que le caractère `"e"` accentué (`é`) s'affiche correctement sur la page. Voici un exemple de code plus complet qui utilise l'encodage de caractères UTF-8 :

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Examples of the UTF-8 encoding</title>
  </head>
  <body>
    <p>Café</p>
  </body>
</html>
```

Pour chaque nouveau projet que tu crées, tu dois inclure cet élément `meta` avec l'attribut `charset` défini sur `UTF-8`.
