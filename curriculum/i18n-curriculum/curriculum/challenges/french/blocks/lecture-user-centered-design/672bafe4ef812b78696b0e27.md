# --interactive--

**NOTE** : Certains des exemples interactifs peuvent utiliser du CSS que tu n'as pas encore appris. Ne t'inquiète pas si tu essaies de tout comprendre. Le but des exemples est de te montrer des aperçus de ces concepts de design pour que tu comprennes mieux comment les choses fonctionnent.

Le mode sombre est une fonctionnalité spéciale sur les applications web où tu peux changer la palette de couleurs claires par défaut en une palette de couleurs sombres. Cela aide à réduire la fatigue oculaire et améliore la lisibilité dans des conditions de faible luminosité. Quand tu conçois tes fonctionnalités de mode sombre, il est important de comprendre les bonnes pratiques pour t'assurer que ta fonctionnalité de mode sombre est efficace et conviviale.

Active l'éditeur interactif et clique sur le bouton `Toggle Dark Mode` dans l'exemple ci-dessous pour voir comment le mode sombre fonctionne.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<button id="theme-toggle">Toggle Dark Mode</button>

<div class="content">
  <p>This is a simple dark mode example.</p>
</div>

<script src="index.js"></script>
```

```css
body {

  color: #222;
  font-family: sans-serif;
  transition: background 0.3s, color 0.3s;
}

  margin: 1rem;
```

```js
document.getElementById('theme-toggle').addEventListener('click', () => {
  document.body.classList.toggle('dark');
});
```

:::

La première considération est d'éviter les couleurs saturées en mode sombre. Les couleurs saturées sont des couleurs vives et intenses. Par exemple, un bouton magenta vif sur un fond gris foncé peut être trop intense et causer de la fatigue oculaire. À la place, tu devrais utiliser des couleurs désaturées en mode sombre. Les couleurs désaturées sont des couleurs moins intenses, avec un niveau de saturation plus bas, et plus confortables à regarder en mode sombre. Pour voir les aperçus, tu devras activer l'éditeur interactif.

Une autre considération avec le mode sombre est l'utilisation de fonds noirs purs avec du texte blanc. Bien que ce contraste élevé puisse être efficace, il peut aussi être trop dur pour les yeux. À la place, considère utiliser un fond gris foncé avec du texte gris clair pour un contraste plus doux. Le texte sera plus facile pour les yeux et plus confortable à lire en mode sombre. Pour voir les aperçus, tu devras activer l'éditeur interactif.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<button id="theme-toggle">Toggle Dark Mode</button>

      <h2>High Contrast</h2>
      <p>Pure black background with white text. While readable, it can be harsh on the eyes.</p>
    </div>

      <h2>Soft Contrast</h2>
      <p>Dark gray background with light gray text. Easier to read and more comfortable for long periods.</p>
```

Une autre considération est l'utilisation du mode sombre avec l'identité de marque du site. Une identité de marque est un ensemble d'éléments visuels qui représentent une marque, comme le logo, les couleurs et la typographie. Quand tu implémentes le mode sombre, tu devrais considérer comment la fonctionnalité de mode sombre est cohérente avec les couleurs et le style de ta marque. Il est correct d'avoir l'icône de la marque et les boutons à pleine saturation, tandis que les éléments environnants sont désaturés.

En général, quand il s'agit de design, tu veux toujours être attentif à l'expérience utilisateur et aux niveaux de contraste. Le mode sombre ne fait pas exception, et en suivant ces bonnes pratiques, tu peux créer une fonctionnalité de mode sombre qui est efficace et conviviale.
