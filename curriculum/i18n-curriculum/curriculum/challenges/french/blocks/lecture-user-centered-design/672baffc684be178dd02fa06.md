# --interactive--

**NOTE** : Certains des exemples interactifs peuvent utiliser du CSS que tu n'as pas encore appris. Ne t'inquiète pas si tu essaies de tout comprendre. Le but des exemples est de te montrer des aperçus de ces concepts de design pour que tu comprennes mieux comment les choses fonctionnent. Pour voir les aperçus, tu devras activer l'éditeur interactif.

Les composants carte sont très courants sur les sites e-commerce, les réseaux sociaux et les sites d'actualités. Ils sont utilisés pour aider à afficher l'information de manière structurée. Quand tu conçois tes cartes, il est important de comprendre les bonnes pratiques pour que tes utilisateurs puissent facilement comprendre l'information que tu essaies de transmettre.

La première considération pour le design de carte devrait être la simplicité. Tu ne veux pas que tes cartes soient visuellement encombrées ou affichent trop d'information. Par exemple, si un design de carte est visuellement encombré, il y aura trop d'information pour que l'utilisateur puisse tout traiter d'un coup.

Voici un exemple de design de carte encombré :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2><strong><em>Product Name: Ultimate Edition</em></strong></h2>
  <p>
    This is a great product that you will love! It has many features and benefits that will make your life easier and more enjoyable. Whether you're at home or on the go, this product is perfect for all situations. Don’t miss out on this incredible offer, buy now and experience the difference!
  </p>
  <button>Buy Now</button>
  <button>Learn More</button>
  <button>Add to Wishlist</button>
  <p><strong>Only 3 left in stock!</strong></p>
  <p><em>Rated 4.8 stars by 2,391 customers</em></p>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}
.card p {
  font-size: 1em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

Avoir moins d'information et un bon espacement entre les éléments de la carte rend plus facile pour l'utilisateur de traiter l'information, et permet d'avoir plusieurs cartes sur la page.

Voici un exemple de design de carte simple :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <img src="https://placehold.co/150x150" alt="Example Product Image">
  <h2>Product Name</h2>
  <button>Buy Now</button>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 200px;
  text-align: center;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

Une autre chose à considérer est où l'utilisateur peut cliquer sur la carte. Certains designs de carte auront un seul bouton, rendant évident où l'utilisateur peut cliquer. D'autres designs de carte auront toute la carte cliquable. Quand l'utilisateur survole n'importe quelle partie de la carte, la carte changera de couleur ou aura un effet d'ombre pour indiquer que la carte est cliquable. Quel que soit le design que tu choisis, il doit être cohérent sur tout ton site et facile à comprendre pour l'utilisateur.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <a href="#">
    <img src="https://placehold.co/150x150" alt="Example Product Image">
    <h2>Product Name</h2>
    <button>Buy Now</button>
  </a>
</div>
```

```css
.card {
  border: 1px solid #ccc;
  padding: 16px;
  width: 300px;
  cursor: pointer;
}

.card:hover {
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}

.card a {
  text-decoration: none;
  color: inherit;
}

.card img {
  max-width: 100%;
  height: auto;
}

.card h2 {
  font-size: 1.5em;
  margin: 8px 0;
}

.card button {
  background-color: #007BFF;
  color: white;
  padding: 10px 16px;
  border: none;
  cursor: pointer;
}
```

:::

Une autre considération est l'utilisation de médias sur tes cartes. Choisir des médias de haute qualité peut grandement améliorer l'expérience utilisateur. Si tu utilises des images ou des vidéos pour, disons, une carte de produit, plus la qualité est élevée, plus l'utilisateur sera intéressé par ce produit. Mais si tu utilises une qualité média médiocre, l'utilisateur pourrait ne pas faire confiance à la qualité de tes produits et services.

Une des dernières choses à considérer est l'utilisation de la hiérarchie des couleurs. Tu veux t'assurer que l'information la plus importante sur la carte est la plus visible. Tu peux utiliser des couleurs vives pour les éléments importants comme un bouton call-to-action, et des couleurs claires pour les éléments moins importants sur une carte.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">

</div>
```

```css

  text-align: center;
}

.card img {
  max-width: 100%;
```
