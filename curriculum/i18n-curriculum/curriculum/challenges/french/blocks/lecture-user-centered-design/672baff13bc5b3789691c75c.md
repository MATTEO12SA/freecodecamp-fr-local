# --interactive--

**NOTE** : Certains des exemples interactifs peuvent utiliser du CSS que tu n'as pas encore appris. Ne t'inquiète pas si tu essaies de tout comprendre. Le but des exemples est de te montrer des aperçus de ces concepts de design pour que tu comprennes mieux comment les choses fonctionnent. Pour voir les aperçus, tu devras activer l'éditeur interactif.

Quand il s'agit de design web, il y a beaucoup de types d'aides à la navigation que tu peux utiliser. Des exemples incluent les barres de navigation supérieures, les barres latérales et les pieds de page. Mais si ton site est plutôt complexe avec des niveaux de navigation plus profonds, tu pourrais vouloir considérer l'utilisation de fils d'Ariane (breadcrumbs).

Les fils d'Ariane sont une aide à la navigation qui montre à l'utilisateur où il se trouve dans la hiérarchie du site. Voici un exemple de ce à quoi ressemblent les fils d'Ariane pour un site web d'électronique fictif :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home / Electronics / Phones / Smartphone XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Dans la plupart des sites web, les fils d'Ariane sont affichés en haut de la page, montrant à l'utilisateur le chemin qu'il a pris pour arriver à la page actuelle. À partir de la `Homepage`, l'utilisateur a navigué vers la catégorie `Electronics`, puis vers la catégorie `Phones`, et enfin vers le produit `Smartphone XYZ`. Tu as probablement interagi avec des fils d'Ariane sur un site web pendant que tu cherchais un produit ou une information spécifique.

L'utilisation des fils d'Ariane est utile parce qu'elle peut aider les utilisateurs à comprendre où ils se trouvent dans la hiérarchie du site et comment naviguer vers les pages précédentes. C'est particulièrement utile quand un utilisateur vient d'un résultat de recherche ou d'un lien externe et a besoin de comprendre le contexte de la page sur laquelle il se trouve.

Quand il s'agit de concevoir des fils d'Ariane, il y a quelques considérations à garder en tête. La première est de décider quel sera le séparateur. Le séparateur est le caractère qui sépare les différents niveaux de la hiérarchie. Les séparateurs courants incluent le signe plus grand que (`>`), les guillemets angulaires droits (`»`), et la barre oblique avant (`/`).

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

La deuxième considération est le placement des fils d'Ariane. Les fils d'Ariane sont généralement placés en haut de la page, soit au-dessus, soit en dessous de la barre de navigation principale. Les utilisateurs ne devraient pas avoir à lutter pour trouver les fils d'Ariane, donc assure-toi qu'ils sont visibles et faciles à localiser.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<nav class="main-nav">
  <a href="#">Home</a>
  <a href="#">Electronics</a>
  <a href="#">Phones</a>
</nav>
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
```

```css
.main-nav {
  background-color: #333;
  padding: 1rem;
}

.main-nav a {
  color: white;
  margin-right: 1rem;
  text-decoration: none;
}

.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Une autre considération est la taille des fils d'Ariane. Tu veux t'assurer que les fils d'Ariane sont assez grands pour être facilement lus, mais pas si grands qu'ils prennent trop d'espace sur la page. Rappelle-toi, les fils d'Ariane ne sont pas censés servir d'outil de navigation principal, mais plutôt d'aide de navigation secondaire. Dans les sites web où il y a beaucoup d'informations sur une page, les utilisateurs peuvent facilement voir où ils sont dans la hiérarchie et naviguer vers les pages précédentes en utilisant les fils d'Ariane.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
<p>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
</p>
```

```css
.breadcrumbs {
  font-size: 1rem;
  color: #555;
  margin: 1rem 0;
}   

p {
  font-size: 1.2rem;
  line-height: 1.6;
}
```

:::

Alors quand devrais-tu utiliser des fils d'Ariane sur ton site ? Si ton site est simple et a une hiérarchie peu profonde, les fils d'Ariane pourraient ne pas être nécessaires. Une hiérarchie peu profonde signifie que le site n'a que quelques niveaux de navigation, donc les fils d'Ariane pourraient ne pas ajouter beaucoup de valeur. Généralement, tu verras des fils d'Ariane sur les sites e-commerce, les sites d'actualités, et autres sites avec une hiérarchie profonde comme les sites de documentation technique.
