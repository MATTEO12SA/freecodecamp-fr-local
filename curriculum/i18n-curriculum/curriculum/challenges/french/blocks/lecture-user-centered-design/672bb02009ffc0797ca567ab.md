L'indication de progression est une façon de montrer aux utilisateurs jusqu'où ils sont dans un processus. Elle peut être utilisée dans les formulaires, l'inscription et les processus de configuration. Le but est d'aider les utilisateurs à comprendre où ils en sont dans le processus et combien il leur reste à faire.

Par exemple, tu peux utiliser une barre d'indication de progression pour montrer aux utilisateurs ce qu'il reste à faire en remplissant des formulaires. Tu ne veux pas créer une situation où l'utilisateur doit remplir un long formulaire et ne sait pas combien d'étapes supplémentaires il doit compléter. La transparence est essentielle pour que l'utilisateur sache s'il a assez de temps pour s'asseoir et compléter le formulaire ou s'il doit revenir plus tard.

```js
const form = document.getElementById('multiStepForm');
const steps = form.querySelectorAll('.form-step');
const progressBar = form.querySelector('.progress-bar');
const progressText = form.querySelector('.progress-text');
const totalSteps = steps.length;

let currentStep = 0;

function updateProgress() {
  const percent = ((currentStep + 1) / totalSteps) * 100;
  progressBar.style.width = percent + '%';
  progressText.textContent = `Step ${currentStep + 1} of ${totalSteps}`;
}

function showStep(index) {
  steps.forEach((step, i) => {
    step.classList.toggle('active', i === index);
  });
  updateProgress();
}

form.querySelectorAll('.next-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep < totalSteps - 1) {
      currentStep++;
      showStep(currentStep);
    }
  });
});

form.querySelectorAll('.prev-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep > 0) {
      currentStep--;
      showStep(currentStep);
    }
  });
});

showStep(currentStep);

form.addEventListener('submit', e => {
  e.preventDefault();
  alert('Form submitted!');
});

```

Quand tu conçois une section d'indication de progression, il y a quelques bonnes pratiques à garder en tête. La première considération est de la garder simple. Tu ne veux pas submerger l'utilisateur avec trop d'informations qui le frustreraient et le feraient quitter le site.

La deuxième considération est de rendre possible le retour aux étapes précédentes. C'est important parce que les utilisateurs peuvent vouloir revenir en arrière et vérifier leurs réponses précédentes ou faire des changements.

Une autre considération est de rendre la section d'indication de progression facile à trouver. Si l'utilisateur ne peut pas la trouver, il ne saura pas jusqu'où il est dans le processus.

La dernière considération est d'avoir des titres de section clairs, des pourcentages ou des étapes. Si tu as juste une barre de progression sans contexte, l'utilisateur ne saura pas ce qu'elle signifie.

const form = document.getElementById('progressForm');
const steps = form.querySelectorAll('.form-step');
const progressBar = form.querySelector('.progress-bar');
const currentStepSpan = document.getElementById('currentStep');
const totalStepsSpan = document.getElementById('totalSteps');
const percentageSpan = document.getElementById('percentage');

const totalSteps = steps.length;
let currentStep = 0;

totalStepsSpan.textContent = totalSteps;

function updateProgress() {
  const percent = Math.round(((currentStep + 1) / totalSteps) * 100);
  progressBar.style.width = percent + '%';
  currentStepSpan.textContent = currentStep + 1;
  percentageSpan.textContent = percent + '%';

  // Update ARIA attribute
  form.querySelector('.progress-container').setAttribute('aria-valuenow', percent);
}

function showStep(index) {
  steps.forEach((step, i) => {
    step.classList.toggle('active', i === index);
  });
  updateProgress();
}

form.querySelectorAll('.next-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep < totalSteps - 1) {
      currentStep++;
      showStep(currentStep);
    }
  });
});

form.querySelectorAll('.prev-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    if (currentStep > 0) {
      currentStep--;
      showStep(currentStep);
    }
  });
});

form.addEventListener('submit', e => {
  e.preventDefault();
  alert('Form submitted! Thanks!');
});

showStep(currentStep);
