# --interactive--

Les leçons précédentes t'ont présenté `console.log()`, mais cette leçon va approfondir son objectif et son utilisation.

En JavaScript, `console.log()` est un outil simple mais puissant utilisé pour afficher des messages ou des informations dans la console du navigateur. Il est surtout utilisé par les développeurs pour déboguer et inspecter le code pendant qu'ils travaillent sur leurs programmes.

Tu peux utiliser `console.log()` pour afficher du texte ou des variables dans la console et vérifier que ton code s'exécute correctement.

Pour utiliser `console.log()`, tu appelles la méthode avec la valeur ou le message que tu veux afficher à l'intérieur des parenthèses. Voici quelques exemples :

:::interactive_editor

```js
console.log("Hello, world!");

let num = 5;
console.log(num); // 5
```

:::

Le premier exemple affiche `Hello, world!` dans la console du navigateur, tandis que le second affiche la valeur `5`.

Voici un autre exemple d'utilisation de `console.log()` :

:::interactive_editor

```js
let name = "Alice";
console.log("Hello, " + name + "!"); // Hello, Alice!
```

:::

Tu peux aussi passer plusieurs valeurs à `console.log()` en les séparant par des virgules. Par exemple :

:::interactive_editor

```js
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```

:::

C'est pratique pour afficher plusieurs informations en une seule fois.

La méthode `console.log()` t'aide à surveiller ton code pendant son exécution, ce qui facilite la détection des erreurs et la compréhension du comportement de ton programme.
