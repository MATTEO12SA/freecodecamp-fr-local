## Introduction à CSS Flexbox et au modèle flex

- **Définition** : CSS Flexbox est un modèle de mise en page unidimensionnel qui te permet d'organiser des éléments en lignes et en colonnes dans un conteneur.
- **Modèle flex** : Ce modèle définit comment les éléments flex sont organisés dans un conteneur flex. Chaque conteneur flex a deux axes : l'axe principal et l'axe transversal.

## La propriété `flex-direction`

- **Définition** : Cette propriété définit la direction de l'axe principal. La valeur par défaut de `flex-direction` est `row`, ce qui place tous les éléments flex sur la même ligne, dans la direction de la langue par défaut de ton navigateur (de gauche à droite ou de droite à gauche).
- **`flex-direction: row-reverse;`** : Cela inverse les éléments dans la ligne.
- **`flex-direction: column;`** : Cela alignera les éléments flex verticalement plutôt qu'horizontalement.
- **`flex-direction: column-reverse;`** : Cela inverse verticalement l'ordre des éléments flex.

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: row;
  gap: 10px;
}

.box {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
```

:::

## La propriété `flex-wrap`

- **Définition** : Cette propriété détermine comment les éléments flex passent à la ligne dans un conteneur flex pour tenir dans l'espace disponible. `flex-wrap` peut prendre trois valeurs possibles : `nowrap`, `wrap` et `wrap-reverse`.
- **`flex-wrap: nowrap;`** : C'est la valeur par défaut. Les éléments flex ne passeront pas sur une nouvelle ligne, même si leur largeur dépasse celle du conteneur.
- **`flex-wrap: wrap;`** : Cette propriété fera passer les éléments à la ligne quand ils dépassent la largeur de leur conteneur.
- **`flex-wrap: wrap-reverse;`** : Cette propriété fera passer les éléments flex à la ligne dans l'ordre inverse.
- **Propriété `flex-flow`** : Cette propriété est une propriété raccourcie pour `flex-direction` et `flex-wrap`.

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
  <div class="box">4</div>
  <div class="box">5</div>
</div>
```

```css
.container {
  display: flex;
  flex-wrap: wrap;
  width: 150px;
  background: #f0f0f0;
}

.box {
  width: 60px;
  padding: 10px;
  margin: 5px;
  background: skyblue;
  text-align: center;
}
```

:::

## La propriété `justify-content`

- **Définition** : Cette propriété aligne les éléments enfants le long de l'axe principal du conteneur flex.
- **`justify-content: flex-start;`** : Dans ce cas, les éléments flex seront alignés au début de l'axe principal. Cela peut être horizontal ou vertical.
- **`justify-content: flex-end;`** : Dans ce cas, les éléments flex sont alignés à la fin de l'axe principal, horizontalement ou verticalement.
- **`justify-content: center;`** : Cela centre les éléments flex le long de l'axe principal.
- **`justify-content: space-between;`** : Cela distribuera les éléments uniformément le long de l'axe principal.
- **`justify-content: space-around;`** : Cela distribuera les éléments flex uniformément sur l'axe principal, en ajoutant un espace avant le premier élément et après le dernier élément.
- **`justify-content: space-evenly;`** : Cela distribuera les éléments uniformément le long de l'axe principal.

:::interactive_editor

```html
<div class="container">
  <div class="box">A</div>
  <div class="box">B</div>
  <div class="box">C</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
  background: #eee;
}

.box {
  padding: 20px;
  background: salmon;
}
```

:::
  
## La propriété `align-items`

- **Définition** : Cette propriété est utilisée pour distribuer les éléments le long de l'axe transversal. Rappelle-toi que l'axe transversal est perpendiculaire à l'axe principal.
- **`align-items: center;`** : Cela sert à centrer les éléments le long de l'axe transversal.
- **`align-items: flex-start;`** : Cela aligne les éléments au début de l'axe transversal.
- **`align-items: stretch;`** : Cela sert à étirer les éléments flex le long de l'axe transversal.

:::interactive_editor

```html
<div class="container">
  <div class="box tall">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  align-items: center;
  height: 150px;
  background: #ddd;
}

.box {
  background: lightgreen;
  padding: 10px;
}

.tall {
  height: 100px;
}
```
