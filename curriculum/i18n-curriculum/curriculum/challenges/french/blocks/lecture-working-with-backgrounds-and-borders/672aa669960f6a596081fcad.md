# --interactive--

Quand tu travailles avec des images d'arrière-plan en CSS, tu as plusieurs propriétés à ta disposition pour contrôler comment ces images sont affichées.

Les principales propriétés sur lesquelles on va se concentrer sont `background-size`, `background-repeat`, `background-position` et `background-attachment`.

Regardons d'abord la propriété `background-image` :

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

Le CSS ci-dessus définit une image de fond de chat pour l'élément `body`.

Si tu veux définir la taille pour l'image d'arrière-plan, tu peux utiliser la propriété `background-size`.

Tu peux utiliser `contain` pour mettre l'image à l'échelle aussi grande que possible sans la rogner ou l'étirer.

Voici un exemple avec `background-size: contain` :

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

:::

On définit `min-height` à `100px` pour que l'image d'arrière-plan soit visible et que la mise en page maintienne une hauteur de base, garantissant que même avec un contenu minimal, le design apparaisse équilibré et visuellement attrayant.

Si on change la propriété `background-size` pour utiliser la valeur `cover`, alors l'image d'arrière-plan se mettra à l'échelle pour couvrir tout l'élément `body` tout en conservant son ratio d'aspect.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

:::

Dans les exemples précédents, tu as probablement remarqué que l'image d'arrière-plan se répétait continuellement.

Par défaut, les images d'arrière-plan se répètent à la fois horizontalement et verticalement pour remplir tout l'élément. Cependant, tu peux contrôler ce comportement.

Tu peux utiliser la propriété `background-repeat` avec la valeur définie à `no-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Avec le `background-size` défini à `contain` et le `background-repeat` défini à `no-repeat`, l'image ne se répétera plus sur l'écran.

Si tu voulais répéter l'image d'arrière-plan horizontalement, tu peux utiliser la valeur `repeat-x` pour la propriété `background-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

:::

Si tu voulais répéter l'image d'arrière-plan verticalement, tu peux utiliser la valeur `repeat-y` pour la propriété `background-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

:::

Pour positionner une image d'arrière-plan sur l'écran, tu peux utiliser la propriété `background-position`.

La propriété `background-position` te permet de définir où dans l'élément l'image d'arrière-plan apparaît. Tu peux utiliser des mots-clés comme `top`, `bottom`, `left`, `right` et `center`, ou des valeurs spécifiques en pixels ou en pourcentage.

Voici un exemple utilisant `center` et `top` pour la `background-position` :

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

Ce code CSS positionne l'image d'arrière-plan au centre de l'élément horizontalement et au sommet verticalement.

Enfin, `background-attachment` détermine si l'image d'arrière-plan défile avec le contenu ou reste fixe quand la page est défilée.

Les valeurs principales sont `scroll` (par défaut), où l'image d'arrière-plan défile avec le contenu, et `fixed`, où l'image d'arrière-plan reste à la même position sur l'écran.

Voici un exemple utilisant la valeur `fixed` pour la propriété `background-attachment` :

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

:::

Ce code CSS définit l'image d'arrière-plan pour qu'elle reste fixe en place même quand la page est défilée.

Si tu voulais combiner quelques-unes des propriétés en une seule ligne, tu peux le faire en utilisant la propriété raccourcie `background`.

Voici un exemple :

:::interactive_editor

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

L'exemple ci-dessus combine le `background-image`, `background-position` et `background-attachment` en une seule ligne.

En maîtrisant ces propriétés, tu auras un grand contrôle sur la façon dont les images d'arrière-plan sont affichées dans tes designs web, permettant des mises en page plus visuellement attrayantes et fonctionnelles.
