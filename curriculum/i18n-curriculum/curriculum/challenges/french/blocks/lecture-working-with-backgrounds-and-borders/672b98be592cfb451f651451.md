# --interactive--

Un dégradé d'arrière-plan en CSS est une transition douce entre deux couleurs ou plus qui peut être appliquée à l'arrière-plan d'un élément. Les dégradés te permettent de créer des arrière-plans visuellement attrayants sans avoir besoin d'images.

Il y a deux types principaux de dégradés en CSS : les dégradés linéaires et les dégradés radiaux.

Un dégradé linéaire fait passer les couleurs le long d'une ligne droite. Tu peux définir la direction et les couleurs impliquées.

Voici la syntaxe de base :

```css
background: linear-gradient(direction, color-stop1, color-stop2, ...);
```

Dans cet exemple, on utilise la propriété CSS `background` avec une valeur de `linear gradient`.

La direction spécifie la direction du dégradé. Ça peut être un angle (comme `45deg`), un mot-clé (comme `to right`, `to bottom`), ou un côté/coin.

`color-stop` spécifie les couleurs et les positions où les transitions du dégradé se produisent.

Pour mieux comprendre comment les dégradés linéaires fonctionnent, regardons l'exemple suivant :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient{
  background: linear-gradient(to right, red, yellow);
  height: 40vh;
}
```

:::

Ce CSS crée un dégradé linéaire qui passe de `red` à `left` à `yellow` à `right`. Le dégradé est appliqué à un élément avec une hauteur de `40%` de la hauteur du viewport. Tu en apprendras plus sur les unités `vh` dans une future leçon.

La direction `to right` signifie que le dégradé s'exécute horizontalement de gauche à droite.

Un autre type de dégradé serait le dégradé `radial`.

Un dégradé radial fait passer les couleurs en rayonnant depuis une origine (généralement le centre) vers l'extérieur dans une forme circulaire ou elliptique.

Voici la syntaxe de base :

```css
background: radial-gradient(shape size at position, color-stop1, color-stop2, ...)
```

Sur la syntaxe, le `shape` spécifie la forme du dégradé qui pourrait être `circle` ou `ellipse`.

Le `size` détermine la taille de la forme finale du dégradé qui pourrait être `closest-side`, `closest-corner`, `farthest-side` ou `farthest-corner`.

`position` détermine la position du centre du dégradé qui pourrait être spécifiée en utilisant des mots-clés (comme `center`, `top left`, `bottom right`) ou des valeurs précises (comme `50% 50%`, `10px 20px`).

Enfin, les color stops sont une liste de couleurs à travers lesquelles le dégradé passe. Chaque color stop peut éventuellement inclure une valeur de position (pourcentage ou longueur) indiquant où la couleur devrait être placée.

Un exemple serait :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient{
  background: radial-gradient(circle closest-side at center, red, yellow 50%, green);
  height: 60vh;
}
```

:::

Ce CSS crée un dégradé radial circulaire centré dans l'élément. Il commence par `red` au centre, passe à `yellow` à `50%` du rayon, et finit par `green`.

Le mot-clé `closest-side` fait que la forme finale du dégradé s'adapte au côté le plus proche de l'élément. Le dégradé est appliqué à un élément avec une hauteur de `60%` de la hauteur du viewport.

Comprendre comment travailler avec les dégradés CSS peut grandement améliorer tes designs en fournissant des arrière-plans visuellement attrayants sans le besoin d'images.

Avec des options comme les dégradés linéaires pour des transitions douces et les dégradés radiaux pour des effets circulaires, ils offrent à la fois de la flexibilité et de la créativité dans le design web.
