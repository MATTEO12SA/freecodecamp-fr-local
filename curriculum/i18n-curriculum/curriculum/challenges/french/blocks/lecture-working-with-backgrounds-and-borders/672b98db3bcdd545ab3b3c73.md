# --interactive--

En CSS, il y a plusieurs façons d'ajouter des bordures autour des images, chacune offrant différentes options de mise en forme et niveaux de contrôle.

Explorons quelques-unes des méthodes les plus courantes et polyvalentes.

La façon la plus simple d'ajouter une bordure à une image est d'utiliser la propriété `border`. Cette propriété est un raccourci qui te permet de définir la largeur, le style et la couleur de la bordure en une seule fois.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

Cette règle CSS ajoute une bordure rouge continue de 2 pixels autour de tous les éléments `img`. Tu peux ajuster la largeur, le style (comme `dashed`, `dotted` ou `double`), et la couleur pour répondre à tes besoins de design.

Si tu as besoin de plus de contrôle sur les côtés individuels de la bordure, tu peux utiliser les propriétés de bordure spécifiques pour chaque côté :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border-top: 10px solid red;
  border-right: 10px dashed green;
  border-bottom: 10px dotted blue;
  border-left: 10px double purple;
}
```

:::

Ça te permet de créer des styles de bordure uniques pour chaque côté de l'image.

Une autre façon de créer un effet de bordure est d'utiliser la propriété `outline`. Bien que similaire à border, outline n'affecte pas les dimensions ou la mise en page de l'élément :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  outline: 3px solid gold;
}
```

:::

Cela crée un contour doré autour de l'image sans changer sa taille ou sa position.

Si tu veux créer des coins arrondis sur ta bordure, tu peux utiliser la propriété `border-radius` en conjonction avec border :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

Rappelle-toi, ces techniques peuvent être combinées et personnalisées pour créer des effets de bordure uniques. Le choix de la méthode dépend de tes exigences de design spécifiques et du niveau de complexité dont tu as besoin.
