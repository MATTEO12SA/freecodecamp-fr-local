## Propriété CSS Overflow

- **Définition** : Le débordement (overflow) fait référence à la façon dont les éléments gèrent le contenu qui dépasse, ou « déborde », la taille de l'élément contenant. L'overflow est bidimensionnel.
- **`overflow-x`** : L'axe x détermine le débordement horizontal.
- **`overflow-y`** : L'axe y détermine le débordement vertical.
- **`overflow`** : Propriété raccourcie pour `overflow-x` et `overflow-y`. Si on lui donne une valeur, les deux débordements l'utiliseront. Si on lui donne deux valeurs, `overflow-x` utilisera la première, et `overflow-y` utilisera la seconde.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="overflow-box">
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
</div>
```

```css
.overflow-box {
  width: 140px;
  height: 70px;
  border: 2px solid black;
  overflow-x: auto;
  overflow-y: auto;
  white-space:nowrap;
}
```

## Propriété CSS Transform

- **Définition** : Cette propriété te permet d'appliquer diverses transformations aux éléments, comme les faire pivoter, les mettre à l'échelle, les fausser ou les translater (déplacer) dans un espace 2D ou 3D.
- **Fonction `translate()`** : Cette fonction est utilisée pour déplacer un élément de sa position actuelle.
- **Fonction `scale()`** : Cette fonction te permet de changer la taille d'un élément.
- **Fonction `rotate()`** : Cette fonction te permet de faire pivoter un élément.
- **Fonction `skew()`** : Cette fonction te permet de fausser un élément.
- **Transforms et accessibilité** : Si tu utilises transform pour cacher ou révéler du contenu, assure-toi que le contenu reste accessible aux lecteurs d'écran et à la navigation au clavier. Le contenu caché devrait être véritablement caché, comme en utilisant `display: none` ou `visibility: hidden`, plutôt qu'être simplement déplacé visuellement hors écran.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="transform-box translate">Translate</div>
  <div class="transform-box scale">Scale</div>
  <div class="transform-box rotate">Rotate</div>
  <div class="transform-box skew">Skew</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 16px;
  flex-wrap: wrap;
}

.transform-box {
  width: 100px;
  height: 100px;
  background: coral;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

/* translate() */
.translate {
  transform: translateX(20px);
}

/* scale() */
.scale {
  transform: scale(1.2);
}

/* rotate() */
.rotate {
  transform: rotate(15deg);
}

/* skew() */
.skew {
  transform: skew(10deg, 5deg);
}
```

## Le modèle de boîte

- **Définition** : Dans le modèle de boîte CSS, chaque élément est entouré d'une boîte. Cette boîte est composée de quatre composants : la zone de contenu, `padding`, `border`, `margin`.
- **Zone de contenu** : La zone de contenu est la partie la plus interne de la boîte. C'est l'espace qui contient le contenu réel d'un élément, comme du texte ou des images.
- **`padding`** : Le padding est la zone immédiatement après la zone de contenu. C'est l'espace entre la zone de contenu et la bordure d'un élément.
- **`border`** : La bordure est le bord extérieur ou le contour d'un élément dans le modèle de boîte CSS. C'est la limite visuelle de l'élément.
- **`margin`** : La margin est l'espace à l'extérieur de la bordure d'un élément. Elle détermine la distance entre un élément et d'autres éléments autour de lui.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="box-model">Content</div>
```

```css
.box-model {
  width: 120px;
  background: lightblue;
  padding: 16px;
  border: 4px solid teal;
  margin: 20px;
  text-align: center;
}
```

:::

## Effondrement de margin

- **Définition** : Ce comportement se produit quand les margins verticales d'éléments adjacents se chevauchent, résultant en une seule margin égale à la plus grande des deux. Ce comportement s'applique uniquement aux margins verticales (haut et bas), pas aux margins horizontales (gauche et droite).

## Les valeurs de propriété `content-box` et `border-box`

- **Propriété `box-sizing`** : Cette propriété est utilisée pour déterminer comment la `width` et la `height` finales sont calculées pour un élément HTML.
- **Valeur `content-box`** : Dans le modèle `content-box`, la `width` et la `height` que tu définis pour un élément déterminent les dimensions de la zone de contenu mais elles n'incluent pas le `padding`, la `border` ou la `margin`.
- **Valeur `border-box`** : Avec `border-box`, la `width` et la `height` d'un élément incluent la zone de contenu, le `padding` et la `border`, mais elles n'incluent pas la `margin`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sizing-box content-box">Content-box</div>
<div class="sizing-box border-box">Border-box</div>
```

```css
.sizing-box {
  width: 150px;
  padding: 10px;
  border: 4px solid teal;
  margin-bottom: 10px;
  background: peachpuff;
  text-align: center;
}

.content-box {
  box-sizing: content-box;  
}

.border-box {
  box-sizing: border-box;
  background: lightgreen;
}
```

## CSS Reset

- **Définition** : Un CSS reset est une feuille de style qui supprime tout ou partie du formatage par défaut que les navigateurs web appliquent aux éléments HTML. Les options tierces pour les CSS resets incluent `sanitize.css` et `normalize.css`.

## Propriété CSS Filter

- **Définition** : Cette propriété peut être utilisée pour créer divers effets comme le flou, le décalage de couleur et l'ajustement du contraste.
- **Fonction `blur()`** : Cette fonction applique un flou gaussien à l'élément. La quantité est définie en pixels et représente le rayon du flou.
- **Fonction `brightness()`** : Cette fonction ajuste la luminosité de l'élément. Une valeur de 0% rendra l'élément complètement noir, tandis que des valeurs supérieures à 100% augmenteront la luminosité.
- **Fonction `contrast()`** : Cette fonction ajuste le contraste de l'élément. Une valeur de 0% rendra l'élément complètement gris, 100% fera apparaître l'élément normalement, et des valeurs supérieures à 100% augmenteront le contraste.
- **Fonction `grayscale()`** : Cette fonction convertit l'élément en niveaux de gris. La quantité est définie en pourcentage, où 100% est complètement en niveaux de gris et 0% laisse l'image inchangée.
- **Fonction `sepia()`** : Cette fonction applique un ton sépia à l'élément. Comme grayscale, elle utilise une valeur en pourcentage.
- **Fonction `hue-rotate()`** : Cette fonction applique une rotation de teinte à l'élément. La valeur est définie en degrés et représente une rotation autour du cercle des couleurs.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <img class="filter blur" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Blur" />
  <img class="filter brightness" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Brightness" />
  <img class="filter contrast" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Contrast" />
  <img class="filter grayscale" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Grayscale" />
  <img class="filter sepia" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Sepia" />
  <img class="filter hue" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Hue Rotate" />
</div>
```

```css
.container {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
}

.filter {
  width: 120px;
}

/* blur() */
.blur {
  filter: blur(3px);
}

/* brightness() */
.brightness {
  filter: brightness(130%);
}

/* contrast() */
.contrast {
  filter: contrast(150%);
}

/* grayscale() */
.grayscale {
  filter: grayscale(100%);
}

/* sepia() */
.sepia {
  filter: sepia(100%);
}

/* hue-rotate() */
.hue {
  filter: hue-rotate(90deg);
}
```
