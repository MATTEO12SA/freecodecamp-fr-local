# --description--

Remarque : tout le contenu de la page est imbriqué dans un élément `html`. Le `html` est l'élément **racine** d'une page : il englobe tout le reste.

Tu peux préciser la **langue** de ta page en ajoutant l'attribut `lang` à la balise `html`.

Ajoute `lang="en"` à la balise ouvrante `<html>` pour indiquer que la page est en anglais.

# --hints--

Le `html` doit toujours être présent (balise ouvrante et fermante).

```js
assert(code.match(/\<html.*?\>/) && code.match(/\<\/html\>/));
```

L'attribut `lang` doit valoir `en`.

```js
const extraSpacesRemoved = code.replace(/\s+/g, " ");
assert(extraSpacesRemoved.match(/\<html\s+lang\s*\=\s*("?|'?)en\1\s*\>/));
```
