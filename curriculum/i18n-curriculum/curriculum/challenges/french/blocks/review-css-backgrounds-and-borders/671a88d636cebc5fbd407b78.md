## Mise en forme des listes

- **Propriété `line-height`** : Cette propriété est utilisée pour créer de l'espace entre les lignes de texte. Les valeurs acceptées pour `line-height` incluent le mot-clé `normal`, des nombres, des pourcentages et des unités de longueur comme l'unité `em`.
- **Propriété `list-style-type`** : Cette propriété est utilisée pour spécifier le marqueur d'un élément de liste. Les valeurs acceptables peuvent inclure un cercle, un disque ou un décimal.

Voici un exemple utilisant `list-style-type` pour changer le style des puces :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
ul {
  list-style-type: square;
}
```

:::

- **Propriété `list-style-position`** : Cette propriété est utilisée pour définir la position du marqueur de liste. Les seules deux valeurs acceptables sont inside et outside.

Voici un exemple montrant la différence entre `inside` et `outside` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="inside-list">
  <li>Item with inside position</li>
  <li>Item with inside position</li>
</ul>
<ul class="outside-list">
  <li>Item with outside position</li>
  <li>Item with outside position</li>
</ul>
```

```css
.inside-list {
  list-style-position: inside;
}

.outside-list {
  list-style-position: outside;
}
```

:::

- **Propriété `list-style-image`** : Cette propriété est utilisée pour utiliser une image comme marqueur d'élément de liste. Un cas d'usage courant est d'utiliser la fonction `url` avec une valeur définie à un emplacement d'image valide.

## Espacement des éléments de liste avec `margin`

- En plus de `line-height`, les margins peuvent aussi être utilisées en CSS pour améliorer l'espacement et la lisibilité des éléments de liste.
- Les margins créent de l'espace à l'extérieur de chaque élément `li`, permettant le contrôle de l'écart entre les éléments de liste.
- `margin-bottom` est utilisé pour créer de l'espace en dessous de chaque élément de liste. Par exemple, `margin-bottom: 10px;` créera un écart de 10 pixels en dessous de chaque élément de liste.

Voici un exemple utilisant `margin-bottom` pour espacer les éléments de liste :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="list">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
.list li {
  margin-bottom: 20px;
}
```

:::

## Mise en forme des liens

- **`pseudo-class`** : C'est un mot-clé ajouté à un sélecteur qui te permet de sélectionner des éléments en fonction d'un état particulier. Les états courants incluent les états `:hover`, `:visited` et `:focus`.
- **Pseudo-classe `:link`** : Cette pseudo-classe est utilisée pour styliser les liens qui n'ont pas été visités par l'utilisateur.

Voici un exemple utilisant la pseudo-classe `:link` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:link {
  color: red;
}
```

:::

- **Pseudo-classe `:visited`** : Cette pseudo-classe est utilisée pour styliser les liens qu'un utilisateur a déjà visités.
- **Pseudo-classe `:hover`** : Cette pseudo-classe est utilisée pour styliser les éléments quand un utilisateur passe activement la souris dessus.

Voici un exemple utilisant la pseudo-classe `:hover` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Hover over me!</a>
```

```css
a:hover {
  color: green;
  text-decoration: underline;
}
```

:::

- **Pseudo-classe `:focus`** : Cette pseudo-classe est utilisée pour styliser un élément quand il reçoit le focus. Des exemples incluraient les éléments `input` ou `select` quand l'utilisateur clique ou tabule dessus pour le focaliser.

Voici un exemple utilisant la pseudo-classe `:focus` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:focus {
  outline: 2px solid orange;
}
```

:::

- **Pseudo-classe `:active`** : Cette pseudo-classe est utilisée pour styliser un élément qui a été activé par l'utilisateur. Un exemple courant serait quand l'utilisateur clique sur un bouton.

Voici un exemple utilisant la pseudo-classe `:active` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Click me!</a>
```

```css
a:active {
  color: pink;
}
```

:::

## Travailler avec les arrière-plans et les bordures

- **Propriété `background-size`** : Cette propriété est utilisée pour définir la taille d'arrière-plan d'un élément. Quelques valeurs courantes incluent `cover` pour que l'image d'arrière-plan couvre tout l'élément et `contain` pour que l'image d'arrière-plan tienne dans l'élément.

Voici un exemple utilisant `background-size: cover` :

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Voici un exemple utilisant `background-size: contain` :

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **Propriété `background-repeat`** : Cette propriété est utilisée pour déterminer comment les images d'arrière-plan devraient se répéter le long des axes horizontal et vertical. La valeur par défaut pour `background-repeat` est `repeat`, ce qui signifie que l'image se répétera à la fois horizontalement et verticalement. Tu peux aussi spécifier qu'il ne devrait y avoir aucune répétition en utilisant la propriété `no-repeat`.

Voici un exemple utilisant `background-repeat: no-repeat` :

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **Propriété `background-position`** : Cette propriété est utilisée pour spécifier la position de l'image d'arrière-plan. Elle peut être définie à une longueur spécifique, un pourcentage, ou des valeurs de mots-clés comme `top`, `bottom`, `left`, `right` et `center`.

Voici un exemple utilisant `background-position` :

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

- **Propriété `background-attachment`** : Cette propriété est utilisée pour spécifier si l'image d'arrière-plan devrait défiler avec le contenu ou rester fixe en place. Les valeurs principales sont `scroll` (par défaut), où l'image d'arrière-plan défile avec le contenu, et `fixed`, où l'image d'arrière-plan reste à la même position sur l'écran.
- **Propriété `background-image`** : Cette propriété est utilisée pour définir l'image d'arrière-plan d'un élément. Tu peux définir plusieurs images d'arrière-plan en même temps et utiliser soit les fonctions `url`, `radial-gradient` ou `linear-gradient` comme valeurs.

Voici un exemple utilisant `background-image` :

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **Propriété `background`** : C'est la propriété raccourcie pour définir toutes les propriétés d'arrière-plan en une seule déclaration. Voici un exemple de définition de l'image d'arrière-plan et de réglage pour qu'elle ne se répète pas : `background: no-repeat url("example-url-goes-here");`

Voici un exemple utilisant la propriété raccourcie `background` :

:::interactive_editor

```html
<style>
  body {
    background: center top no-repeat url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **Bon contraste pour les couleurs d'arrière-plan et de premier plan** : Il est important de s'assurer que les couleurs d'arrière-plan et de premier plan ont un bon contraste pour rendre le texte lisible. Les Web Content Accessibility Guidelines (WCAG) recommandent un ratio de contraste minimum de 4,5:1 pour le texte normal et 3:1 pour le grand texte.

## Bordures

- **Propriété `border-top`** : Cette propriété est utilisée pour définir les styles pour la bordure supérieure d'un élément. `border-top: 3px solid blue;` définit une bordure bleue continue de 3 pixels de large sur le côté supérieur de l'élément.
- **Propriété `border-right`** : Cette propriété est utilisée pour définir les styles pour la bordure droite d'un élément. `border-right: 2px solid red;` définit une bordure rouge continue de 2 pixels de large sur le côté droit de l'élément.
- **Propriété `border-bottom`** : Cette propriété est utilisée pour définir les styles pour la bordure inférieure d'un élément. `border-bottom: 1px dashed green;` définit une bordure verte en pointillés de 1 pixel de large sur le côté inférieur de l'élément.
- **Propriété `border-left`** : Cette propriété est utilisée pour définir les styles pour la bordure gauche d'un élément. `border-left: 4px dotted orange;` définit une bordure orange en points de 4 pixels de large sur le côté gauche de l'élément.

Voici un exemple utilisant les propriétés de bordure individuelles :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="bordered-box">Box with different borders</div>
```

```css
.bordered-box {
  border-top: 3px solid blue;
  border-right: 2px solid red;
  border-bottom: 1px dashed green;
  border-left: 4px dotted orange;
  padding: 20px;
}
```

:::

- **Propriété `border`** : C'est la propriété raccourcie pour définir la largeur, le style et la couleur de la bordure d'un élément. `border: 1px solid black;` définit une bordure noire continue de 1 pixel de large.

Voici un exemple utilisant la propriété raccourcie `border` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

- **Propriété `border-radius`** : Cette propriété est utilisée pour créer des coins arrondis pour la bordure d'un élément.

Voici un exemple utilisant `border-radius` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

- **Propriété `border-style`** : Cette propriété est utilisée pour définir le style de la bordure d'un élément. Quelques valeurs acceptées incluent `solid`, `dashed`, `dotted` et `double`.

Voici un exemple utilisant différentes valeurs de `border-style` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="solid-border">Solid border</div>
<div class="dashed-border">Dashed border</div>
<div class="dotted-border">Dotted border</div>
<div class="double-border">Double border</div>
```

```css
.solid-border {
  border: 3px solid blue;
  margin-bottom: 10px;
  padding: 10px;
}

.dashed-border {
  border: 3px dashed red;
  margin-bottom: 10px;
  padding: 10px;
}

.dotted-border {
  border: 3px dotted green;
  padding: 10px;
}

.double-border {
  border: 3px double blueviolet;
  padding: 10px;
}
```

:::

## Dégradés

- **Fonction `linear-gradient()`** : Cette fonction CSS est utilisée pour créer une transition entre plusieurs couleurs le long d'une ligne droite.

Voici un exemple utilisant `linear-gradient()` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

- **Fonction `radial-gradient()`** : Cette fonction CSS crée une image qui rayonne à partir d'un point particulier, comme un cercle ou une ellipse, et transitionne progressivement entre plusieurs couleurs.

Voici un exemple utilisant `radial-gradient()` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
