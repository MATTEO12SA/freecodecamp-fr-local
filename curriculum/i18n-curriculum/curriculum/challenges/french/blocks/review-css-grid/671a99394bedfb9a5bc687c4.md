## Bases de CSS Grid

- **Définition** : CSS Grid est un système de mise en page en deux dimensions utilisé pour créer des mises en page complexes dans les pages web. Les grilles ont des lignes et des colonnes avec des espaces entre elles. Pour définir une mise en page en grille, tu dois définir la propriété `display` sur `grid`.

```css
.container {
  display: grid;
}
```

- **L'unité `fr` (fractionnaire)** : Cette unité représente une fraction de l'espace dans le conteneur de grille. Tu peux utiliser l'unité `fr` pour créer des grilles flexibles.
- **Créer des espaces entre les pistes** : Il existe trois façons de créer des espaces entre les pistes dans CSS Grid. Tu peux utiliser la propriété `column-gap` pour créer des espaces entre les colonnes. Tu peux utiliser la propriété `row-gap` pour créer des espaces entre les lignes. Ou tu peux utiliser la propriété raccourcie `gap` pour créer des espaces entre les lignes et les colonnes.
- **`grid-template-columns`** : Cette propriété sert à définir les noms de lignes et la taille des pistes de colonnes de la grille.

```css
.container {
  display: grid;
  width: 100%;
  grid-template-columns: 30px 1fr;
}
```

- **`grid-template-rows`** : Cette propriété sert à définir les noms de lignes et la taille des pistes de rangées de la grille.
- **`grid-auto-flow`** : Cette propriété détermine comment les éléments placés automatiquement s'insèrent dans la grille.

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-flow: column;
}
```

- **`grid-auto-columns`** : Cette propriété sert à définir la taille des colonnes créées implicitement.

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-columns: auto;
}
```

- **`place-items`** : Cette propriété sert à aligner les éléments à la fois dans les directions de bloc et en ligne.
- **`align-items`** : Cette propriété sert à définir l'alignement des éléments dans un conteneur de grille.
- **Fonction `repeat()`** : Cette fonction sert à répéter des sections dans la liste des pistes. Au lieu d'écrire `grid-template-columns: 1fr 1fr 1fr;`, tu peux utiliser la fonction `repeat()`.

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

- **Grille explicite** : Tu peux préciser le nombre de lignes ou de pistes avec les propriétés `grid-template-columns` ou `grid-template-rows`.
- **Grille implicite** : Quand des éléments sont placés en dehors de la grille, des lignes et colonnes sont automatiquement créées pour ces éléments extérieurs.
- **Fonction `minmax()`** : Cette fonction sert à définir les tailles minimale et maximale d'une piste.

```css
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: minmax(150px, auto);
}
```

- **Placement basé sur les lignes** : Toutes les grilles ont des lignes. Pour préciser où l'élément commence sur une ligne, tu peux utiliser les propriétés `grid-column-start` et `grid-row-start`. Pour préciser où l'élément se termine sur une ligne, tu peux utiliser les propriétés `grid-column-end` et `grid-row-end`. Tu peux aussi choisir d'utiliser les propriétés raccourcies `grid-column` ou `grid-row`.

Voici un exemple qui utilise la propriété `grid-column` pour faire s'étendre un élément sur toutes les colonnes.

```css
.element {
  grid-column: 1 / -1;
}
```

- **`grid-template-areas`** : Cette propriété sert à donner un nom aux éléments que tu vas positionner sur la grille.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="header">Header</div>
  <div class="sidebar">Sidebar</div>
  <div class="main">Main Content</div>
  <div class="footer">Footer</div>
</div>

```

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr; 
  grid-template-rows: auto 1fr auto; 
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer"; 
  gap: 20px; 
}

.header {
  grid-area: header; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}

.sidebar {
  grid-area: sidebar;
  background-color: #f4f4f4;
  padding: 10px;
}

.main {
  grid-area: main; 
  background-color: #e0e0e0;
  padding: 10px;
}

.footer {
  grid-area: footer; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}
```

:::

## Déboguer le CSS

- **DevTools (outils de développement)** : Les DevTools te permettent d'inspecter et de modifier ton CSS en temps réel. Le panneau Styles montre toutes les règles CSS appliquées à l'élément sélectionné, y compris les styles hérités. Tu peux activer ou désactiver des propriétés individuellement, modifier des valeurs et même ajouter de nouvelles règles directement dans le navigateur. Ce retour immédiat est extrêmement utile pour tester différents styles sans modifier ton code source.
- **Validateurs CSS** : Les validateurs servent à vérifier ton CSS par rapport aux spécifications officielles et signalent les erreurs ou avertissements. Le validateur CSS du W3C est un validateur populaire que tu peux utiliser.
- **Déboguer les designs responsives** : Les DevTools ont une option qui te permet de simuler l'apparence de ton site sur différentes tailles d'écran et différents appareils. Cela peut t'aider à identifier des problèmes de points de rupture ou des styles qui ne s'adaptent pas bien à différentes tailles de viewport.
