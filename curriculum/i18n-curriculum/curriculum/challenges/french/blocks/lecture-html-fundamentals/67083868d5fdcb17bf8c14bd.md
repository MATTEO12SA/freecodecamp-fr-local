# --interactive--

Une entité HTML, ou référence de caractère, est un ensemble de caractères utilisé pour représenter un caractère réservé en HTML.

Imaginons que tu veuilles afficher le texte `This is an <img/> element` à l'écran. Si tu utilises le code actuellement dans l'éditeur, il n'affichera pas le résultat voulu. Même si tu ajoutais des attributs `src` et `alt` à l'exemple, il afficherait une image au milieu du paragraphe. Ce n'est pas le résultat souhaité. Pour interagir avec l'exemple, tu dois activer l'éditeur interactif.

:::interactive_editor

```html
<p>This is an <img /> element</p>
```

:::

Quand l'analyseur HTML voit le symbole inférieur à (`<`) suivi d'un nom de balise HTML, il l'interprète comme un élément HTML. C'est pour cela que tu n'obtiens pas le résultat voulu, `This is an <img/> element`, à l'écran.

Pour corriger ce problème, tu peux utiliser des entités HTML. Voici un exemple mis à jour qui utilise les bonnes entités HTML pour les symboles inférieur à (`<`) et supérieur à (`>`). Tu devrais maintenant voir `This is an <img/> element` à l'écran.

Active l'éditeur interactif et essaie d'ajouter `&lt;p&gt;learning is fun&lt;/p&gt;` sous l'élément paragraphe. Tu devrais voir `<p>learning is fun</p>` à l'écran.

:::interactive_editor

```html
<p>This is an &lt;img /&gt; element</p>

```

:::

Ces types de références de caractères sont appelés références de caractères nommées. Les références nommées commencent par une esperluette (`&`) et se terminent par un point-virgule (`;`). En utilisant une référence de caractère nommée, l'analyseur HTML ne la confondra pas avec un véritable élément HTML.

Un autre type de référence de caractère est la référence numérique décimale. Cette référence commence par une esperluette et un symbole dièse (`#`), suivis d'un ou plusieurs chiffres décimaux, puis d'un point-virgule.

Voici un exemple utilisant la référence numérique décimale pour le symbole inférieur à.

Active l'éditeur interactif et modifie le code pour voir différents symboles. Tu peux utiliser `&#169;` pour le symbole copyright et `&#174;` pour le symbole de marque déposée.

:::interactive_editor

```html
&#60;
```

:::

Le dernier type de référence de caractère est la référence numérique hexadécimale. Cette référence commence par une esperluette, un symbole dièse et la lettre `x`. Elle est ensuite suivie d'un ou plusieurs chiffres hexadécimaux ASCII et se termine par un point-virgule.

Voici un exemple utilisant la référence numérique hexadécimale pour le symbole inférieur à.

Active l'éditeur interactif et modifie le code pour voir différents symboles. Tu peux utiliser `&#x20AC;` pour le symbole euro et `&#x03A9;` pour la lettre grecque majuscule Oméga.

:::interactive_editor

```html
&#x3C;
```

:::
