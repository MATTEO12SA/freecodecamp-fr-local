# --description--

Pour créer un fichier sur Windows avec l'Explorateur, tu peux cliquer sur l'option « Nouveau » en haut à gauche. Ça t'affichera une liste des différents types de fichiers que tu peux créer.

Tu peux aussi créer des fichiers dans leurs applications correspondantes. Créons un fichier texte pour te montrer le processus.

Quand tu cliques sur « Nouveau » et que tu sélectionnes le type de fichier, tu verras un nouveau fichier texte dans ton dossier actuel avec un nom par défaut. Tu peux le renommer si tu veux.

Le processus de création de dossiers est exactement le même. Clique sur le bouton « Nouveau » en haut à gauche et sélectionne l'option « Dossier », ou clique droit sur un espace vide et va dans « Nouveau ». Puis sélectionne l'option « Dossier ». Tu verras un nouveau dossier vide à ton emplacement actuel.

Tu peux renommer des fichiers et dossiers en les sélectionnant et en cliquant sur l'option « Renommer » en haut. Quand le nom actuel du fichier est mis en surbrillance, tu peux commencer à écrire le nouveau nom du fichier. Appuie sur Entrée pour confirmer les modifications.

Alternativement, tu peux cliquer droit sur le fichier ou le dossier et sélectionner l'option « Renommer », ou utiliser le raccourci clavier F2. Écris le nouveau nom et appuie sur Entrée pour confirmer les modifications.

Tu peux déplacer un fichier ou un dossier dans un autre dossier en le glissant-déposant dans le dossier de destination, si tu l'as ouvert comme un onglet.

Tu peux aussi sélectionner le fichier ou le dossier et choisir l'option « Couper » en haut.

Ensuite, va dans le dossier de destination et colle-le en cliquant sur le bouton « Coller » en haut. Si tu as copié ou coupé le fichier précédemment, ce bouton sera activé.

Tu peux aussi le faire en cliquant droit sur un espace vide puis en cliquant sur l'icône coller. Ça supprimera le fichier ou le dossier de son emplacement d'origine.

Pour supprimer un fichier ou un dossier, clique dessus puis sur l'icône Corbeille en haut. Tu trouveras aussi une option « Supprimer » si tu cliques droit dessus.

Super. Maintenant, voyons comment faire ça sur macOS avec le Finder.

Le Finder est le gestionnaire de fichiers par défaut sur macOS.

Pour créer un fichier sur macOS, tu dois ouvrir une application qui te permet de créer le type de document dont tu as besoin. Actuellement, il n'y a pas de moyen intégré de créer un fichier directement dans le Finder.

Par exemple, pour créer un simple fichier texte brut, tu peux ouvrir TextEdit et enregistrer ton fichier. Le processus fonctionne exactement de la même manière pour tout type de fichier.

Il existe plusieurs façons d'ouvrir TextEdit. L'une d'elles est d'aller dans le Launchpad. Puis, recherche une application où tu peux créer le type de fichier qui t'intéresse. Dans ce cas, ce serait « TextEdit ». Clique sur l'icône pour ouvrir l'application.

Une fois dans l'application, tu devrais voir une option pour enregistrer ton fichier dans le menu Fichier. Ce menu se trouve en haut à gauche, à côté de l'icône du menu Apple.

Dans l'une des options du menu de l'application, tu devrais voir une commande pour enregistrer le fichier. C'est généralement sous le menu Fichier.

À côté, tu verras le raccourci clavier pour enregistrer ton fichier. C'est généralement `Commande + S` sur macOS.

Quand tu enregistres le fichier pour la première fois, tu devrais voir une fenêtre de dialogue où tu peux écrire le nom du fichier et choisir un emplacement.

Habituellement, l'extension du fichier est ajoutée automatiquement par l'application, mais tu peux la modifier avant d'enregistrer le fichier.

Après avoir enregistré le fichier, tu peux ouvrir le Finder en cliquant sur l'icône du Finder dans le Dock. Tu devrais voir ton nouveau fichier dans le dossier où tu l'as enregistré.

Pour créer un nouveau dossier, tu dois aller à l'emplacement où tu veux créer ce nouveau dossier et cliquer droit sur un espace vide. Tu verras une liste d'options. La première option est « Nouveau dossier ».

Si tu cliques dessus, tu peux attribuer un nom à ton nouveau dossier. Écris le nom et appuie sur Entrée pour confirmer.

Il existe plusieurs façons de déplacer des fichiers et dossiers dans le Finder. Si tu dois les déplacer vers un dossier listé dans la section « Favoris » du Finder, tu peux simplement les glisser-déposer.

Si tu dois le déplacer dans un autre dossier, tu as deux options.

Tu peux soit ouvrir le dossier de destination comme un onglet et glisser-déposer le fichier ou le dossier dans l'onglet.

Ou tu peux utiliser des raccourcis clavier. D'abord, copie le fichier avec `Commande + C`, va dans le nouveau dossier, puis utilise `Commande + Option + V` pour coller le fichier ou le dossier à cet endroit. Ça le supprimera aussi du dossier d'origine.

Pour supprimer un fichier ou un dossier, tu peux cliquer droit dessus et sélectionner « Déplacer vers la corbeille ».

Tu peux aussi les glisser-déposer dans l'icône de la corbeille du Dock. C'est équivalent.

Quand le fichier ou le dossier est dans la corbeille, tu peux cliquer droit dessus pour le supprimer immédiatement, ou tu peux vider la corbeille pour supprimer définitivement tous les fichiers et dossiers supprimés.

Savoir créer, déplacer et supprimer des fichiers et dossiers sur Windows et macOS est très important. En maîtrisant ces compétences, tu peux organiser tes fichiers, les retrouver rapidement et libérer de l'espace de stockage sur ton ordinateur en supprimant les fichiers inutiles.
