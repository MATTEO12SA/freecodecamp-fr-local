# --interactive--

L'élément `time` est utilisé pour représenter un moment précis dans le temps.

Voici un exemple qui utilise l'élément `time` pour représenter vingt heures, soit huit heures du soir.

:::interactive_editor

```html
<p>The reservations are for <time datetime="20:00">20:00 </time></p>
```

:::

L'attribut `datetime` est utilisé pour traduire les dates et les heures dans un format lisible par une machine.

C'est important, parce que ça aide pour les résultats des moteurs de recherche et ça aide le navigateur à traiter les informations de date et d'heure plus efficacement.

La valeur de l'attribut `datetime` doit être soit une année valide, un mois valide, une heure valide, une date locale, une date globale, ou une chaîne de durée valide.

Voici un autre exemple qui utilise l'élément time pour représenter une date particulière :

:::interactive_editor

```html
<p>
  The graduation will be on <time datetime="2024-06-15T15:00">June 15</time>
</p>
```

:::

La valeur de l'attribut `datetime` est au format ISO 8601. ISO 8601 est une norme internationale pour représenter les dates et les heures.

La première partie de cette valeur est l'année, le mois et le jour. Le T majuscule dans la valeur est un séparateur entre la date et l'heure.

Quinze heures correspondrait à trois heures de l'après-midi.

Chaque fois que tu as besoin de représenter des événements, des dates de publication, ou des rendez-vous, il vaut mieux utiliser l'élément `time`.
