# --interactive--

L'élément d'adresse de contact est utilisé pour représenter les informations de contact d'une section d'une page web. L'élément `address` est polyvalent et peut être utilisé pour des pages d'entreprise, des pages d'auteur, des sites personnels, et plus.

Quand il s'agit de construire les sections de contact de ton site web, tu devrais utiliser l'élément sémantique `address` plutôt qu'un élément générique comme un `div`.

Voici un exemple d'utilisation de l'élément `address` pour une page de contact d'entreprise :

:::interactive_editor

```html
<address>
  <h2>Company Name</h2>
  <p>
    1234 Elm Street<br />
    Springfield, IL 62701<br />
    United States
  </p>
  <p>Phone: <a href="tel:+15555555555">+1 (555) 555-5555</a></p>
  <p>Email: <a href="mailto:contact@company.com">contact@company.com</a></p>
</address>
```

:::

Dans cet exemple, il y a le nom de l'entreprise, l'adresse physique, le téléphone et l'email. Pour l'adresse physique, l'élément de saut de ligne, `br`, est utilisé pour montrer la séparation entre le nom de rue, la ville et le pays.

Pour le numéro de téléphone, on a un élément d'ancrage avec la valeur `href` définie pour les numéros de téléphone. La valeur `tel:+` à l'intérieur de l'attribut `href` crée un lien cliquable qui lance un appel téléphonique sur certains appareils qui le supportent.

Pour l'adresse email, un autre élément d'ancrage est utilisé avec la valeur `href` définie sur un lien `mailto`. Les liens `mailto` sont utilisés dans les documents HTML pour permettre aux utilisateurs d'ouvrir un nouvel email dans leur client de messagerie préféré.

L'un des inconvénients de l'utilisation d'un lien `mailto`, c'est que les utilisateurs le perçoivent souvent comme du spam. Malheureusement, beaucoup de spammeurs utilisent cette option pour envoyer des emails aux utilisateurs. Garde simplement cela en tête quand tu l'utilises.
