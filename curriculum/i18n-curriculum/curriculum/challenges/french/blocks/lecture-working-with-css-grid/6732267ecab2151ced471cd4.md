# --interactive--

Dans les leçons précédentes, nous avons un peu parlé de la création d'espace entre les éléments de grille. Mais dans cette leçon, nous allons voir plus en détail comment utiliser les propriétés `row-gap`, `column-gap` et `gap` dans une mise en page de grille.

Mais d'abord, il faut revoir ce qu'est une piste dans CSS Grid. Une piste est l'espace entre deux lignes de grille voisines. Ces lignes sont créées automatiquement quand tu utilises CSS Grid. Dans ce contexte, les pistes désignent généralement les lignes et les colonnes qui composent la mise en page de grille.

Pour créer des espaces entre les colonnes dans une grille CSS, tu peux utiliser la propriété `column-gap`. Les valeurs acceptées pour cette propriété incluent les pixels, l'unité `em`, les pourcentages ou le mot-clé `normal`.

Si tu utilises la valeur `normal` pour la propriété `column-gap`, le résultat sera `0` pour les mises en page de grille et `1em` pour les mises en page multi-colonnes.

Voici un exemple de balisage pour une mise en page de grille à quatre colonnes :

```html
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

Pour le CSS, on définit `display` sur `grid` et la propriété `column-gap` sur 10 pixels :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  height: 100px;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.grid-container div {
  background-color: darkblue;
}
```

:::

Si on voulait modifier l'exemple pour avoir deux lignes de boîtes bleues et créer plus d'espace entre les lignes, on pourrait utiliser la propriété `row-gap` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  column-gap: 10px;
  row-gap: 30px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

Dans cet exemple révisé, on définit la propriété `row-gap` sur 30 pixels et on modifie `grid-template-columns` pour utiliser seulement deux unités `1fr` au lieu de quatre, afin de créer deux lignes de boîtes.

Comme pour la propriété `column-gap`, les valeurs acceptées pour la propriété `row-gap` peuvent inclure les pourcentages, `em` et les pixels.

Si tu veux utiliser un raccourci pour créer des espaces entre les lignes et les colonnes, tu peux utiliser la propriété `gap`. Voici la syntaxe de base :

```css
gap: row-value optional-column-value;
```

Si tu précises une seule valeur pour la propriété `gap`, cette valeur sera appliquée à la fois aux lignes et aux colonnes. Si tu précises deux valeurs, la première ira aux lignes et la seconde sera appliquée aux colonnes :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 30px 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

Les valeurs acceptées pour la propriété raccourcie `gap` incluent les pourcentages, les pixels, `em` et même les valeurs `calc()`. Mais tu ne peux pas utiliser d'unités `fr` ici.

Les propriétés `row-gap`, `column-gap` et `gap` fournissent des façons flexibles de contrôler l'espacement entre les éléments dans une mise en page CSS Grid. En utilisant ces propriétés, tu peux facilement créer des grilles visuellement agréables avec des espaces cohérents et ajustables entre les lignes et les colonnes.
