# --interactive--

CSS Grid est un système de mise en page puissant qui permet aux développeurs web de créer facilement des mises en page de pages web complexes et responsives.

Imagine que tu organises des meubles dans une pièce : CSS Grid revient à avoir une grille invisible au sol qui t'aide à placer chaque élément exactement où tu le veux.

Quand on construit des sites web, on doit souvent organiser différents éléments sur la page.

Avant CSS Grid, cela pouvait parfois être délicat, surtout pour les mises en page complexes. CSS Grid simplifie ce processus en divisant ta page web en lignes et colonnes, ce qui crée une structure de type grille.

Imaginons que tu travailles avec un `div` conteneur contenant plusieurs éléments imbriqués comme ceci :

:::interactive_editor

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

:::

Si tu veux styliser ces éléments sous forme de grille, tu peux définir `display` sur `grid` et appliquer des colonnes comme ceci :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-gap: 20px;
}

.item {
  background-color: lightgray;
  padding: 20px;
  text-align: center;
  border: 1px solid #ccc;
}
```

:::

Dans ce code, on demande au navigateur de créer une grille avec trois colonnes de largeur égale. C'est ce que signifie `1fr 1fr 1fr`, et on ajoute un espace de 20 pixels entre chaque élément de grille.

Tu te demandes peut-être : « Et Flexbox ? Ce n'est pas aussi utilisé pour les mises en page ? »

Tu as raison. Flexbox est un autre modèle de mise en page CSS, et il est très utile aussi. Mais il existe quelques différences importantes.

Flexbox est unidimensionnel, tandis que Grid est bidimensionnel. Cela signifie que Flexbox fonctionne très bien pour organiser des éléments sur une seule ligne ou une seule colonne, tandis que Grid excelle pour créer des mises en page avec des lignes et des colonnes.

Flexbox part d'abord du contenu, ce qui signifie qu'il ajuste la mise en page selon le contenu. Grid, en revanche, part d'abord de la mise en page : tu peux créer la structure, puis y placer les éléments. Grid te donne un contrôle plus précis sur le placement. Tu peux indiquer exactement quelle ligne et quelle colonne un élément doit occuper.

Voici un exemple Flexbox pour comparer :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

:::

Cela crée un conteneur flex où les éléments sont espacés régulièrement le long de l'axe principal.

Grid et Flexbox ont tous les deux leurs forces, et les meilleures mises en page utilisent souvent une combinaison des deux. Tu peux utiliser Grid pour la mise en page globale de la page, puis Flexbox pour aligner les éléments à l'intérieur de chaque zone de grille.

En résumé, CSS Grid est un outil puissant qui permet de créer des mises en page bidimensionnelles précises. Même s'il peut sembler complexe au début, avec de la pratique, il devient un outil précieux pour créer des mises en page web responsives et complexes.
