# --interactive--

Les propriétés `grid-column` et `grid-row` te permettent de préciser le placement horizontal et vertical des éléments de grille dans une mise en page en grille.

Autrement dit, elles te permettent toutes les deux de contrôler où un élément de grille commence et se termine en faisant référence aux lignes de grille. Ces lignes de grille sont les limites qui séparent les lignes et les colonnes que tu as déjà définies avec les propriétés `grid-template-rows` et `grid-template-columns`.

Regardons la syntaxe de `grid-row` et de `grid-column`, ainsi qu'un exemple pratique.

Voici la syntaxe de la propriété `grid-row` :

```css
grid-row: <start-line> / <end-line>;
```

Et celle de `grid-column` :

```css
grid-column: <start-line> / <end-line>;
```

`<start-line>` est la ligne de grille où l'élément commence, et `<end-line>` est la ligne de grille où l'élément se termine. Les deux sont indexées à partir de 1, c'est-à-dire que tu commences à compter depuis 1, pas depuis 0.

Souviens-toi que les lignes de grille pour les rangées sont générées selon le nombre de rangées indiqué dans la propriété `grid-template-rows`. La même logique s'applique aux colonnes avec la propriété `grid-template-columns`.

Pour l'exemple pratique, commençons avec les lignes générées par le navigateur quand tu utilises les propriétés `grid-template-rows` et `grid-template-columns`.

Voici un exemple de grille avec 4 colonnes et 3 lignes :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}
```

:::

Si tu inspectes le conteneur de grille (la classe `grid`), tu verras que chaque ligne et chaque colonne est délimitée par deux lignes : une ligne de départ au début de la ligne ou de la colonne, et une ligne de fin à la fin de la ligne ou de la colonne.

Tu peux cibler ces lignes avec les propriétés `grid-row` et `grid-column` pour déterminer où placer un élément.

Voici comment faire en sorte que le premier élément de grille occupe les deux premières colonnes :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
}
```

:::

Avec cela, tu indiques que le premier élément de grille doit commencer à la colonne 1 et se terminer juste avant la colonne 3.

Le premier élément occupera maintenant deux colonnes, et le quatrième élément sera poussé sur la deuxième ligne. Super !

Si tu veux aussi que le premier élément occupe deux lignes, tu peux définir `grid-row` sur `1 / 3` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}
```

:::

Tu peux aussi utiliser le mot-clé `span` pour indiquer à l'élément de grille combien de lignes et de colonnes il doit couvrir. Par exemple, `1 / 3` revient au même que `1 / span 2` :

:::interactive_editor

```html

<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / span 2;
  grid-row: 1 / span 2;
}
```

:::

Tu peux continuer à appliquer cette technique à n'importe quel élément de la grille et le placer où tu veux.
