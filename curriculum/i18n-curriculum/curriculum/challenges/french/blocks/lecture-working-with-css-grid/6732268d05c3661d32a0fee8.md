# --interactive--

Dans les leçons précédentes, nous avons travaillé avec la propriété `grid-template-columns` et défini sa valeur avec quelques unités fractionnaires.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

Même si le code suivant est totalement valide, il existe une façon plus simple de répéter une section ou toutes tes listes de pistes.

La fonction `repeat()` est utilisée pour répéter une section ou toutes les pistes des colonnes ou des lignes. Cette fonction reçoit un nombre de répétitions et les pistes que tu veux répéter.

Voici une version révisée de l'exemple précédent qui utilise la fonction `repeat()` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

Il n'y aura pas de changement dans les styles affichés par le navigateur, mais c'est une façon plus concise d'écrire des valeurs répétées pour les colonnes.

La fonction `repeat()` accepte n'importe quel motif valide que tu peux utiliser pour les lignes ou les colonnes.

Voici un exemple qui utilise la fonction `repeat()` pour définir les première et troisième colonnes sur 20 pixels, et les deuxième et quatrième colonnes sur une unité fractionnaire.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

Parfois, tu peux choisir d'écrire chaque valeur individuellement au lieu d'utiliser la fonction `repeat()`. Mais il y a des cas où cette fonction est pratique, surtout quand tu veux répéter un motif particulier dans une liste de pistes.
