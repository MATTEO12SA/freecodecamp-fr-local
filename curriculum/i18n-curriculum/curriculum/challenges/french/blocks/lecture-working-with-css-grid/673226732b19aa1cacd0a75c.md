# --interactive--

Dans la leçon précédente, tu as découvert CSS Grid, qui peut servir à créer des mises en page complexes et fluides dans tes pages web. Dans cette leçon, nous allons voir comment créer des mises en page de grille flexibles avec l'unité `fr`.

Commençons avec ce balisage HTML qui représentera notre conteneur de grille :

```html
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

Dans le CSS, on définit la propriété `display` du conteneur sur `grid`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

`grid-template-columns` sert à définir la taille de chaque colonne. Dans ce cas, chaque colonne aura une taille de `25%` du conteneur. Ensuite, la propriété `gap` sert à créer de l'espace autour de chaque colonne.

Jusqu'ici, nous avons utilisé des pourcentages pour la taille des colonnes, mais nous pouvons aussi utiliser l'unité `fr`.

L'unité `fr` est une unité fractionnaire qui représente une fraction de l'espace du conteneur de grille.

Voici à quoi ressemble le code quand il est refactorisé pour utiliser des unités `fr` au lieu de pourcentages.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

Chaque colonne occupera une fraction de l'espace disponible. Comme il y a quatre colonnes, chaque colonne aura une part égale de l'espace disponible dans le conteneur de grille.

Quand tu commenceras à construire tes mises en page de grille, tu voudras utiliser les unités `fr` plus souvent, car elles fournissent une façon flexible et proportionnelle de répartir l'espace. Elles te permettent de créer des mises en page responsives qui s'adaptent à différentes tailles d'écran sans avoir besoin d'ajuster manuellement des valeurs en pixels.
