# --interactive--

La propriété `grid-template-areas` te permet de concevoir une mise en page visuelle en grille avec des libellés nommés.

Tu assignes ensuite ces libellés à des éléments de grille spécifiques avec la propriété `grid-area`. Autrement dit, ces libellés nommés sont aussi appelés « noms de zones de grille ».

Voici la syntaxe de base de la propriété `grid-template-areas` :

```css
grid-template-areas:
 'header header header'
 'left-sidebar main right-sidebar'
 'footer footer footer';
```

Voici les points à retenir de cette syntaxe de base :

- Des valeurs comme `header` et `main` sont les noms des zones de grille.

- Chaque valeur séparée par un espace dans une chaîne correspond à une colonne.

- Chaque chaîne représente une ligne dans la grille.

Donc, dans cette syntaxe, nous avons un conteneur de grille de 3 par 3.

Après avoir défini le modèle, tu dois utiliser la propriété `grid-area` et les libellés nommés (ou zones) comme valeurs pour indiquer à CSS qu'un élément précis appartient à cette zone de la grille.

La propriété `grid-area` relie l'élément de grille à la région nommée que tu définis dans `grid-template-areas`.

Une façon populaire de montrer les possibilités de la propriété `grid-template-areas` est de créer la mise en page classique du Holy Grail.

Si tu ne sais pas ce que c'est, la mise en page Holy Grail est un modèle de design web avec un en-tête, un pied de page, deux barres latérales et une zone de contenu principale. Elle garantit que le contenu principal reste prioritaire, tandis que les barres latérales et les autres sections s'ajustent de façon responsive dans la mise en page.

Il existe de nombreuses solutions pour implémenter la mise en page Holy Grail, mais utiliser `grid-template-areas` et la propriété `grid-area` est la façon la plus directe de la créer.

Voici un exemple de mise en page Holy Grail :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="header">
    <h2>Header</h2>
  </div>
  <div class="sidebar-left">
    <h2>Left Sidebar</h2>
  </div>
  <div class="main"><h2>Main Content</h2></div>
  <div class="sidebar-right">
    <h2>Right Sidebar</h2>
  </div>
  <div class="footer">
    <h2>Footer</h2>
  </div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-areas:
    'header header header'
    'sidebar-left main sidebar-right'
    'footer footer footer';
  grid-gap: 10px;
  background-color: #2196F3;
  padding: 10px;
}

.header {
  grid-area: header;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-left {
  grid-area: sidebar-left;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.main {
  grid-area: main;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-right {
  grid-area: sidebar-right;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.footer {
  grid-area: footer;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}
```

:::

Note que les propriétés `grid-template-areas` et `grid-area` peuvent être utilisées indépendamment l'une de l'autre.

La propriété `grid-template-areas` sert précisément à définir une mise en page visuelle en dessinant des zones de grille nommées dans le conteneur de grille.

De son côté, la propriété `grid-area` sert à positionner des éléments de grille individuels, soit en précisant leurs positions de ligne et de colonne, soit en faisant référence aux zones nommées définies avec la propriété `grid-template-areas`.
