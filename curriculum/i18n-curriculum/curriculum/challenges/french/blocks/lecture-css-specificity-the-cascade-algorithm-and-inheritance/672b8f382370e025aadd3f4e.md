# --interactive--

L'héritage est un concept clé en CSS qui détermine comment les styles sont transmis des éléments parents à leurs éléments enfants.

Tout comme dans le monde réel, où les enfants héritent souvent des traits de leurs parents, en CSS, certaines propriétés peuvent être héritées par les éléments enfants de leurs éléments parents.

Cela permet une façon plus efficace d'appliquer une mise en forme cohérente sur un document entier.

En CSS, toutes les propriétés ne sont pas héritées par défaut. Par exemple, des propriétés comme `color`, `font-family` et `line-height` sont héritées. Cela signifie que si tu définis la couleur du texte sur un élément parent, tous ses éléments enfants hériteront de cette couleur sauf si tu la remplaces spécifiquement.

Par exemple, considère l'exemple suivant où un élément `div` parent a sa couleur définie en utilisant un style inline, et l'élément `p` enfant hérite de cette couleur :

:::interactive_editor

```html
<div style="color: blue;">
  This is the parent element.
  <p>This is the child element inheriting the color.</p>
</div>
```

:::

Dans ce cas, le `div` parent et le `p` enfant afficheront tous les deux leur texte en bleu parce que la couleur est héritée.

Par contre, des propriétés comme `margin`, `padding`, `border` et `background` ne sont pas héritées par défaut. Si tu veux qu'un élément enfant hérite de ces styles, tu dois les définir explicitement, soit directement sur l'élément enfant, soit en utilisant le mot-clé `inherit`.

Le mot-clé `inherit` peut être utilisé pour forcer l'héritage d'une propriété depuis un élément parent, même si cette propriété n'est normalement pas héritée.

Par exemple, si tu veux qu'un élément enfant spécifique ait le même `padding` que son parent, tu peux définir `padding: inherit` sur l'élément enfant :

:::interactive_editor

```html
<div style="padding: 20px;">
  This is the parent element with padding.
  <p style="padding: inherit;">This is the child element inheriting the padding.</p>
</div>
```

:::

Dans ce cas, l'élément `p` enfant héritera des `20px` de padding de son élément `div` parent.

L'héritage est particulièrement utile pour maintenir la cohérence et réduire la redondance dans tes feuilles de style.

Au lieu d'écrire la même règle de style pour plusieurs éléments, tu peux la définir une fois sur un élément parent, et les éléments enfants en hériteront. Cela peut rendre ton CSS plus concis et plus facile à gérer.

Cependant, il est important de te rappeler que l'héritage ne fonctionne que dans une direction – du parent à l'enfant. Si tu remplaces un style sur un élément enfant, ça n'affectera pas l'élément parent.
