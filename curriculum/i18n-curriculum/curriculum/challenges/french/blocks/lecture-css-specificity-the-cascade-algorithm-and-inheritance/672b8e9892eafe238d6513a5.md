# --interactive--

Les sélecteurs de classe sont un élément clé du CSS, permettant aux développeurs de cibler plusieurs éléments avec le même attribut de classe et d'appliquer une mise en forme cohérente.

Cela les rend très polyvalents et efficaces pour appliquer des styles sur un site web.

Les sélecteurs de classe sont définis par un point (`.`) suivi du nom de la classe. Ils peuvent être appliqués à n'importe quel élément du document HTML.

Voici un exemple utilisant un sélecteur de classe :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="highlight">Example paragraph</p>
```

```css
.highlight {
  color: green;
}
```

:::

Dans cet exemple, tout élément avec la classe `highlight` aura sa couleur de texte définie à `green`.

Les sélecteurs de classe ont une spécificité plus élevée que les sélecteurs de type mais plus faible que les sélecteurs ID et les styles inline.

La valeur de spécificité pour un sélecteur de classe est `(0, 0, 1, 0)`. Cela signifie que les sélecteurs de classe peuvent remplacer les sélecteurs de type, mais peuvent être remplacés par les sélecteurs ID et les styles inline.

Les sélecteurs de classe peuvent être combinés avec d'autres sélecteurs pour créer des règles plus spécifiques.

Voici un exemple combinant un sélecteur de type paragraphe avec un sélecteur de classe :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="bold-text">Example paragraph</p>
<p class="bold-text">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p.bold-text {
  font-weight: bold;
}
```

:::

Cette règle s'applique uniquement aux éléments `p` qui ont aussi la classe `bold-text`, rendant leur texte en gras.
