# --interactive--

Les sélecteurs de type, aussi appelés sélecteurs d'élément, ciblent les éléments en fonction de leur nom de balise.

Ces sélecteurs sont fondamentaux en CSS et te permettent d'appliquer des styles à toutes les instances d'un élément HTML spécifique.

Les sélecteurs de type sont simples à utiliser et sont écrits simplement comme le nom de la balise de l'élément que tu veux styliser.

Voici un exemple de sélecteur de type ciblant tous les éléments paragraphe de la page :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  color: blue;
}
```

:::

Dans cet exemple, tous les éléments `p` auront leur couleur de texte définie à `blue`.

Les sélecteurs de type ont une spécificité relativement faible par rapport aux autres sélecteurs. La valeur de spécificité pour un sélecteur de type est `(0, 0, 0, 1)`.

Cela signifie que les sélecteurs de type seront remplacés par les sélecteurs de classe, les sélecteurs ID et les styles inline, mais peuvent toujours appliquer des styles à moins que ces règles de plus haute spécificité ne soient présentes.

Regardons un exemple où les sélecteurs de classe remplaceront les styles du sélecteur de type.

Voici un exemple avec deux éléments paragraphe :

:::interactive_editor

```html
<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

:::

Les deux éléments paragraphe ont une classe appelée `para`.

À l'intérieur du fichier CSS, le sélecteur de type cible les paragraphes, et le sélecteur de classe cible les éléments avec la classe `para`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

```css
p {
  color: blue;
}

.para {
  color: red;
}
```

:::

Tous les paragraphes de la page avec la classe `para` auront la couleur du texte définie à `red` au lieu de `blue` parce que les sélecteurs de classe ont une spécificité plus élevée que les sélecteurs de type.
