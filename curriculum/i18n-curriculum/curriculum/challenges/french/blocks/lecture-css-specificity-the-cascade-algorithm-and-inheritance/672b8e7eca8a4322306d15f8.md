# --interactive--

Le sélecteur universel (`*`) est un type spécial de sélecteur CSS qui correspond à n'importe quel élément du document.

Il est souvent utilisé pour appliquer un style à tous les éléments de la page, ce qui peut être utile pour réinitialiser ou normaliser les styles entre différents navigateurs.

Le sélecteur universel peut être utilisé pour sélectionner tous les éléments dans un contexte spécifique ou globalement dans tout le document.

Voici un exemple d'utilisation du sélecteur universel pour définir le `margin` et le `padding` pour tout le document HTML :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1>Heading element</h1>
<p>example paragraph element</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

Dans cet exemple de code, le sélecteur `*` réinitialise le margin et le padding de tous les éléments à zéro, ce qui est une technique courante utilisée dans les resets CSS.

Le sélecteur universel a la valeur de spécificité la plus faible de tous les sélecteurs. Il contribue 0 à toutes les parties de la valeur de spécificité `(0, 0, 0, 0)`.

Cela signifie que tout autre sélecteur, y compris les sélecteurs de type, les sélecteurs de classe, les sélecteurs ID et les styles inline, remplacera les styles définis par le sélecteur universel.

Regardons l'exemple HTML et CSS suivant :

:::interactive_editor

```html
<head>
  <style>
    * {
      color: blue;
    }
    p {
      color: red;
    }
    .highlight {
      color: green;
    }
    #unique {
      color: purple;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text has multiple styles applied.</p>
</body>
```

:::

Le sélecteur universel `*` définit la couleur du texte à `blue` pour tous les éléments.

Cependant, le sélecteur de type `p` remplace ça en définissant la couleur du texte à `red` spécifiquement pour les éléments `p`.

Si un élément a la classe `highlight`, le sélecteur de classe prend la priorité, changeant la couleur du texte à `green`.

Enfin, le sélecteur ID `#unique`, qui a la spécificité la plus élevée, remplacera tous les autres, définissant la couleur du texte à `purple`.
