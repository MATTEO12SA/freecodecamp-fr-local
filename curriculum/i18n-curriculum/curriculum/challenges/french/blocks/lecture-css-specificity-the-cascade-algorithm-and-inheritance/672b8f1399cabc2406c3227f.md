# --interactive--

Le mot-clé `!important` en CSS est utilisé pour donner à une règle de style la priorité la plus élevée, lui permettant de remplacer toutes les autres déclarations pour une propriété.

Quand il est utilisé, il force le navigateur à appliquer le style spécifié, indépendamment de la spécificité des autres sélecteurs.

Disons que tu as un élément paragraphe HTML avec des styles inline comme ceci :

:::interactive_editor

```html
<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

:::

Dans cet exemple, l'élément paragraphe a une couleur d'arrière-plan définie à `lightblue` et une couleur de texte définie à `black`.

Le fichier CSS applique les styles suivants à l'élément paragraphe :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black;
  color: white;
}
```

:::

Étant donné que les styles inline ont une précédence plus élevée que les sélecteurs de classe, ID et de type, la couleur d'arrière-plan noire et la couleur de texte blanche ne seront pas appliquées à cet élément paragraphe.

Pour remplacer ces styles inline, tu peux utiliser le mot-clé `!important` comme ceci :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black !important;
  color: white !important;
}
```

:::

Le mot-clé `!important` est utilisé après la valeur CSS et avant le point-virgule.

Maintenant, l'élément paragraphe aura ces couleurs noires et blanches appliquées parce que les styles inline sont remplacés avec l'utilisation du mot-clé `!important`.

Le mot-clé `!important` en CSS est utilisé pour donner à une règle de style la priorité la plus élevée, remplaçant effectivement les autres déclarations, y compris celles avec une spécificité plus élevée et les styles inline.

Cependant, le mot-clé `!important` ne change pas la spécificité du sélecteur CSS lui-même. Il garantit simplement que la règle avec `!important` est appliquée, même s'il y a d'autres règles conflictuelles avec une spécificité plus élevée.

Un autre cas d'usage approprié pour le mot-clé `!important` est de remplacer les styles de bibliothèques ou de frameworks tiers quand tu n'as pas le contrôle sur le CSS d'origine.

Cependant, abuser du mot-clé `!important` peut entraîner des difficultés à maintenir et déboguer ton CSS, car ça rompt la cascade naturelle des styles et peut conduire à des conséquences imprévues.

Donc, il vaut mieux utiliser le mot-clé `!important` avec parcimonie.
