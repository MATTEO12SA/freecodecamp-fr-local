# --interactive--

Les sélecteurs ID font partie des sélecteurs les plus puissants en CSS, permettant aux développeurs d'appliquer des styles à des éléments spécifiques avec des identifiants uniques.

Cela les rend très efficaces pour cibler des éléments individuels qui ont besoin d'une mise en forme unique.

Les sélecteurs ID sont définis par un symbole dièse (`#`) suivi du nom de l'ID. Ils devraient être uniques dans un document HTML, ce qui signifie que deux éléments ne devraient jamais partager le même ID.

Voici un exemple utilisant un sélecteur ID :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="unique">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#unique {
  color: purple;
}
```

:::

Dans cet exemple, l'élément avec l'ID `unique` aura sa couleur de texte définie à `purple`.

Les sélecteurs ID ont une très haute spécificité, plus élevée que les sélecteurs de type et les sélecteurs de classe, mais plus faible que les styles inline. La valeur de spécificité pour un sélecteur ID est `(0, 1, 0, 0)`.

Cela signifie que les sélecteurs ID peuvent remplacer les sélecteurs de classe et les sélecteurs de type mais peuvent être remplacés par les styles inline.
