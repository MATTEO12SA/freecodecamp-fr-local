# --interactive--

Avec la fonction `calc()`, tu peux effectuer des calculs directement dans tes feuilles de style pour déterminer dynamiquement des valeurs de propriété. Cela signifie que tu peux créer des interfaces utilisateur flexibles et responsives en calculant des dimensions basées sur la taille du viewport ou d'autres éléments.

`calc()` est une fonction CSS. Tu en apprendras plus sur les fonctions quand tu commenceras à apprendre JavaScript, mais dans cette leçon, tu vas apprendre les bases que tu dois connaître pour comprendre comment `calc()` fonctionne.

Une fonction est un bloc de code qui effectue une tâche spécifique. Certaines fonctions sont déjà définies en CSS, donc tu peux les utiliser directement et leur passer toutes les valeurs nécessaires pour personnaliser comment leurs tâches seront effectuées.

Dans le monde de la programmation, quand on exécute la tâche effectuée par une fonction, on dit qu'on « appelle » la fonction. Les valeurs qu'on passe à la fonction sont connues sous le nom d'arguments.

Comme tu peux le voir dans le code ci-dessous, pour appeler une fonction, tu écris son nom suivi des arguments entre parenthèses, séparés par des virgules. Il ne devrait pas y avoir d'espace entre le nom de la fonction et la parenthèse ouvrante :

```css
function(argument1, argument2, argument3)
```

Une fonction peut n'avoir besoin que d'une seule valeur pour savoir quoi faire. Dans ce cas, elle ne prendra qu'un seul argument. C'est ce qui se passe avec la fonction `calc()`. Elle prend un argument parce qu'elle a besoin de savoir quoi calculer.

Pour cela, tu passes quelque chose appelé une expression comme argument. Une expression est une combinaison de valeurs et d'opérateurs qui produit un résultat.

Voici comment tu peux appeler la fonction `calc()`. Tu écris le nom calc, suivi des parenthèses, et à l'intérieur des parenthèses, tu écris l'expression :

```css
calc(expression)
```

L'expression est évaluée pour calculer le résultat final. « Évaluée » signifie juste que les valeurs et les opérateurs sont convertis en une seule valeur en coulisses. Le résultat est assigné à la propriété CSS où le calcul est effectué.

Tu peux effectuer des calculs sur des valeurs qui représentent une longueur, un angle, un temps, des pourcentages, des nombres et des couleurs. Tu peux aussi combiner différentes unités comme pixels, pourcentages et ems.

Avec les nombres, toutes les valeurs dans l'expression, aussi appelées les opérandes, doivent avoir leurs unités correspondantes, comme `px`, `em` et pourcentage (`%`). Selon l'opérateur, différents opérandes peuvent avoir différentes unités.

Tu peux utiliser les opérateurs d'addition (`+`), de soustraction (`-`), de multiplication (`*`) et de division (`/`) dans l'expression.

S'il y a plusieurs opérandes et opérateurs, `calc()` suivra la règle de précédence des opérateurs standard. Tu peux aussi ajouter des parenthèses pour établir l'ordre des opérations si nécessaire.

Dans l'exemple ci-dessous, tu peux voir un `div` avec le texte `Hello, World!`.

En utilisant le sélecteur de type CSS pour sélectionner le `div`, tu peux le styliser avec du texte blanc et un fond bleu foncé :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>Hello, World!</div>
```

```css
div {
  color: white;
  background-color: #1b1b32;
  width: calc(50% - 20px);
}
```

:::

Ce qui est nouveau ici, c'est que la largeur est calculée dynamiquement. Remarque comment on appelle la fonction `calc()` et lui passe une expression comme argument. L'expression a deux opérandes avec différentes unités et un opérateur, l'opérateur de soustraction.

Le pourcentage est une unité relative. La valeur (`50%`) sera déterminée par la largeur du conteneur parent. Puis, `20px` est soustrait de la valeur. Le résultat de cette expression déterminera la largeur du `div`.

La largeur du `div` est approximativement la moitié de la largeur totale de son conteneur, et si tu redimensionnes le conteneur parent, la largeur sera automatiquement recalculée.

C'est l'avantage clé d'utiliser `calc()`. Tu peux déterminer la valeur d'une propriété CSS dynamiquement en fonction de différents aspects de l'application ou du viewport.

L'expression peut aussi contenir des fonctions CSS et des variables si tu as besoin de les utiliser dans tes calculs. Tu en apprendras plus sur les variables CSS dans les prochaines leçons.

Super. Maintenant que tu connais les bases de la fonction `calc()`, couvrons quelques-unes de ses bonnes pratiques.

D'abord, tu dois entourer les opérateurs d'addition (`+`) et de soustraction (`-`) avec des espaces.

Par exemple, l'expression ci-dessous ne serait pas une expression valide parce que l'opérateur de soustraction est immédiatement avant le second opérande.

```css
calc(100% -30px)
```

L'opérateur de soustraction (`-`) doit être entouré d'espaces, comme ceci. Ajouter l'espace créera une expression valide.

```css
calc(100% - 30px)
```

Ce n'est pas nécessaire pour les opérateurs de multiplication et de division mais c'est fortement recommandé.

Tu peux aussi imbriquer des appels de fonction `calc()` si tu as besoin d'effectuer des calculs et d'utiliser ces résultats dans d'autres calculs.

De plus, si tu utilises la valeur zéro pour représenter une longueur dans l'expression que tu passes à la fonction `calc()`, tu dois inclure les unités. Par exemple, cette expression ne serait pas valide :

```css
calc(100% - 0)
```

Tu devrais ajouter les unités, comme px.

```css
calc(100% - 0px)
```

Tu devrais aussi savoir qu'actuellement, si tu utilises les opérateurs de multiplication ou de division, l'un des opérandes doit être sans unité. Pour l'opérateur de division, spécifiquement l'opérande de droite doit être sans unité. Ce ne serait pas une expression valide parce que les deux opérandes ont des unités (pixels). L'un des opérandes, soit 5 soit 50, doit être sans unité :

```css
calc(5px * 50px)
```

Tu devrais omettre les unités dans l'un d'eux. Ces deux alternatives seraient valides :

```css
calc(5 * 50px)
calc(5px * 50)
```

Et voici un exemple avec l'opérateur de division. Ce ne serait pas une expression valide puisqu'ils ont tous les deux des unités :

```css
calc(50% / 5%)
```

Tu devrais enlever l'unité de l'opérande de droite quand tu as l'opérateur de division :

```css
calc(50% / 5)
```

La fonction `calc()` peut être très utile pour toi en tant que développeur web. Avec cette fonction, tu peux définir des valeurs de propriété dynamiquement pour créer des designs flexibles et responsives.
