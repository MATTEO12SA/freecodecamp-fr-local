# --interactive--

À mesure que tu conçois tes pages, tu travailleras avec différentes propriétés comme les largeurs, hauteurs, padding, margins, et plus. Quand tu définis ces propriétés, tu devras spécifier les unités de mesure de longueur que tu veux utiliser.

Il existe deux types d'unités que tu peux utiliser pour définir ces propriétés : les unités relatives et les unités absolues. Dans cette leçon, on se concentrera juste sur les unités absolues.

Les unités de longueur absolues ont une longueur fixe et ne sont relatives à rien d'autre. Relatif signifie que la longueur est relative à autre chose, comme la taille de l'écran ou la taille de l'élément parent. On parlera plus des unités relatives dans les prochaines leçons.

L'unité absolue la plus courante est le pixel (`px`). Les pixels sont une unité de mesure de taille fixe en CSS, offrant un contrôle précis sur les dimensions. Cela signifie que 1px est toujours égal à 1/96e de pouce.

Il est important de noter que bien que 1px soit standardisé à 1/96e de pouce pour les besoins de la mise en page CSS, la taille physique réelle d'un pixel peut différer selon l'affichage.

Regardons un exemple d'utilisation de pixels pour définir la largeur et la hauteur d'une box :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
}
```

:::

Dans cet exemple, on a une box rouge avec une largeur et une hauteur définies à `100px`. Les dimensions de cette box resteront les mêmes indépendamment des changements liés à la taille de l'écran.

Donc, si tu es sur un appareil avec un petit écran, la box fera toujours `100px` de large. Si tu es sur un grand écran de bureau, la box fera toujours `100px` de large.

Alors, quand devrais-tu utiliser des unités absolues comme les pixels ? Eh bien, ça dépend de la situation. Mais généralement tu utiliseras des pixels là où tu as besoin d'un contrôle précis sur les dimensions, l'espacement et la mise en page des éléments. Parfois tu pourrais utiliser des pixels pour les margins, le padding et les bordures.

Voici un exemple d'utilisation de pixels pour définir la margin pour deux boxes :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  margin: 10px;
}
```

:::

Rappelle-toi que margin est l'espace à l'extérieur de la box. Donc, dans cet exemple, la box aura une margin de `10px` sur tous les côtés.

D'autres types d'unités absolues incluent les suivants :

- L'unité `in` (pouces), qui est égale à 96px
- L'unité `cm` (centimètres), qui est égale à 25.2/64 de pouce
- L'unité `mm` (millimètres), qui est égale à 1/10e de centimètre
- L'unité `q` (quart-millimètres), qui est égale à 1/40e de centimètre
- L'unité `pc` (picas), qui est égale à 1/6e de pouce
- L'unité `pt` (points), qui est égale à 1/72e de pouce

La plupart de ces unités seront utilisées pour l'impression et non pour les écrans.

Bien que l'unité la plus courante que tu utiliseras soit les pixels, il est important de savoir que les autres unités absolues existent.
