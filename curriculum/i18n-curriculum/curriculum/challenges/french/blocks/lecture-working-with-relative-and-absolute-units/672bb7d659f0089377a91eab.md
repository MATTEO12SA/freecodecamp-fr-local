# --interactive--

Les pourcentages en CSS sont des unités relatives qui te permettent de définir des tailles, des dimensions et d'autres propriétés comme une proportion de leur élément parent. Quand tu utilises une valeur en pourcentage, tu dis essentiellement « Rends ceci X% de son conteneur ».

Par exemple, si tu définis `width: 50%` sur un élément, il occupera la moitié de la largeur de son conteneur parent. Cela rend les pourcentages incroyablement utiles pour créer des designs responsives qui s'adaptent à différentes tailles d'écran.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  width: 400px;
  height: 200px;
  background-color: lightgray;
}

.box {
  width: 50%;
  height: 100%;
  background-color: red;
}
```

:::

Les pourcentages sont idéaux pour créer des mises en page fluides qui s'ajustent à diverses tailles d'écran. Par exemple, définir un conteneur à `width: 80%` garantit qu'il prend 80% de la largeur de son parent, indépendamment de l'appareil.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 100%;
  height: 300px;
  background-color: lightblue;
}

.child {
  width: 80%;
  height: 100%;
  background-color: red;
}
```

:::

Utiliser des pourcentages pour des images flexibles est une autre pratique courante. En appliquant `max-width: 100%` aux images, tu leur permets de se réduire sur des écrans plus petits tout en conservant leur ratio d'aspect.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<img src="https://placehold.co/150x150" alt="Example Product Image" />
```

```css
img {
  max-width: 100%;
  height: auto;
}
```

:::

Bien que moins courant, les pourcentages peuvent aussi être utilisés pour les tailles de police pour créer une typographie évolutive. Par exemple, `font-size: 120%` rendrait le texte 20% plus grand que la taille de police de son parent.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="text-container">
  Parent text.
  <p class="text">This is some example text.</p>
</div>
```

```css
.text-container {
  font-size: 16px;
}

.text {
  font-size: 120%;
}
```

:::

Les pourcentages peuvent être particulièrement pratiques pour le centrage vertical. Voici un exemple de comment tu pourrais utiliser les pourcentages avec la propriété `transform` pour centrer un élément verticalement.

Cet exemple positionne l'élément à 50% depuis le haut de son conteneur, puis utilise `transform` pour le déplacer vers le haut de la moitié de sa propre hauteur, le centrant effectivement verticalement.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="centered"></div>
```

```css
.centered {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  width: 300px;
  height: 300px;
  background-color: red;
}
```

:::

Tu en apprendras plus sur le fonctionnement du positionnement absolu et des propriétés transform dans les futures leçons.

Rappelle-toi, les pourcentages sont toujours relatifs à quelque chose. Pour les propriétés horizontales comme `width`, ils sont relatifs à la largeur du parent. Pour les propriétés verticales comme `height`, ils sont généralement relatifs à la hauteur du parent (si spécifiée).

Cependant, sois prudent quand tu imbriques des éléments avec des dimensions basées sur les pourcentages, car cela peut conduire à des résultats inattendus. De plus, garde en tête que les hauteurs basées sur les pourcentages peuvent être délicates si le parent n'a pas de hauteur définie.

En résumé, les pourcentages en CSS sont des outils puissants pour créer des designs flexibles et responsives. Utilise-les quand tu veux que les éléments s'adaptent proportionnellement à leurs conteneurs ou quand tu as besoin de mises en page qui s'adaptent parfaitement à différentes tailles d'écran.
