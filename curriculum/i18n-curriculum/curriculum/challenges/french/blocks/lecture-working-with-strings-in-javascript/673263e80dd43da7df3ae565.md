# --interactive--

Quand tu travailles avec des chaînes en JavaScript, il peut arriver que tu aies besoin de localiser la position d'une sous-chaîne précise dans une chaîne plus grande.

Une sous-chaîne est une suite de caractères qui apparaît à l'intérieur d'une chaîne plus grande. Par exemple, dans la chaîne `hello world`, `hello` et `world` sont des sous-chaînes.

Pour localiser la position d'une sous-chaîne à l'intérieur d'une chaîne, tu peux utiliser la méthode `indexOf()`. La méthode `indexOf()` en JavaScript te permet de rechercher une sous-chaîne dans une chaîne.

Si la sous-chaîne est trouvée, `indexOf()` renvoie l'indice (ou la position) de la première occurrence de cette sous-chaîne. Si la sous-chaîne n'est pas trouvée, `indexOf()` renvoie `-1`, ce qui indique que la recherche a échoué.

La méthode `indexOf()` prend deux arguments : le premier est la sous-chaîne que tu veux trouver dans la chaîne plus grande, et le second est une position de départ facultative pour la recherche. Si tu ne fournis pas de position de départ, la recherche commencera au début de la chaîne.

Dans ce contexte, un **argument** est une valeur que tu donnes à une fonction ou à une méthode lorsque tu l'appelles, ce qui permet à cette fonction ou méthode d'accomplir sa tâche en utilisant les informations précises que tu fournis. Tu en apprendras davantage sur les arguments dans de prochaines leçons.

Voici un exemple d'utilisation de la méthode `indexOf()` pour trouver la position de la chaîne `awesome` :

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("awesome!");
console.log(position); // 14
```

:::

Dans cet exemple, le mot `awesome` commence à l'indice `14` dans la chaîne `JavaScript is awesome!`, donc la méthode `indexOf()` renvoie `14`.

Maintenant, voyons ce qui se passe quand la sous-chaîne n'est pas trouvée :

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("fantastic");
console.log(position); // -1
```

:::

Comme le mot `fantastic` n'apparaît pas dans la chaîne, la méthode renvoie `-1`.

Tu peux aussi préciser où commencer la recherche dans la chaîne en fournissant un second argument à `indexOf()`. Voici un exemple :

:::interactive_editor

```js
let sentence = "JavaScript is awesome, and JavaScript is powerful!";
let position = sentence.indexOf("JavaScript", 10);
console.log(position); // 27
```

:::

Dans ce cas, la recherche de `JavaScript` commence après le 10ᵉ caractère, et la deuxième occurrence de `JavaScript` est donc trouvée à l'indice `27`.

Il est important de noter que la méthode `indexOf()` est sensible à la casse.

Dans cet exemple, ce qui suit renverrait `-1` car la lettre majuscule `F` ne se trouve pas dans la chaîne `freeCodeCamp`.

:::interactive_editor

```js
console.log("freeCodeCamp".indexOf("F")) // -1
```

:::

Utiliser `indexOf()` peut être très utile quand tu as besoin de vérifier si une sous-chaîne est présente dans une chaîne et de déterminer sa position pour des opérations ultérieures.
