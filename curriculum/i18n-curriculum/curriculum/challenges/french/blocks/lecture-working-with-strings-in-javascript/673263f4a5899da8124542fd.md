# --interactive--

La méthode `prompt()` est un élément important de l'interaction de JavaScript avec l'utilisateur. C'est l'un des moyens les plus simples d'obtenir une saisie de l'utilisateur via une petite boîte de dialogue contextuelle.

Tu la verras souvent utilisée dans les cas où la page web a besoin d'une information de la part de l'utilisateur, comme un nom ou une autre forme de saisie de texte.

Alors, que fait exactement la méthode `prompt()` ? Elle ouvre une boîte de dialogue qui demande une saisie à l'utilisateur, puis elle renvoie le texte saisi par l'utilisateur sous forme de chaîne.

La méthode `prompt()` prend deux arguments : le premier est le message qui apparaîtra dans la boîte de dialogue, invitant généralement l'utilisateur à saisir une information. Et le second est une valeur par défaut, facultative, qui remplira initialement le champ de saisie.

```js
prompt(message, default);
```

Voici un exemple de son fonctionnement.

**REMARQUE** : Cet exemple inclut du code que tu n'as pas encore appris. Ne t'inquiète pas de tout comprendre dans ce code. Il sert juste à illustrer le fonctionnement de la méthode `prompt()` et à éviter que l'invite n'apparaisse immédiatement au chargement de la page, ce qui pourrait être perçu comme intrusif. Si l'éditeur interactif est activé, tu peux l'essayer toi-même.

:::interactive_editor

```html
<button id="prompt-btn">Show Prompt</button>
<p id="output"></p>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("prompt-btn");
const output = document.getElementById("output");
btn.addEventListener("click", () => {
  const userName = prompt("What is your name?", "Guest");
  output.textContent = "Hello, " + userName + "!";
});
```

:::

Dans cet exemple, quand l'utilisateur clique sur le bouton, la méthode `prompt()` affiche une boîte de dialogue avec le message `What is your name?` et un champ de saisie qui contient initialement la valeur `Guest`.

Si l'utilisateur saisit son nom et appuie sur "OK", la variable `userName` stockera la valeur saisie. Si l'utilisateur appuie sur "Annuler", la variable `userName` sera définie à `null`. `null` signifie que l'utilisateur n'a fourni aucune saisie. Le paragraphe de sortie affichera alors un message de salutation en utilisant le nom fourni, ou `null` si l'utilisateur a annulé. Tu apprendras dans de prochaines leçons des techniques pour éviter d'afficher `null` quand un utilisateur annule l'invite.

Garde à l'esprit que la méthode `prompt()` interrompra l'exécution du script jusqu'à ce que l'utilisateur interagisse avec la boîte de dialogue.

Cela signifie que le reste de ton code JavaScript ne s'exécutera pas tant que l'utilisateur n'aura pas fourni une saisie et cliqué sur "OK", ou annulé l'invite.

Un autre point à prendre en compte est que, même si `prompt()` est utile pour des tests rapides ou de petites applications, on l'évite généralement dans les applications web modernes et complexes en raison de son caractère perturbateur et de son comportement incohérent d'un navigateur à l'autre.

En comprenant la méthode `prompt()`, tu disposes d'un moyen simple d'interagir avec les utilisateurs et de récupérer des informations directement via le navigateur, même si elle n'est peut-être pas très utilisée dans les applications web modernes.
