# --interactive--

## Éléments et attributs des formulaires HTML

- **Élément `form`** : sert à créer un formulaire HTML pour la saisie utilisateur.
- **Attribut `action`** : sert à indiquer l’URL où les données du formulaire doivent être envoyées.
- **Attribut `method`** : sert à indiquer la méthode HTTP à utiliser lors de l’envoi des données du formulaire. Les méthodes les plus courantes sont `GET` et `POST`.

```html
<form method="value-goes-here" action="url-goes-here">
  <!-- inputs go inside here -->
</form>
```

- **Élément `input`** : sert à créer un champ de saisie utilisateur.
- **Attribut `type`** : sert à indiquer le type de champ de saisie. Ex. `text`, `email`, `number`, `radio`, `checkbox`, etc.
- **Attribut `placeholder`** : sert à afficher une indication à l’utilisateur pour lui montrer quoi saisir dans le champ.
- **Attribut `value`** : sert à indiquer la valeur de l’input. Si l’input a un type `button`, l’attribut `value` peut servir à définir le texte du bouton.
- **Attribut `name`** : sert à donner un nom à un champ de saisie, qui sert de clé quand les données du formulaire sont soumises. Pour les boutons radio, leur donner le même `name` les regroupe, donc une seule option du groupe peut être sélectionnée à la fois.
- **Attribut `size`** : sert à définir le nombre de caractères visibles pendant que l’utilisateur saisit du texte dans l’input.
- **Attribut `min`** : peut être utilisé avec des types d’input comme `number` pour indiquer la valeur minimale autorisée dans le champ.
- **Attribut `max`** : peut être utilisé avec des types d’input comme `number` pour indiquer la valeur maximale autorisée dans le champ.
- **Attribut `minlength`** : sert à indiquer le nombre minimal de caractères requis dans un champ de saisie.
- **Attribut `maxlength`** : sert à indiquer le nombre maximal de caractères autorisés dans un champ de saisie.
- **Attribut `required`** : sert à indiquer qu’un champ doit être rempli avant la soumission du formulaire.
- **Attribut `disabled`** : sert à indiquer qu’un champ doit être désactivé.
- **Attribut `readonly`** : sert à indiquer qu’un champ est en lecture seule.

:::interactive_editor

```html
<!-- Text input -->
<input 
  type="text"
  id="name"
  name="name"
  placeholder="e.g. Quincy Larson" 
  size="20"
  minlength="5"
  maxlength="30"
  required
/>

<!-- Number input -->
<input 
  type="number"
  id="quantity"
  name="quantity"
  min="2"
  max="10"
  disabled
/>

<!-- Button -->
<input type="button" value="Show Alert" />
```

:::

- **Élément `label`** : sert à créer un libellé pour un champ de saisie.
- **Attribut `for`** : sert à indiquer à quel champ de saisie le label est associé.
- **Association implicite de formulaire** : les inputs peuvent être associés à des labels en plaçant le champ de saisie à l’intérieur de l’élément `label`.

:::interactive_editor

```html
<form action="">
  <label>
    Full Name:
    <input type="text" />
  </label>
</form>
```

:::

- **Association explicite de formulaire** : les inputs peuvent être associés à des labels en utilisant l’attribut `for` sur l’élément `label`.

:::interactive_editor

```html
<form action="">
  <label for="email">Email Address: </label>
  <input type="email" id="email" />
</form>
```

:::

- **Élément `button`** : sert à créer un bouton cliquable. Un bouton peut aussi avoir un attribut `type`, qui sert à contrôler son comportement quand il est activé. Ex. `submit`, `reset`, `button`.

:::interactive_editor

```html
<button type="button">Show Form</button>
<button type="submit">Submit Form</button>
<button type="reset">Reset Form</button>
```

:::

- **Élément `fieldset`** : sert à regrouper des inputs liés.
- **Élément `legend`** : sert à ajouter une légende qui décrit le groupe d’inputs.

:::interactive_editor

```html
<!-- Radio group -->
<fieldset>
  <legend>Was this your first time at our hotel?</legend>

  <label for="yes-option">Yes</label>
  <input id="yes-option" type="radio" name="hotel-stay" value="yes" />

  <label for="no-option">No</label>
  <input id="no-option" type="radio" name="hotel-stay" value="no" />
</fieldset>

<!-- Checkbox group -->
<fieldset>
  <legend>
    Why did you choose to stay at our hotel? (Check all that apply)
  </legend>

  <label for="location">Location</label>
  <input type="checkbox" id="location" name="location" value="location" />

  <label for="price">Price</label>
  <input type="checkbox" id="price" name="price" value="price" />
</fieldset>
```

:::

- **État de focus** : c’est l’état d’un champ de saisie lorsqu’il est sélectionné par l’utilisateur.

## Travailler avec les éléments et attributs de tableaux HTML

- **Élément table** : sert à créer un tableau HTML.
- **Élément Table Head (`thead`)** : sert à regrouper le contenu d’en-tête dans un tableau HTML.
- **Élément Table Row (`tr`)** : sert à créer une ligne dans un tableau HTML.
- **Élément Table Header (`th`)** : sert à créer une cellule d’en-tête dans un tableau HTML.
- **Élément Table body (`tbody`)** : sert à regrouper le contenu du corps dans un tableau HTML.
- **Élément Table Data Cell (`td`)** : sert à créer une cellule de données dans un tableau HTML.
- **Élément Table Foot (`tfoot`)** : sert à regrouper le contenu du pied dans un tableau HTML.
- **Élément `caption`** : sert à ajouter un titre à un tableau HTML.
- **Attribut `colspan`** : sert à indiquer le nombre de colonnes sur lesquelles une cellule doit s’étendre.

:::interactive_editor

```html
<table>
  <caption>Exam Grades</caption>

  <thead>
    <tr>
      <th>Last Name</th>
      <th>First Name</th>
      <th>Grade</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Davis</td>
      <td>Alex</td>
      <td>54</td>
    </tr>

    <tr>
      <td>Doe</td>
      <td>Samantha</td>
      <td>92</td>
    </tr>

    <tr>
      <td>Rodriguez</td>
      <td>Marcus</td>
      <td>88</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="2">Average Grade</td>
      <td>78</td>
    </tr>
  </tfoot>
</table>
```

:::

## Travailler avec les outils HTML

- **Validateur HTML** : outil qui vérifie la syntaxe du code HTML pour s’assurer qu’il est valide.
- **Inspecteur DOM** : outil qui permet d’inspecter et de modifier la structure HTML d’une page web.
- **DevTools** : ensemble d’outils de développement web intégrés directement au navigateur, qui aident à déboguer, profiler et analyser des pages web.
