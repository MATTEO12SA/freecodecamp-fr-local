# --interactive--

En CSS, la propriété `list-style` est utilisée pour contrôler l'apparence des listes sur une page web.

Que tu travailles avec des listes ordonnées (`ol`) ou des listes non ordonnées (`ul`), la propriété `list-style` te permet de personnaliser comment les éléments de liste sont affichés.

La propriété `list-style` est en fait un raccourci pour trois autres propriétés :

- `list-style-type`
- `list-style-position`
- `list-style-image`

Chacune joue un rôle différent dans la définition de l'apparence des listes.

La propriété `list-style-type` te permet de définir le type de puce ou de numéro utilisé dans une liste.

Pour les listes non ordonnées, tu peux choisir parmi plusieurs styles de puces, comme des disques, des cercles ou des carrés.

Pour les listes ordonnées, tu peux utiliser différents systèmes de numérotation, comme décimal, chiffres romains, ou même caractères alphabétiques.

Voici un exemple d'utilisation de `list-style-type` :

:::interactive_editor

```html
<ul style="list-style-type: square;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

Dans cet exemple, les puces de la liste non ordonnée sont changées en carrés.

La propriété `list-style-type` est la plus couramment utilisée des trois, car elle affecte directement l'apparence du style de puce ou de numérotation dans tes listes.

La propriété `list-style-position` contrôle la position de la puce ou du numéro par rapport au contenu de l'élément de liste. Il y a deux valeurs que tu peux utiliser : `inside` et `outside`.

Quand tu utilises la valeur `outside`, la puce ou le numéro apparaît à l'extérieur du contenu, ce qui est le comportement par défaut.

Et quand tu utilises la valeur `inside`, la puce ou le numéro apparaît à l'intérieur du contenu, ce qui peut faire que le texte se replie et s'aligne avec la puce ou le numéro.

Voici un exemple d'utilisation de `list-style-position` :

:::interactive_editor

```html
<ul style="list-style-position: inside;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
<ul style="list-style-position: outside;">
  <li>Item 4</li>
  <li>Item 5</li>
  <li>Item 6</li>
</ul>
```

:::

Dans cet exemple, les valeurs `inside` et `outside` sont fournies pour deux balises de liste non ordonnée différentes.

La propriété `list-style-position` peut être utile quand tu veux contrôler l'alignement du contenu de la liste, surtout dans les situations où tu as plusieurs lignes de texte dans un seul élément de liste.

La propriété `list-style-image` te permet d'utiliser une image comme puce pour tes éléments de liste. Ça peut être utile pour ajouter un style visuel unique à tes listes.

Voici un exemple d'utilisation de `list-style-image` :

:::interactive_editor

```html
<head>
  <style>
    ul {
      list-style-image: url('https://cdn.freecodecamp.org/platform/universal/freecodecamp-org-gravatar.jpeg');
      list-style-position: inside;
    }
  </style>
</head>
<body>
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>
</body>
```

:::

Dans cet exemple, les puces sont remplacées par un logo freeCodeCamp personnalisé, ajoutant une touche personnalisée à la liste.

Quand tu utilises `list-style-image`, assure-toi que l'image que tu choisis est petite et appropriée pour le design de ta page web. Si l'image est trop grande ou complexe, ça peut rendre la liste plus difficile à lire.

Tu peux combiner les trois propriétés – `list-style-type`, `list-style-position` et `list-style-image` – en une seule propriété raccourcie `list-style`.

L'ordre des valeurs dans le raccourci n'a pas d'importance, mais les trois peuvent être spécifiés ensemble.

Voici un exemple utilisant la propriété raccourcie :

:::interactive_editor

```html
<ul style="list-style: square inside url('https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg');">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

Dans cet exemple, les éléments de liste utilisent des puces carrées, positionnées à l'intérieur du contenu, avec une image personnalisée comme puce.

Cependant, si l'image n'est pas disponible ou n'est pas rendue, les puces carrées seront utilisées comme solution de repli.
