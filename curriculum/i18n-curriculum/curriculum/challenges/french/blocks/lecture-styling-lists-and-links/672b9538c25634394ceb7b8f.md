# --interactive--

Il y a différents états d'un lien, dont `link`, `visited`, `hover`, `focus` et `active`. Ces états sont importants pour aider les utilisateurs à reconnaître les liens et fournir un retour clair après les interactions, ce qui améliore à la fois l'utilisabilité et l'accessibilité.

Styliser ces différents états de liens est essentiel pour l'utilisabilité et l'accessibilité, car ça fournit des indices visuels sur l'état actuel du lien. Ça aide les utilisateurs à comprendre quels liens ils ont visités, avec quel lien ils interagissent, et ce qui se passera quand ils cliqueront.

Pour les utilisateurs ayant des déficiences visuelles ou des handicaps cognitifs, ces styles distincts peuvent rendre la navigation beaucoup plus facile et intuitive.

De plus, des états de liens clairs améliorent l'expérience utilisateur globale en fournissant un retour immédiat sur les interactions de l'utilisateur, réduisant la confusion et améliorant la navigabilité du site.

Ces états peuvent être stylisés en utilisant quelque chose appelé `pseudo-classes` en CSS.

Une pseudo-classe est un mot-clé ajouté à un sélecteur qui spécifie un état spécial de l'élément sélectionné.

Par exemple, `:hover` peut changer la couleur d'un bouton quand le pointeur de l'utilisateur passe dessus, tandis que `:visited` peut changer la couleur d'un lien qui a déjà été visité.

Les pseudo-classes te permettent de styliser les éléments en fonction de leur état ou de l'interaction de l'utilisateur avec eux, sans le besoin de balisage supplémentaire dans ton HTML.

La syntaxe d'une `pseudo-class` ressemble à quelque chose comme ça où `A` est le sélecteur et `:B` est la `pseudo-class` :

```css
A:B {
  property: value;
}
```

Pour mieux comprendre comment styliser les différents états de liens, regardons quelques exemples.

La pseudo-classe `:link` stylise les liens non visités, indiquant qu'ils sont cliquables.

Voici un exemple ciblant un élément d'ancrage et utilisant la pseudo-classe `:link` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Normal state (unvisited link) */
a:link { 
  color: red;
}
```

:::

L'exemple ci-dessus changera la couleur bleue par défaut du lien en `red` quand il n'est pas visité.

`:visited` stylise les liens qui ont déjà été visités ou cliqués, aidant les utilisateurs à suivre les liens sur lesquels ils ont cliqué auparavant. Voici un exemple d'utilisation de la pseudo-classe `:visited` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org/learn/" target="_blank">freeCodeCamp</a>
```

```css
/* Visited link */
a:visited {
  color: green;
}
```

:::

Ce code colorera le lien en `green` quand il est cliqué.

`:hover` change le style du lien quand l'utilisateur passe la souris dessus, fournissant un indice visuel que le lien est interactif. Voici un exemple d'utilisation de la pseudo-classe `:hover` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Hover state */
a:hover {
  color: green;
}
```

:::

Ce code colorera les liens en `green` quand la souris est passée dessus.

`:focus` ajoute des styles autour du lien quand il est focalisé, comme quand on navigue avec un clavier, ou améliore l'accessibilité.

Voici un exemple utilisant la propriété `outline` pour appliquer un contour orange continu quand le lien est focalisé.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Focus state */
a:focus {
  outline: 2px solid orange;
}
```

:::

`:active` change les styles du lien pendant que le lien est cliqué, fournissant un retour immédiat à l'utilisateur que son action est enregistrée. Voici un exemple d'utilisation de la pseudo-classe `:active` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Active state */
a:active {
  color: pink;
}
```

:::

Cet exemple de code rendra le lien de couleur `pink` immédiatement quand le lien est cliqué.
