# --interactive--

Les margins et le line-height sont essentiels pour espacer les éléments de liste afin d'améliorer la lisibilité et l'attrait visuel.

D'abord, commençons à espacer les éléments de liste en utilisant les margins !

Les margins peuvent être utilisées pour créer de l'espace entre les éléments de liste en appliquant les propriétés margin aux éléments `li`. Cette méthode te permet de contrôler l'espacement à l'extérieur de chaque élément de liste, augmentant ou diminuant efficacement l'écart entre eux.

Regardons un exemple d'une liste non ordonnée avec trois éléments de liste.

:::interactive_editor

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

Par défaut, le HTML n'appliquera pas beaucoup d'espacement entre les éléments de liste.

Pour appliquer un peu d'espacement au bas de chaque élément de liste, tu peux utiliser la propriété `margin-bottom` comme ceci :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  margin-bottom: 40px;
}
```

:::

Dans cet exemple, `40px` de margin seront appliqués au bas de chaque élément de liste à l'intérieur de la liste non ordonnée.

Une autre façon d'espacer les éléments de liste serait d'utiliser la propriété `line-height`.

La propriété `line-height` ajuste l'espacement vertical entre les lignes de texte dans un seul élément de liste.

Bien qu'elle affecte principalement l'espacement entre les lignes de texte dans chaque élément, elle peut aussi indirectement influencer l'espacement global entre les éléments de liste si les éléments ne contiennent qu'une seule ligne de texte.

Si les éléments de liste ont plusieurs lignes de texte, le `line-height` affectera l'espacement entre ces lignes, mais il n'ajuste pas directement l'espacement entre les éléments de liste séparés.

Pour contrôler l'espacement entre les éléments de liste individuels, tu utiliserais plutôt les propriétés `margin` ou `padding`.

En utilisant la même liste non ordonnée que précédemment, voici un exemple d'application de `line-height` aux éléments de liste :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  line-height: 2; 
}
```

:::

Dans cet exemple, `line-height: 2;` définit la hauteur de ligne à deux fois la taille de la police, créant plus d'espace vertical dans chaque élément de liste.

# --questions--

## --text--

Quelle propriété CSS utiliserais-tu pour créer de l'espace entre les éléments de liste en ajustant l'espace à l'extérieur de chaque élément `li` ?

## --answers--

`padding`

### --feedback--

Cette propriété affecte l'espace à l'intérieur de la `border` de l'élément.

---

`line-height`

### --feedback--

Cette propriété affecte l'espacement entre les lignes de texte, pas l'espace à l'extérieur de l'élément.

---

`margin`

---

`border`

### --feedback--

Cette propriété affecte la `border` de l'élément, pas l'espace à l'extérieur.

## --text--

Étant donné la règle CSS `li { margin-bottom: 15px; }`, quel effet cela a-t-il sur les éléments de liste ?

## --answers--

Ajoute `15px` d'espace à l'intérieur de chaque élément de liste.

### --feedback--

Cette règle affecte l'espace à l'extérieur de l'élément de liste, pas à l'intérieur.

---

Ajoute `15px` d'espace entre le contenu et la `border` de chaque élément de liste.

### --feedback--

Cette règle affecte l'espace à l'extérieur de la `border` de chaque élément de liste.

---

Ajoute `15px` d'espace en dessous de chaque élément de liste.

---

Ajoute `15px` d'espace au-dessus de chaque élément de liste.

### --feedback--

Cette règle ajoute de l'espace en dessous de chaque élément de liste, pas au-dessus.

Que contrôle la propriété `line-height` en CSS ?

## --answers--

La `width` du contenu.

### --feedback--

C'est contrôlé par la propriété `width`, pas `line-height`.

---

L'espace à l'intérieur de la `border` d'un élément.

### --feedback--

C'est contrôlé par des propriétés comme `padding`, pas `line-height`.

---

L'espacement vertical entre les lignes de texte dans un élément.

---

L'espace à l'extérieur de la `border` d'un élément.

### --feedback--

C'est contrôlé par des propriétés comme `margin`, pas `line-height`.
