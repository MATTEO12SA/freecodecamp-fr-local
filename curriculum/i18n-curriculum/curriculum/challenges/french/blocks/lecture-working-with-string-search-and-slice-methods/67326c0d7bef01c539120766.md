# --interactive--

Quand tu travailles avec des chaînes en JavaScript, il existe de nombreux cas où tu pourrais avoir besoin de vérifier si une chaîne contient une sous-chaîne précise, c'est-à-dire une partie plus petite de cette chaîne.

Par exemple, tu pourrais vouloir vérifier si la saisie d'un utilisateur contient un mot ou un caractère précis avant d'effectuer une action. Une façon d'y parvenir est d'utiliser la méthode `includes()`.

La méthode `includes()` sert à vérifier si une chaîne contient une sous-chaîne précise. Si la sous-chaîne est trouvée dans la chaîne, la méthode renvoie `true` ; sinon, elle renvoie `false`.

Voici la syntaxe de base :

```js
string.includes(searchValue);
```

Dans cette syntaxe, `searchValue` est la sous-chaîne que tu veux rechercher dans la chaîne. Et voici un exemple :

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");

console.log(result);  // true
```

:::

Dans cet exemple, le mot `awesome` est trouvé dans la chaîne `JavaScript is awesome!`, donc la méthode `includes()` renvoie `true`.

Il est important de noter que la méthode `includes()` est sensible à la casse. Cela signifie qu'une correspondance exacte des caractères est requise, y compris leur casse.

Par exemple :

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");

console.log(result);  // false
```

:::

Comme `Awesome` (avec un `A` majuscule) ne correspond pas à `awesome` (avec un `a` minuscule), le résultat est `false`.

Tu peux aussi utiliser la méthode `includes()` pour rechercher une sous-chaîne à partir d'un indice précis de la chaîne en fournissant un second paramètre :

:::interactive_editor

```js
let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);

console.log(result);  // true
```

:::

Ici, la recherche de la sous-chaîne `JavaScript` commence à la 7ᵉ position de la chaîne, ce qui garantit qu'elle ignore tous les caractères situés avant cette position.

La méthode `includes()` renvoie uniquement un résultat `true` ou `false`. Elle ne fournit aucune information sur l'endroit où se trouve la sous-chaîne dans la chaîne ni sur le nombre de fois où elle apparaît. Si tu as besoin de ce niveau de détail, d'autres méthodes, comme la méthode `indexOf()`, pourraient être plus adaptées.
