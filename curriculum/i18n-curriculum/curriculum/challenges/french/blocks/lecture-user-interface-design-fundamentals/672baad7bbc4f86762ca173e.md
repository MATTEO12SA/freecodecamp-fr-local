# --interactive--

**NOTE** : Certains des exemples interactifs peuvent utiliser du CSS que tu n'as pas encore appris. Ne t'inquiète pas si tu essaies de tout comprendre. Le but des exemples est de te montrer des aperçus de ces concepts de design pour que tu comprennes mieux comment les choses fonctionnent.

Ajouter des images à tes sites web est une excellente façon d'engager tes utilisateurs et d'augmenter l'attrait visuel de ton site. Mais il y a quelques éléments à considérer quand on travaille avec des images dans tes designs.

La première chose à considérer est de créer des images responsives. Les images responsives sont des images qui s'adaptent à la taille de l'écran sur lequel elles sont consultées. C'est important parce que ça garantit que tes images ont bonne allure sur tous les appareils, des ordinateurs de bureau aux téléphones mobiles.

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 20px;
    background-color: #fefefe;
    color: #333;
    text-align: center;
  }

  img {
    max-width: 100%;
    height: auto;
    border-radius: 8px;
  }

  p {
    font-size: 16px;
    max-width: 600px;
    margin: 20px auto;
    line-height: 1.6;
  }
</style>

<h1>Responsive Cat Image</h1>

<img 
  src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" 
  alt="Two cats peacefully sleeping together."
/>

<p>
  This image automatically scales based on the screen size. Whether you're viewing on a desktop or a mobile phone,
  it adjusts its size without losing proportions, making the design clean and user-friendly on all devices.
</p>
```

:::

Une autre chose à considérer est la résolution pour les images. Les images de meilleure qualité avec une meilleure résolution ont plus de pixels par pouce. Les pixels sont des petits carrés qui composent une image.

Pixels par pouce, ou PPI, est le nombre de pixels dans un pouce d'une image. Plus le PPI est élevé, meilleure est la qualité de l'image.

Tu veux t'assurer que tes images sont de haute qualité et ont bonne allure sur tous les appareils. Ça signifie que tu devrais utiliser des images haute résolution optimisées pour le web.

Une autre chose à considérer est la taille de tes images et comment elles s'adaptent aux espaces dans la mise en page. Tu veux t'assurer que tes images sont de la bonne taille et ne sont pas trop grandes ou trop petites.

Utiliser de grandes images destinées à s'adapter à de plus petits espaces dans le design peut ralentir ton site web et rendre plus difficile pour les utilisateurs de charger ton site. Tu veux t'assurer que tes images sont de la bonne taille et sont optimisées pour le web.

Quand il s'agit du placement des images, tu veux penser à l'équilibre, à la hiérarchie et à l'alignement pour aider à garantir que tes images sont optimisées pour le web.

L'équilibre est la distribution du poids visuel dans un design. Tu veux t'assurer qu'il y a un bon équilibre entre le texte et les images sur le site pour que ça crée un design harmonieux.

```html
<div class="container">
  <div class="text">
    <h2>Balanced Layout</h2>

  </div>

```

:::

La hiérarchie est l'ordre dans lequel les éléments sont vus sur une page. Tu veux t'assurer que les images qui s'alignent avec du contenu important sont placées plus haut que les images qui sont moins importantes.

:::interactive_editor

```html
  }

  .section {
    margin-bottom: 40px;
  }

  .section img {
```

L'alignement est le placement des éléments les uns par rapport aux autres. Tu veux t'assurer que tes images sont alignées avec le texte et les autres éléments de ton site pour que ça crée un design cohérent.

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 40px 20px;
    background-color: #ffffff;
    color: #222;
    max-width: 900px;
    margin: 0 auto;
  }

  .aligned-content {
    display: flex;
    flex-wrap: wrap;
    align-items: center;
    gap: 20px;
  }

  .aligned-content img {
    width: 300px;
    height: auto;
    border-radius: 8px;
    flex-shrink: 0;
  }

  .aligned-text {
    flex: 1;
    min-width: 250px;
  }

  .aligned-text h2 {
    font-size: 24px;
    margin-bottom: 10px;
  }

  .aligned-text p {
    font-size: 16px;
    line-height: 1.6;
  }
</style>

<div class="aligned-content">
  <img 
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" 
    alt="Two cats peacefully sleeping together."
  >

  <div class="aligned-text">
    <h2>Aligned Elements</h2>
    <p>
      In this example, the image is aligned horizontally next to the text block. This creates a visually cohesive layout, 
      where both elements feel like they belong together. Proper alignment like this improves readability and reinforces 
      structure in your design.
    </p>
  </div>
</div>
```

:::

La dernière chose à considérer est l'accessibilité pour les images. Tu veux t'assurer que tes images sont accessibles à tous les utilisateurs, y compris ceux ayant des déficiences visuelles. Ça signifie que tu devrais utiliser un texte alt pour tes images pour que les lecteurs d'écran puissent lire le texte aux utilisateurs malvoyants.
