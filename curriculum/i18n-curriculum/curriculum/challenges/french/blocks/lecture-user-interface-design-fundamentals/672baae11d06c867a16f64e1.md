# --description--

L'amélioration progressive (progressive enhancement) est une approche de design qui garantit que tous les utilisateurs, indépendamment du navigateur ou de l'appareil, peuvent accéder au contenu essentiel et aux fonctionnalités d'une application.

Elle se concentre sur la livraison d'une expérience de base qui fonctionne pour tout le monde, tout en offrant des fonctionnalités et des améliorations supplémentaires aux utilisateurs avec des navigateurs plus avancés ou de meilleures connexions internet.

L'approche de l'amélioration progressive vit selon ces principes de base :

- Tout le contenu de base et la fonctionnalité de base devraient être accessibles sur tous les navigateurs
- Toutes les mises en page avancées devraient être fournies via des feuilles de style CSS externes
- Toute la fonctionnalité avancée devrait être fournie via des fichiers JavaScript externes
- Les préférences du navigateur d'un utilisateur devraient être respectées

Utiliser une approche d'amélioration progressive rend tes applications plus accessibles parce que tout le contenu de base et la fonctionnalité ne devraient pas être bloqués dans des environnements non pris en charge.

En termes de vitesse, une approche d'amélioration progressive peut aussi aider à améliorer les performances de tes applications.

Les utilisateurs qui travaillent avec des vitesses de connexion internet plus lentes pourront toujours accéder au contenu parce que le navigateur téléchargera d'abord les ressources nécessaires.

En ce qui concerne le SEO, l'amélioration progressive peut aussi aider à améliorer la visibilité de tes applications.

Les moteurs de recherche pourront explorer le contenu de tes applications parce que le contenu de base est disponible dans la réponse HTML initiale.

Bien que certains aient critiqué cette approche en considérant qu'elle n'est pas toujours réaliste pour les applications qui dépendent beaucoup de JavaScript pour leur fonctionnalité, c'est toujours une bonne pratique à suivre quand on construit des applications.
