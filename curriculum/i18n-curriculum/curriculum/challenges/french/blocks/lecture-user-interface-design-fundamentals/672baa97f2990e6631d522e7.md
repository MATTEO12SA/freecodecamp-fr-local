# --interactive--

**NOTE** : Certains des exemples interactifs peuvent utiliser du CSS que tu n'as pas encore appris. Ne t'inquiète pas si tu essaies de tout comprendre. Le but des exemples est de te montrer des aperçus de ces concepts de design pour que tu comprennes mieux comment les choses fonctionnent.

D'abord, tu auras peut-être besoin de te familiariser avec le contraste. Le contraste est la différence entre deux couleurs - ou à quel point il est facile de les distinguer.

Les couleurs avec un contraste plus élevé seront plus visuellement distinctes, tandis que les couleurs avec un contraste plus faible seront plus visuellement similaires. Par exemple, le noir et le blanc ont un ratio de contraste très élevé. Tandis que le bleu clair et le violet clair ont moins de contraste.

:::interactive_editor

```html
<style>
  .example {
    padding: 20px;
    margin-bottom: 20px;
    font-size: 18px;
    font-family: sans-serif;
  }

  .high-contrast {
    background-color: black;
    color: white;
  }

  .low-contrast {
    background-color: #add8e6; 
    color: #dda0dd; 
  }

  .label {
    font-weight: bold;
    margin-bottom: 5px;
  }
</style>

<div class="example high-contrast">
  <div class="label">High Contrast</div>
  This text has high contrast (black background and white text).
</div>

<div class="example low-contrast">
  <div class="label">Low Contrast</div>
  This text has low contrast (light blue background and light purple text).
</div>
```

:::

Bien sûr, dans ces exemples la distinction pourrait être rendue plus claire à cause de la mise en page. Mais qu'en est-il d'appliquer ces couleurs au texte ? Tu peux avoir le texte noir à fort contraste sur un fond blanc. Et le texte violet à faible contraste sur un fond bleu.

:::interactive_editor

```html
<style>
  .text-example {
    padding: 10px;
    margin-bottom: 20px;
    font-size: 18px;
    font-family: sans-serif;
  }

  .text-high-contrast {
    background-color: white;
    color: black;
  }

  .text-low-contrast {
    background-color: #add8e6; 
    color: #dda0dd; 
  }

  .label {
    font-weight: bold;
    margin-bottom: 5px;
    display: block;
  }
</style>

<div class="text-example text-high-contrast">
  <span class="label">High Contrast Text</span>
  This is high contrast text: black text on a white background.
</div>

<div class="text-example text-low-contrast">
  <span class="label">Low Contrast Text</span>
  This is low contrast text: purple text on a light blue background.
</div>
```

:::

Mais comment déterminer ce qui est un contraste « assez bon » ? Tu ne peux pas te baser uniquement sur l'apparence du texte pour toi, car chaque utilisateur aura une expérience différente. Heureusement, les Web Content Accessibility Guidelines, ou WCAG, fournissent un standard pour ça.

Le texte avec un ratio de contraste de 4,5:1 est considéré comme le standard AA, qui est le minimum strict que tu devrais suivre pour être accessible à la plupart des utilisateurs. Le texte avec un ratio de contraste de 7:1 est considéré comme le standard AAA, et garantit le meilleur niveau d'accessibilité.

:::interactive_editor

```html
<style>
  .contrast-example {
    padding: 15px;
    margin-bottom: 20px;
    font-size: 18px;
    font-family: sans-serif;
  }

  .aa-contrast {
    background-color: #ffffff; 
    color: #4b4b4b; 
  }

  .aaa-contrast {
    background-color: #ffffff; 
    color: #1a1a1a; 
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }
</style>

<div class="contrast-example aa-contrast">
  <span class="label">AA Standard (Contrast Ratio ~4.5:1)</span>
  This text meets the minimum accessibility standard for contrast.
</div>

<div class="contrast-example aaa-contrast">
  <span class="label">AAA Standard (Contrast Ratio ~7:1)</span>
  This text meets the highest accessibility standard for contrast.
</div>
```

:::

Il existe plusieurs sites web qui peuvent vérifier le ratio de contraste entre deux couleurs, mais la plupart des navigateurs te permettront de faire ça directement dans les outils de développement sur ton site web.

Regardons quelques exemples supplémentaires de ratio de contraste pour mieux comprendre le concept.

Voici un exemple de texte noir sur un fond blanc qui a un ratio de contraste de 21:1.

```html

  <span class="label">Contrast Ratio 21:1</span>
  This is black text on a white background, which has the highest contrast ratio of 21:1.
```

:::

Cet exemple dépasse largement le standard AAA. Maintenant, regardons du texte violet sur un fond bleu.

:::interactive_editor

```html
<style>
  .contrast-21 {

  }

<div class="contrast-21">
  <span class="label">Contrast Ratio 21:1</span>
  This example more than meets the AAA standard with black text on white background.
</div>

  <span class="label">Purple on Blue (Lower Contrast)</span>
```

Cet exemple a un ratio de contraste de 1,48:1 qui ne respecte même pas le standard AA.

Comment corriger ça ? Eh bien, il y a trois aspects qui affectent le ratio de contraste.

Le premier est la teinte (hue), qui représente le type général de couleur, comme rouge, bleu, orange. Passons à une teinte plus éloignée du bleu. Dans ce cas, utilisons le rouge.

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }

  .red-on-blue {
    background-color: #0000cc; 
    color: #ff0000; 
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }
</style>

<div class="contrast-21">
  <span class="label">Contrast Ratio 21:1</span>
  This example more than meets the AAA standard with black text on white background.
</div>

<div class="red-on-blue">
  <span class="label">Red on Blue (Higher Contrast Hue Shift)</span>
  This doesn't meet accessibility standards.
</div>
```

:::

Malheureusement, ce changement n'a fait passer le ratio de contraste qu'à 1,49:1, ce qui signifie que tu dois changer la saturation, ou la « quantité » de couleur présente. Augmentons la quantité de rouge dans le texte. Cela nous rapproche beaucoup plus du but, avec un ratio de contraste de 3,54:1.

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }

  .low-sat-red-on-blue {
    background-color: #0000cc; 
    color: #b23333; 
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .high-sat-red-on-blue {
    background-color: #0000cc; 
    color: #ff4d4d;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }
</style>

<div class="contrast-21">
  <span class="label">Contrast Ratio 21:1</span>
  This example more than meets the AAA standard with black text on white background.
</div>

<div class="low-sat-red-on-blue">
  <span class="label">Low Saturation Red on Blue (Contrast ~1.49:1)</span>
  This red has low saturation, resulting in a poor contrast ratio.
</div>

<div class="high-sat-red-on-blue">
  <span class="label">Higher Saturation Red on Blue (Contrast ~3.54:1)</span>
  Increasing the saturation of red improves contrast but it’s still below AA standard.
</div>
```

:::

Cependant, tu peux potentiellement te rapprocher encore plus en changeant la dernière valeur, qui est la luminosité (lightness). La luminosité représente la quantité de blanc présente dans la couleur.

Diminuons la luminosité du rouge. Maintenant il y a un rouge beaucoup plus foncé contre le fond bleu clair, ce qui amène le ratio de contraste à 10,34:1.

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }

  .blue-bg {
    background-color: #add8e6; 
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .low-sat-red-on-blue {
    background-color: #0000cc; 
    color: #b23333;
    margin-bottom: 20px;
  }

  .high-sat-red-on-blue {
    background-color: #0000cc; 
    color: #ff4d4d; 
    margin-bottom: 20px;
  }
  
  .dark-red-on-light-blue {
    background-color: #add8e6; 
    color: #8b0000;
  }
</style>

<div class="contrast-21">
  <span class="label">Contrast Ratio 21:1</span>
  This example more than meets the AAA standard with black text on white background.
</div>

<div class="low-sat-red-on-blue blue-bg">
  <span class="label">Low Saturation Red on Medium Blue (Contrast ~1.49:1)</span>
  This red has low saturation, resulting in a poor contrast ratio.
</div>

<div class="high-sat-red-on-blue blue-bg">
  <span class="label">Higher Saturation Red on Medium Blue (Contrast ~3.54:1)</span>
  Increasing the saturation of red improves contrast but it’s still below AA standard.
</div>

<div class="dark-red-on-light-blue">
  <span class="label">Darker Red on Light Blue (Contrast ~10.34:1)</span>
  Decreasing the lightness of the red increases the contrast ratio significantly.
</div>
```

:::

Tu pourrais continuer à ajuster les couleurs pour trouver l'équilibre entre l'effet visuel que tu veux et un ratio de contraste accessible. Mais il est important de te rappeler que l'accessibilité devrait toujours avoir la priorité.
