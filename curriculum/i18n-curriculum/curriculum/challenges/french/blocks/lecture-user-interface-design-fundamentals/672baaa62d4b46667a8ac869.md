# --interactive--

**NOTE** : Certains des exemples interactifs peuvent utiliser du CSS que tu n'as pas encore appris. Ne t'inquiète pas si tu essaies de tout comprendre. Le but des exemples est de te montrer des aperçus de ces concepts de design pour que tu comprennes mieux comment les choses fonctionnent.

La hiérarchie visuelle fait référence à la façon dont tu mets en page et affiches le contenu de ta page pour guider l'attention du spectateur.

Une hiérarchie forte peut fournir un chemin clair que l'œil peut suivre, garantissant que l'information que tu transmets est consommée dans l'ordre que tu as prévu.

Considérons une mise en page de base où le HTML de la page est sémantiquement correct, mais le style appliqué ne crée pas une hiérarchie visuelle forte.

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    color: #333;
    background-color: #fff;
  }

  header, nav, main, section, footer {
    padding: 10px;
    margin-bottom: 10px;
    border: 1px solid #ccc;
  }

  h1, h2, h3 {
    font-weight: normal;
    font-size: 16px;
    margin: 5px 0;
  }

  nav a {
    text-decoration: none;
    color: #333;
    margin-right: 10px;
    font-size: 14px;
  }

  p {
    font-size: 14px;
    margin: 5px 0;
  }
</style>

<header>
  <h1>My Website</h1>
</header>

<nav>
  <a href="#">Home</a>
  <a href="#">About</a>
  <a href="#">Services</a>
  <a href="#">Contact</a>
</nav>

<main>
  <section>
    <h2>Welcome</h2>
    <p>This is the welcome section of the homepage.</p>
  </section>

  <section>
    <h2>Our Services</h2>
    <p>Here we describe what we offer.</p>
  </section>

  <section>
    <h2>Get in Touch</h2>
    <p>Contact us for more information.</p>
  </section>
</main>

<footer>
  <p>&copy; 2025 My Website</p>
</footer>
```

:::

Si la taille de la police n'est pas distincte, il n'y a pas d'indication visible du flux du document, bien que les choses soient séparées par des titres.

Pour créer une hiérarchie visuelle, tu devrais appliquer différentes tailles de police aux niveaux de titres. Tu pourrais aussi utiliser quelque chose comme une « callout box » (boîte de mise en avant) pour mettre en évidence une section spécifique.

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    line-height: 1.6;
    padding: 20px;
    background: #f9f9f9;
    color: #333;
  }

  h1 {
    font-size: 32px;
    margin-bottom: 10px;
  }

  h2 {
    font-size: 24px;
    margin-top: 20px;
    margin-bottom: 8px;
  }

  h3 {
    font-size: 18px;
    margin-top: 15px;
    margin-bottom: 6px;
  }

  p {
    font-size: 16px;
    margin-bottom: 12px;
  }

  .callout {
    background-color: #fff3cd;
    border-left: 5px solid #ffc107;
    padding: 15px;
    margin: 20px 0;
  }
</style>

<h1>Understanding Visual Hierarchy</h1>
<p>Visual hierarchy helps users navigate and understand content by guiding their attention.</p>

<h2>Heading Tiers</h2>
<p>Using different font sizes for headings creates structure and makes content scannable.</p>

<h3>Level 3 Heading</h3>
<p>This smaller heading further breaks down the section without overpowering it.</p>

<div class="callout">
  <strong>Tip:</strong> Use a callout box like this to highlight important notes or key takeaways.
</div>
```

:::

La hiérarchie visuelle peut aussi aider à augmenter ta conversion d'utilisateurs. Par exemple, tu peux profiter de la callout box pour attirer davantage l'attention sur un bouton Call to Action (CTA).

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 20px;
    background-color: #f4f4f4;
    color: #333;
  }

  h2 {
    font-size: 24px;
    margin-bottom: 10px;
  }

  p {
    font-size: 16px;
    margin-bottom: 15px;
  }

  .callout {
    background-color: #e0f7fa;
    border-left: 5px solid #00acc1;
    padding: 20px;
    margin-top: 20px;
    text-align: center;
  }

  .cta-button {
    display: inline-block;
    background-color: #00acc1;
    color: white;
    padding: 12px 20px;
    font-size: 16px;
    text-decoration: none;
    border-radius: 4px;
    margin-top: 10px;
  }

  .cta-button:hover {
    background-color: #008b9a;
  }
</style>

<h2>Ready to Boost Your Productivity?</h2>
<p>Join thousands of users who are getting more done with our simple and effective tools.</p>

<div class="callout">
  <strong>Don’t wait!</strong> Start your free trial today and see the difference.
  <br>
  <a href="#" class="cta-button">Start Free Trial</a>
</div>
```

:::

Enfin, ta hiérarchie visuelle peut être importante pour transmettre d'autres composants, comme une barre de navigation ou un pied de page.

Cela rend plus facile pour tes utilisateurs de trouver l'information essentielle qu'ils peuvent chercher.
