# --interactive--

L'élément exposant est utilisé pour afficher un morceau de texte en exposant. Un exposant est un symbole ou une lettre imprimés au-dessus de la ligne normale du texte.

Voici un exemple qui utilise l'élément exposant pour illustrer des puissances (pour voir les aperçus, active l'éditeur interactif) :

:::interactive_editor

```html
<p>2<sup>2</sup> (2 squared) is 4.</p>
```

:::

Dans cet exemple, le nombre 2 est enveloppé dans des balises `sup` pour représenter l'exposant à l'intérieur du paragraphe. Dans la fenêtre d'aperçu, tu verras que le second 2 est plus petit et légèrement plus haut que le premier 2.

Les cas d'usage courants pour l'élément exposant sont les puissances, les lettres supérieures et les nombres ordinaux. Voici un exemple qui utilise l'élément exposant pour des lettres supérieures (pour voir les aperçus, active l'éditeur interactif) : 

:::interactive_editor

```html
<p>
  Monseigneur is often written as <strong>M<sup>gr</sup></strong>.
</p>
```

:::

Les lettres supérieures sont des lettres écrites en exposant, généralement pour indiquer des abréviations. Les lettres `g` et `r` sont entourées de balises exposant pour illustrer l'abréviation dans cet exemple.

Il est important de noter que l'élément exposant ne devrait être utilisé que pour des raisons typographiques. Si tu veux styler un morceau de texte avec une ligne de base remontée, tu devrais utiliser du CSS au lieu de l'élément exposant.

Pour représenter des équations chimiques en HTML, tu utiliserais l'élément indice. Cet élément utilise un indice qui affiche une ligne de base abaissée en utilisant un texte plus petit.

Voici un exemple qui utilise l'élément indice pour montrer la représentation chimique du dioxyde de carbone (pour voir les aperçus, active l'éditeur interactif) :

:::interactive_editor

```html
<p>CO<sub>2</sub></p>
```

:::

Le nombre deux est enveloppé dans des balises `sub` pour illustrer que le caractère devrait être un indice.

Les cas d'usage courants pour l'élément indice sont les formules chimiques, les notes de bas de page et les indices de variables.
