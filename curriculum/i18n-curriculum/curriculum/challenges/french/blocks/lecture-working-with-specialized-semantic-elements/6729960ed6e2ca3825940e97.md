# --interactive--

L'élément code en ligne est utilisé pour représenter de courts extraits de code à l'intérieur d'un texte. Les cas d'usage courants pour l'élément code sont les articles techniques et les pages de documentation.

Voici un exemple qui utilise l'élément `code` pour montrer un extrait de code CSS :

:::interactive_editor

```html
<p>
  To set the text color to blue in CSS, use the following code:
  <code>color: blue;</code>
</p>
```

:::

Dans cet exemple, la propriété CSS `color` est utilisée pour mettre la couleur du texte à `blue`. En enveloppant cet extrait de code dans des balises `<code>`, on indique au navigateur que ce texte est un morceau de code en ligne.

Le navigateur appliquera des styles par défaut au contenu à l'intérieur de l'élément `code`. Le style par défaut est une police à chasse fixe (monospaced).

L'élément `code` est destiné à représenter une seule ligne de code. Si tu veux représenter plusieurs lignes de code, tu auras besoin de placer un élément `code` à l'intérieur d'un élément texte préformaté.

L'élément texte préformaté est utilisé pour représenter du texte préformaté. Voici un exemple qui utilise l'élément texte préformaté pour montrer une déclaration CSS :

:::interactive_editor

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

:::

Quand tu utilises l'élément `pre`, tu dois faire attention à l'espacement parce qu'il s'affichera exactement comme écrit dans le document HTML.

Dans le navigateur, tu verras que le code est en retrait de plusieurs espaces vers la droite. Si tu changeais l'indentation dans l'exemple de code, tu verrais une différence d'indentation à l'écran.

Quand il s'agit d'inclure des exemples de code dans ton document HTML, tu devrais utiliser l'élément `code` pour de courts exemples en ligne. 

Si tu as besoin d'afficher des extraits de code plus longs, tu auras besoin d'utiliser les éléments `pre` et `code`.
