# --interactive--

L'élément annotation non articulée, ou élément `u` en court, est utilisé pour représenter du texte en ligne auquel une annotation non textuelle est appliquée.

Voici un exemple qui utilise l'élément `u` pour mettre en valeur diverses fautes d'orthographe :

:::interactive_editor

```html
<p>
  You can use the unarticulated annotation element to highlight
  <u>inccccort</u> <u>spling</u> <u>issses</u>.
</p>
```

:::

Dans cet exemple, les mots `incorrect`, `spelling` et `issues` sont mal orthographiés. Le style par défaut de l'élément `u` est un soulignement noir sous le texte.

En HTML4, l'élément `u` était utilisé à des fins de style. Mais en HTML5, l'élément `u` ne devrait être utilisé que pour indiquer qu'une annotation non textuelle est appliquée au texte.

Si tu veux styler un morceau de texte avec un soulignement, tu devrais utiliser du CSS plutôt que du HTML.

L'élément barré, ou élément `s` en court, devrait être utilisé pour représenter du texte qui n'est plus exact ou pertinent. Voici un exemple qui utilise l'élément `s` pour montrer l'annulation d'une activité :

:::interactive_editor

```html
<p><s>Tomorrow's hike will be meeting at noon.</s></p>

<p>Due to unforeseen weather conditions, the hike has been canceled.</p>
```

:::

Dans cet exemple, la première phrase est barrée parce que la randonnée a été annulée pour des raisons météorologiques.

L'élément `s` ne devrait jamais être utilisé juste pour montrer des changements sur un document. Dans ce cas, des éléments plus appropriés seraient l'élément texte supprimé et l'élément texte inséré.

L'élément `ruby` représente un petit texte affiché au-dessus ou en dessous du texte principal. Il est généralement utilisé pour montrer la prononciation de caractères d'Asie de l'Est. Voici l'exemple de l'élément `ruby` tiré de la page MDN web docs :

:::interactive_editor

```html
<ruby> 明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp> </ruby>
```

:::

L'élément `rp`, ou élément parenthèse de repli ruby, est utilisé comme repli pour les navigateurs qui ne prennent pas en charge l'affichage des annotations ruby.

L'élément `rt`, ou élément texte ruby, est utilisé pour indiquer le texte de l'annotation ruby. Ce texte est généralement utilisé pour la prononciation, ou les détails de traduction en typographie d'Asie de l'Est.

Bien que l'élément `ruby` puisse être utilisé pour d'autres types d'annotations, le cas d'usage le plus courant est la typographie d'Asie de l'Est.
