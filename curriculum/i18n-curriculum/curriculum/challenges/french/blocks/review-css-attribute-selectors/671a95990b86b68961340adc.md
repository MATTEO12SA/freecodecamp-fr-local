# --interactive--

## Travailler avec différents sélecteurs d'attribut et des liens

- **Définition** : Le sélecteur d'`attribute` te permet de cibler des éléments HTML selon leurs attributs, comme les attributs `href` ou `title`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Link with href</a>
<a>No href</a>
```

```css
a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

- **Attribut `title`** : Cet attribut fournit des informations supplémentaires sur un élément. Voici comment cibler les liens avec l'attribut `title` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Tooltip">Link with title</a>
<a href="#">Normal link</a>
```

```css
a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

- **Combiner des sélecteurs pour faire correspondre les liens qui ont à la fois les attributs `href` et `title`** : Combine plusieurs sélecteurs d'attribut.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Info">Href + Title</a>
<a href="#">Only href</a>
```

```css
a[href][title] {
  color: green;
}
```

:::

- **Faire correspondre un seul mot dans une liste séparée par des espaces** : Cible les liens qui ont un mot de classe précis.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a class="link primary">Primary link</a>
<a class="link secondary">Secondary link</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

- **Faire correspondre les valeurs qui commencent par un préfixe précis** : Cible les liens qui commencent par `https://`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">HTTPS link</a>
<a href="http://www.freecodecamp.org">HTTP link</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

- **Faire correspondre les valeurs qui se terminent par un suffixe précis** : Cible les liens qui se terminent par `.jpg`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="photo.jpg">Image link</a>
<a href="index.html">HTML link</a>
```

```css
a[href$=".jpg"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

- **Faire correspondre les valeurs qui contiennent une sous-chaîne n'importe où** : Cible les liens qui contiennent `https` n'importe où dans la valeur.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Secure link</a>
<a href="page.html">Local link</a>
```

```css
a[href*="https"] {
  color: teal;
}
```

:::

- **Résumé** : Ces motifs permettent de styliser facilement et de manière cohérente les liens selon leurs attributs et leurs valeurs.

## Cibler des éléments avec les attributs `lang` et `data-lang`

- **Attribut `lang`** : Cet attribut est utilisé en HTML pour préciser la langue du contenu à l'intérieur d'un élément. Tu peux vouloir styliser des éléments différemment selon la langue dans laquelle ils sont écrits, surtout sur un site web multilingue.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">English text</p>
<p lang="fr">French text</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

- **Attribut `data-lang`** : Les attributs de données personnalisés comme l'attribut `data-lang` sont couramment utilisés pour stocker des informations supplémentaires dans des éléments, par exemple pour préciser la langue utilisée dans une section précise de texte. Voici comment styliser des éléments selon l'attribut `data-lang` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div data-lang="fr">French content</div>
<div data-lang="en">English content</div>
```

```css
div[data-lang="fr"] {
  color: blue;
}
```

:::

## Travailler avec le sélecteur d'attribut, les éléments de liste ordonnée et l'attribut `type`

- **Attribut `type`** : Quand tu travailles avec des listes ordonnées en HTML, l'attribut `type` te permet de préciser le style de numérotation utilisé, comme une numérotation numérique, alphabétique ou en chiffres romains.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Alpha</li>
  <li>Beta</li>
</ol>

<ol type="i">
  <li>One</li>
  <li>Two</li>
</ol>
```

```css
/*Example targeting uppercase alphabetical numbering*/
ol[type="A"] {
  color: purple;
  font-weight: bold;
}

/*Example targeting lowercase Roman numerals*/
ol[type="i"] {
  color: green;
}
```
