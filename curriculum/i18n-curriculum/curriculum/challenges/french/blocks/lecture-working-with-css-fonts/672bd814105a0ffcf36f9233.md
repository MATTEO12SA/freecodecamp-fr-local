# --interactive--

Une famille de polices est un groupe de polices qui partagent un design commun. Toutes les polices qui appartiennent à la même famille sont basées sur la même famille de caractères principale, mais elles ont aussi des variations dans leur style, leur épaisseur et leur largeur. Tu peux les imaginer comme des frères et sœurs qui partagent des caractéristiques similaires tout en ayant quelques différences. Par exemple, Arial est une famille de polices qui inclut des variations comme Arial Bold et Arial Italic.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-font-families-and-how-do-they-work-1.png" alt="Schéma montrant 'Arial' en haut avec deux flèches bleues pointant vers 'Arial Bold' à gauche et 'Arial Italic' à droite, illustrant les variations d'une famille de polices" />

En CSS, tu peux définir la famille de polices avec la propriété `font-family`. Voici un exemple qui définit la police du paragraphe sur `Arial` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="arial-font">Example paragraph using Arial font.</p>
<p>Paragraph not using Arial font.</p>
```

```css
#arial-font {
  font-family: Arial;
}
```

:::

Mais que se passe-t-il si la famille de polices n'est pas trouvée ? Tu peux préciser plusieurs familles de polices par ordre de priorité, de la plus élevée à la plus basse, en les séparant par des virgules. Ces polices alternatives serviront d'options de secours. Dans cet exemple, Arial est une police principale et Lato est une police alternative :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="specified-font">Example paragraph using specified fonts.</p>
<p>Paragraph not using specified fonts.</p>
```

```css
#specified-font {
  font-family: Arial, Lato;
}
```

:::

Le navigateur rendra la première police (Arial) si elle est trouvée sur l'appareil. Sinon, la deuxième police (Lato) sera rendue.

Tu devrais aussi savoir comment ce processus fonctionne en coulisses. Le processus de sélection ne s'arrête pas si la première police est disponible. La famille de polices est choisie caractère par caractère, donc si une police ne contient pas un caractère précis, le navigateur le cherche dans les polices de priorité inférieure. Intéressant, non ?

Dans le contexte du développement web, tu trouveras aussi des familles de polices génériques. Elles sont très importantes. Une famille de polices générique est une police par défaut que les navigateurs web rendront quand les polices de priorité supérieure ne sont pas disponibles. Pour s'assurer que le contenu reste lisible, le navigateur remplace la police d'origine par la police la plus appropriée trouvée, selon la famille de polices générique précisée. Parmi les plus couramment utilisées, on trouve :

- serif
- sans-serif
- monospace
- cursive
- fantasy

Mais il y a plus d'options parmi lesquelles choisir. Dans le code ci-dessous, nous avons trois familles de polices alternatives :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="specified-font">Example paragraph using specified fonts.</p>
<p>Paragraph not using specified fonts.</p>
```

```css
#specified-font {
  font-family: Arial, Lato, sans-serif;
}
```

:::

Arial a la priorité la plus élevée. Si Arial n'est pas trouvée, le navigateur essaiera de rendre Lato. Si aucune des deux n'est trouvée, le navigateur utilisera la famille de polices générique sans-serif, en sélectionnant une police avec ces caractéristiques parmi celles installées sur le système de l'utilisateur.

Tu devrais toujours inclure une famille de polices générique à la fin de la liste `font-family`. Garde à l'esprit que la police générique peut avoir une apparence différente de la police que tu avais imaginée à l'origine dans ton design, donc il est toujours utile de vérifier l'apparence de ces polices de secours dans différents navigateurs.

Pour rendre l'expérience utilisateur aussi cohérente que possible, il est aussi recommandé d'utiliser des polices web-safe. Ces familles de polices sont généralement installées sur la plupart des appareils, donc elles ont de fortes chances d'être trouvées et rendues correctement pour la plupart des utilisateurs. Tu en apprendras davantage à leur sujet dans les prochaines leçons.

Les familles de polices sont essentielles dans le monde du design web. En utilisant la propriété `font-family`, tu peux transformer instantanément l'apparence et l'ambiance de ton texte.
