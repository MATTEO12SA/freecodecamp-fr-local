# --interactive--

CSS n'applique aucune ombre au texte par défaut. Voici un exemple de paragraphe sans aucune ombre. Mais si tu dois ajouter des ombres, tu peux utiliser la propriété `text-shadow`.

```css
text-shadow: /* Values */
```

En CSS, tu peux décrire une ombre avec une combinaison de son décalage `X`, de son décalage `Y`, de son rayon de flou et de sa couleur. D'abord, tu dois préciser les décalages `X` et `Y`, qui représentent respectivement la distance horizontale et verticale entre l'ombre et le texte. Ces valeurs sont obligatoires.

Voici un exemple d'application d'un décalage `X` et `Y` sur une ombre. Nous appliquons la propriété `text-shadow` avec un décalage x de `3px` et un décalage y de `2px` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px;
}
```

:::

Dans le navigateur, le texte et l'ombre auront une apparence similaire, mais nous pouvons aussi personnaliser la couleur de l'ombre en précisant la valeur avant ou après le décalage. Définissons la couleur de l'ombre. Nous utiliserons ici une couleur hexadécimale, mais tu peux utiliser n'importe quel format de couleur valide.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px #00ffc3;
}
```

:::

Nous écrivons la couleur après le décalage, mais tu peux aussi écrire la couleur avant le décalage si tu préfères. C'est équivalent :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: #00ffc3 3px 2px;
}
```

:::

Dans le navigateur, l'ombre aura une couleur précise, ce qui nous permet de la distinguer très facilement du texte. Maintenant que nous pouvons différencier l'ombre du texte, il est aussi important de voir comment les valeurs positives et négatives affectent le décalage de l'ombre. Les valeurs positives du décalage `X` et du décalage `Y` déplacent l'ombre respectivement vers la droite et vers le bas, tandis que les valeurs négatives la déplacent vers la gauche et vers le haut. Voici un exemple qui utilise plutôt des valeurs négatives :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: -3px -2px #00ffc3;
}
```

:::

Maintenant, l'ombre se déplacera vers la gauche et vers le haut par rapport au texte. Très bien. Mais l'ombre n'est pas très esthétique, car elle ressemble exactement au texte original, simplement placé derrière lui. Pour obtenir un meilleur rendu, nous devons ajouter une troisième valeur : le rayon de flou. Cette valeur est facultative, mais elle rend l'ombre beaucoup plus douce et subtile. La valeur par défaut du rayon de flou est zéro. Plus la valeur est élevée, plus le flou est grand, ce qui signifie que l'ombre devient plus légère.

Ici, nous réglons le rayon de flou sur `3px` et nous revenons à des valeurs positives pour le décalage :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px 3px #00ffc3;
}
```

:::

Maintenant, cela commence à ressembler à une ombre. Elle est plus floue et subtile, ce qui permet de se concentrer sur le texte principal pendant que l'ombre ajoute du style en arrière-plan. À partir de là, nous pouvons ajuster un peu les valeurs jusqu'à trouver une combinaison de décalage, de couleur et de rayon de flou qui répond à nos besoins.

Il est aussi utile de savoir que le texte peut avoir plus d'une ombre. Il te suffit de les écrire dans la même propriété `text-shadow`, séparées par des virgules. Elles seront appliquées en couches, de l'avant vers l'arrière, avec la première ombre au premier plan.

Voici un exemple de paragraphe avec trois ombres appliquées en couches. Remarque que toutes ces ombres sont définies dans la même propriété `text-shadow` et séparées par des virgules :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow:
    3px 2px 3px #00ffc3,
    -3px -2px 3px #0077ff,
    5px 4px 3px #dee7e5;
}
```

:::

`text-shadow` est une propriété CSS puissante pour faire ressortir ton texte. Elle peut aussi améliorer le design global de ton application web. En choisissant le décalage, le flou et la couleur, tu peux créer une grande variété d'effets d'ombre.
