## Théorie des couleurs

- **Définition de la théorie des couleurs** : C'est l'étude de comment les couleurs interagissent les unes avec les autres et comment elles affectent notre perception. Elle couvre les relations entre couleurs, l'harmonie des couleurs et l'impact psychologique de la couleur.
- **Couleurs primaires** : Ces couleurs, qui sont jaune, bleu et rouge, sont les teintes fondamentales à partir desquelles toutes les autres couleurs sont dérivées.
- **Couleurs secondaires** : Ces couleurs résultent du mélange de quantités égales de deux couleurs primaires. Le vert, l'orange et le violet sont des exemples de couleurs secondaires.
- **Couleurs tertiaires** : Ces couleurs résultent de la combinaison d'une couleur primaire avec une couleur secondaire voisine. Jaune-Vert, Bleu-Vert et Bleu-Violet sont des exemples de couleurs tertiaires.
- **Couleurs chaudes** : Ces couleurs qui incluent les rouges, oranges et jaunes, évoquent des sentiments de confort, de chaleur et de cocooning.
- **Couleurs froides** : Ces couleurs qui incluent les bleus, verts et violets, évoquent des sentiments de calme, de sérénité et de professionnalisme.
- **Roue chromatique** : La roue chromatique est un diagramme circulaire qui montre comment les couleurs se rapportent les unes aux autres. C'est un outil essentiel pour les designers parce qu'elle les aide à sélectionner des combinaisons de couleurs.
- **Schémas de couleurs analogues** : Ces schémas de couleurs créent des expériences cohérentes et apaisantes. Ils ont des couleurs analogues, qui sont adjacentes les unes aux autres dans la roue chromatique.
- **Schémas de couleurs complémentaires** : Ces schémas de couleurs créent un fort contraste et un impact visuel. Leurs couleurs sont situées aux extrémités opposées de la roue chromatique, l'une par rapport à l'autre.
- **Schéma de couleurs triadique** : Ce schéma de couleurs a des couleurs vibrantes. Elles sont faites de couleurs qui sont approximativement équidistantes les unes des autres. Si elles sont connectées, elles forment un triangle équilatéral sur la roue chromatique.
- **Schéma de couleurs monochromatique** : Pour ce schéma de couleurs, toutes les couleurs sont dérivées de la même couleur de base en ajustant sa luminosité, son obscurité et sa saturation. Cela évoque un sentiment d'unité et d'harmonie tout en créant du contraste.

## Différentes façons de travailler avec les couleurs en CSS

- **Couleurs nommées** : Ces couleurs sont des noms de couleurs prédéfinis reconnus par les navigateurs. Des exemples incluent `blue`, `darkred`, `lightgreen`.
- **Fonction `rgb()`** : RGB signifie Red, Green et Blue — les couleurs primaires de la lumière. Ces trois couleurs sont combinées à différentes intensités pour créer une large gamme de couleurs. La fonction `rgb()` te permet de définir des couleurs en utilisant le modèle de couleur RGB.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **Fonction `rgba()`** : Cette fonction ajoute une quatrième valeur, alpha, qui contrôle la transparence de la couleur. Si non fournie, la valeur alpha par défaut est `1`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **Fonction `hsl()`** : HSL signifie Hue, Saturation et Lightness — trois composants clés qui définissent une couleur.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
```

```css
p {
  color: hsl(120, 100%, 50%);
}
```

:::

- **Fonction `hsla()`** : Cette fonction ajoute une quatrième valeur, alpha, qui contrôle l'opacité de la couleur.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **Hexadécimal** : Un code hex (abréviation de code hexadécimal) est une chaîne de six caractères utilisée pour représenter les couleurs dans le modèle de couleur RGB. Le « hex » fait référence au système de numérotation en base 16, qui utilise les chiffres de 0 à 9 et les lettres A à F.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## La propriété `box-shadow`

- **Définition** : La propriété `box-shadow` applique une ou plusieurs ombres autour d'un élément.
- **Valeurs de décalage** : Tu dois spécifier des valeurs horizontale (`offset-x`) et verticale (`offset-y`). Un `offset-x` positif déplace l'ombre vers la droite, tandis que des valeurs négatives la déplacent vers la gauche. Un `offset-y` positif déplace l'ombre vers le bas, tandis que des valeurs négatives la déplacent vers le haut. Si une valeur est `0`, tu n'as pas besoin d'inclure une unité.
- **Rayon de flou** : Cette valeur optionnelle contrôle à quel point l'ombre apparaît floue. Si non incluse, elle est par défaut à `0`, ce qui crée des bords nets. Plus la valeur est élevée, plus l'ombre est douce.
- **Rayon d'étalement** : Cette valeur optionnelle contrôle à quel point l'ombre s'étend ou rétrécit. Si non incluse, elle est par défaut à `0`.
- **Couleur de l'ombre** : Tu peux spécifier la couleur en utilisant des couleurs nommées, des valeurs hex, ou les fonctions `rgb()`, `rgba()`, `hsl()` ou `hsla()`.
- **Mot-clé `inset`** : Ajouter le mot-clé `inset` place l'ombre à l'intérieur de l'élément au lieu de l'extérieur.
- **Appliquer plusieurs box shadows** : Tu peux appliquer plusieurs ombres en les séparant par des virgules. Les ombres sont superposées de l'avant vers l'arrière.

### Syntaxe

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="shadow-box">Shadow Color Example</div>
```

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

:::

## Dégradés linéaires et radiaux

- **Dégradés linéaires** : Ces dégradés créent un mélange graduel entre les couleurs le long d'une ligne droite. Tu peux contrôler la direction de cette ligne en utilisant des mots-clés comme `to top`, `to right`, `to bottom right`, ou des angles comme `45deg`. Tu peux utiliser n'importe quelle couleur CSS valide et autant d'arrêts de couleur que tu veux.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **Dégradés radiaux** : Ces dégradés créent des dégradés circulaires ou elliptiques qui rayonnent à partir d'un point central.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::
