# --interactive--

La propriété CSS `filter` est un outil puissant qui te permet d'appliquer des effets graphiques aux éléments d'une page web. Elle est particulièrement utile pour ajuster la présentation visuelle des images, des arrière-plans et même du texte sans modifier la ressource d'origine.

La propriété `filter` peut être utilisée pour créer divers effets, comme le flou, le décalage de couleur et les ajustements de contraste. Voyons comment fonctionne la propriété `filter` et explorons quelques exemples courants. La syntaxe de base d'une propriété `filter` est simple :

```css
selector {
  filter: function(amount);
}
```

Ici, `function` représente l'effet de filtre précis que tu veux appliquer, et `amount` est généralement une valeur qui détermine l'intensité de l'effet. Regardons maintenant quelques fonctions de filtre courantes et leurs usages.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: blur(2px);
}
```

:::

La fonction `blur` applique un flou gaussien à l'élément. La quantité est indiquée en pixels et représente le rayon du flou. Cette règle CSS appliquera un flou de 2 pixels à toutes les images de la page. L'effet `blur` peut être utile pour créer de la profondeur dans ton design ou pour masquer certaines parties d'une image.

La fonction `brightness` ajuste la luminosité de l'élément. Une valeur de `0%` rendra l'élément complètement noir, tandis que les valeurs supérieures à `100%` augmenteront la luminosité.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: brightness(150%);
}
```

:::

Cette règle CSS augmente la luminosité de l'élément image de `50%`. Les ajustements de luminosité peuvent être utilisés pour faire ressortir les images ou créer un effet délavé.

La fonction `grayscale` convertit l'élément en niveaux de gris. La quantité est définie en pourcentage, où `100%` correspond à des niveaux de gris complets et `0%` laisse l'image inchangée.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

Cette règle convertira l'élément image entièrement en niveaux de gris. `grayscale` peut être utilisé pour créer une apparence vintage ou pour mettre certains éléments d'une page au second plan.

La fonction `sepia` applique une teinte sépia à l'élément. Comme `grayscale`, elle utilise une valeur en pourcentage :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: sepia(80%);
}
```

:::

Cette règle applique un effet sépia de `80%` à l'élément image. L'effet sépia est très utile pour créer une apparence vintage ou ancienne.

La fonction `hue-rotate` applique une rotation de teinte à l'élément. La valeur est définie en degrés et représente une rotation autour du cercle des couleurs.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

Cette règle fait pivoter la teinte de l'élément image de `90` degrés. La rotation de teinte peut être utilisée pour créer des effets psychédéliques ou ajuster le schéma de couleurs global d'une image.

L'un des aspects les plus puissants de la propriété `filter` est la possibilité de combiner plusieurs effets. Tu peux appliquer plusieurs filtres au même élément en les séparant par des espaces :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

Cette règle applique un contraste augmenté, une luminosité légèrement augmentée et un léger effet sépia à l'élément image.

En combinant des filtres, tu peux créer des effets visuels complexes et uniques adaptés aux besoins de ton design. La propriété CSS `filter` est un outil polyvalent qui permet de manipuler visuellement les éléments web de façon créative.

Même si nous avons couvert certaines des fonctions de filtre les plus courantes, il en existe d'autres, comme `contrast`, `invert` et `saturate`. Comme pour toute fonctionnalité puissante, il est important de faire attention à la façon dont tu utilises les filtres pour améliorer ton design sans submerger tes utilisateurs ni compromettre l'accessibilité.
