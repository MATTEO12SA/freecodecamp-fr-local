# --interactive--

Le CSS est responsable des styles d'une page web. Tous ces styles sont composés de diverses règles CSS.

Une règle CSS est composée de deux parties principales : un **sélecteur** et un bloc de **déclaration**.

Regardons la syntaxe de base :

```css
selector {
  property: value;
}
```

Un sélecteur est un motif utilisé en CSS pour identifier et cibler des éléments HTML spécifiques pour la mise en forme.

Des exemples de sélecteurs incluent les sélecteurs de type, les sélecteurs de classe et les sélecteurs d'ID.

Les accolades fournies dans la syntaxe de base sont appelées un bloc de déclaration. Un bloc de déclaration applique un ensemble de styles pour un sélecteur ou des sélecteurs donnés.

À l'intérieur du bloc de déclaration, tu auras une série de déclarations. Chaque déclaration est composée d'une propriété et d'une valeur.

La propriété est l'identifiant CSS qui spécifie quelle fonctionnalité est mise en forme. Un exemple de propriété serait la propriété `background-color`.

La valeur serait le réglage spécifique appliqué à cette propriété. Par exemple, si la propriété est `background-color`, une valeur pourrait être `purple`, ce qui définit la couleur d'arrière-plan en violet.

Après chaque nom de propriété, tu dois placer deux points, et après chaque valeur, tu dois avoir un point-virgule.

Maintenant que tu connais la syntaxe d'une règle CSS, regardons un exemple. Active l'éditeur interactif et clique sur l'onglet `styles.css` pour voir le code CSS.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>Example paragraph element</p>
<p>Another example paragraph element</p>
```

```css
p {
  color: blue;
}
```

:::

Dans cette règle CSS, un sélecteur de type cible tous les éléments paragraphe de la page.

À l'intérieur du bloc de déclaration, il y a une déclaration.

La déclaration est composée de la propriété `color` avec une valeur définie à `blue`. Cela changera la couleur du texte pour tous les éléments paragraphe en `blue`.

Si tu veux appliquer le même ensemble de styles à plusieurs sélecteurs, tu peux créer une liste de sélecteurs avec chaque sélecteur séparé par une virgule.

Voici un exemple de mise en forme de plusieurs sélecteurs (pour interagir avec l'exemple, active l'éditeur interactif) :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">Example heading</h1>
<h2 class="subheading">Example subheading</h2>
<p>This paragraph is not affected by the selector.</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

:::

Dans cette liste de sélecteurs, il y a un sélecteur `id` qui cible l'élément HTML ayant la valeur `id` de `title`. Tous les sélecteurs `id` doivent commencer par un symbole dièse `#`.

Puis il y a une virgule suivie d'un sélecteur `class` qui cible tous les éléments HTML ayant la valeur `class` de `subheading`. Tous les sélecteurs de classe doivent commencer par un point `.`

Toute la liste de sélecteurs recevra la même mise en forme, avec la couleur du texte définie à `navy`.
