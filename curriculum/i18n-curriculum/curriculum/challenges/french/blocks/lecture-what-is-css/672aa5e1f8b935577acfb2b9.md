# --interactive--

Le CSS, qui signifie Cascading Style Sheets (feuilles de style en cascade), est un composant essentiel du développement web moderne. C'est un langage de balisage utilisé pour appliquer des styles au HTML. En termes simples, si le HTML est la structure d'une page web, le CSS est ce qui la rend belle.

Voici un exemple de mise en forme d'une barre de navigation.

**NOTE** : Ne t'inquiète pas si tu n'essayes pas de comprendre le code CSS pour le moment. Tu apprendras comment tout cela fonctionne dans les futures leçons et ateliers. C'est juste pour te donner une idée de ce que tu peux faire avec un peu de CSS.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Services</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f4f4f4;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #333;
  padding: 1rem 2rem;
  color: white;
}

.navbar .logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  transition: color 0.3s ease;
}

.nav-links a:hover {
  color: #ff9800;
}
```

:::

Le rôle principal du CSS est de séparer le contenu d'une page web de sa présentation visuelle. Cette séparation permet aux développeurs web de créer des sites plus flexibles et maintenables.

Avec le CSS, tu peux contrôler la mise en page, les couleurs, les polices et l'apparence visuelle globale des pages web sans modifier la structure HTML.

Prenons une analogie simple. Si tu considères un site web comme une maison, le HTML serait les fondations et la charpente, tandis que le CSS serait la peinture, le papier peint et les décorations qui rendent la maison visuellement attrayante et unique.

Le CSS fonctionne en sélectionnant des éléments HTML et en leur appliquant des styles.

Voici un exemple montrant un élément `button` non stylisé et un stylisé. Pour interagir avec cet exemple, tu devras activer l'éditeur interactif.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<button class="regular-btn">Unstyled Button</button>
<button class="round-btn">Round Button</button>
```

```css

.round-btn {
  font-size: 1rem;
  font-family: 'Segoe UI', sans-serif;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.2s ease;
  background-color: #2ecc71;
  color: white;
  border-radius: 50px;
  padding: 0.6rem 1.6rem;
}

.round-btn:hover {
  background-color: #27ae60;
  transform: translateY(-2px);
}
```

:::

Ces styles peuvent inclure des propriétés comme la couleur, la taille de la police, et bien d'autres. En changeant ces propriétés, tu peux modifier radicalement l'apparence d'une page web sans changer son contenu.

L'un des aspects les plus puissants du CSS est sa capacité à créer des designs responsives.

Cela signifie qu'avec le CSS, tu peux faire en sorte que ton site web soit superbe sur n'importe quel appareil, qu'il s'agisse d'un ordinateur de bureau, d'une tablette ou d'un smartphone.

Le CSS te permet d'ajuster les mises en page, les tailles de police et d'autres éléments visuels en fonction de la taille d'écran de l'appareil qui consulte le site web.

Essaie d'activer l'éditeur interactif et ajuste la taille de la fenêtre d'aperçu pour voir comment la mise en page s'adapte à la taille de l'écran.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<header class="header">
    <h1>Welcome to My Responsive Site</h1>
    <p>This layout adapts to your screen size!</p>
</header>

<main class="content">
    <section class="box">Box 1</section>
    <section class="box">Box 2</section>
    <section class="box">Box 3</section>
</main>
```

```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background-color: #f9f9f9;
  color: #333;
}

.header {
  background-color: #4CAF50;
  color: white;
  padding: 2rem;
  text-align: center;
}

.content {
  display: flex;
  gap: 1rem;
  padding: 1rem;
  justify-content: center;
}

.box {
  flex: 1;
  min-width: 200px;
  background-color: #ddd;
  padding: 2rem;
  text-align: center;
  font-size: 1.2rem;
  border-radius: 8px;
  transition: background-color 0.3s;
}

.box:hover {
  background-color: #ccc;
}

/* Responsive: for tablets and below (≤ 768px) */
@media (max-width: 768px) {
  .content {
    flex-direction: column;
    align-items: center;
  }

  .box {
    width: 90%;
    font-size: 1.1rem;
  }
}

/* Responsive: for phones (≤ 480px) */
@media (max-width: 480px) {
  .header {
    padding: 1.5rem 1rem;
  }

  .header h1 {
    font-size: 1.5rem;
  }

  .header p {
    font-size: 1rem;
  }

  .box {
    font-size: 1rem;
    padding: 1.5rem;
  }
}
```

:::

Une autre caractéristique importante du CSS est sa nature en cascade, d'où vient le « cascading » dans son nom.

Cela signifie que les styles peuvent être hérités et remplacés, permettant une structure hiérarchique de mise en forme. Tu en apprendras plus sur le fonctionnement dans les futures leçons.

Le CSS prend aussi en charge l'utilisation de feuilles de style externes.

Cela signifie que tu peux garder toutes tes règles de style dans un fichier séparé, qui peut ensuite être lié à plusieurs pages HTML.

Cette fonctionnalité améliore grandement la maintenabilité des sites web, surtout des plus gros. Au lieu d'avoir à changer les styles sur chaque page individuelle, tu peux faire des changements dans un seul fichier CSS qui affectera toutes les pages liées.

Dans le monde du développement web, le CSS joue un rôle essentiel dans la création de sites web visuellement attrayants, responsives et conviviaux.

Il permet aux développeurs de transformer de simples documents HTML en expériences web captivantes qui attirent l'attention des utilisateurs et améliorent leur interaction avec le contenu web.
