# --interactive--

Les combinateurs CSS sont utilisés pour définir la relation entre les sélecteurs en CSS. Ils aident à sélectionner les éléments en fonction de leur relation avec d'autres éléments, ce qui permet une mise en forme plus précise et efficace.

Nous allons discuter de quelques combinateurs CSS principaux et de leurs cas d'utilisation, en commençant par le combinateur descendant.

Un combinateur descendant est utilisé pour cibler les éléments correspondant au second sélecteur s'ils sont imbriqués dans un élément ancêtre qui correspond au premier sélecteur. Un ancêtre peut être un élément parent ou un parent de parent.

Pour mieux comprendre comment ça fonctionne, regardons un exemple.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<figure>
  <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Relaxing Cat">
  <figcaption>A relaxing cat with a border</figcaption>
</figure>
```

```css
figure img {
  border: 4px solid red;
}
```

:::

Dans l'exemple ci-dessus, on utilise le combinateur descendant pour sélectionner tous les éléments image à l'intérieur des éléments `figure` et appliquer une bordure `solid red` sur les quatre côtés.

Note qu'un `espace` est utilisé entre le sélecteur parent et le sélecteur enfant.

Dans ce cas, `figure` serait le parent et `img` serait l'enfant.

Si tu avais plusieurs images à l'intérieur d'un élément `figure`, utiliser le combinateur descendant serait un bon moyen d'appliquer une bordure noire continue autour de chacune de ces images.

Un autre type de combinateur serait le combinateur enfant.

Le combinateur enfant (`>`) en CSS est utilisé pour sélectionner les éléments qui sont des enfants directs d'un élément parent spécifié.

Ce combinateur cible uniquement les éléments avec un parent spécifique, rendant tes règles CSS plus précises et empêchant la mise en forme involontaire d'éléments imbriqués plus profondément.

Regardons l'exemple HTML suivant :

:::interactive_editor

```html
<div class="container">
  <p>First</p>
  <div>
    <p>Second</p>
  </div>
  <div>
    <p>Third</p>
  </div>
</div>
```

:::

Dans la structure HTML ci-dessus, la classe `container` est appliquée à un élément `div`.

À l'intérieur de ce conteneur, il y a un élément `p` enfant direct (« First »), suivi de deux éléments `div` supplémentaires, chacun contenant un élément `p` (« Second » et « Third »).

Le premier élément `p` est un enfant direct de l'élément `.container`, tandis que les deux autres éléments `p` sont imbriqués dans d'autres éléments `div`, ce qui les rend descendants plus profonds.

Pour appliquer des styles uniquement à l'enfant direct de la classe `container`, tu peux utiliser le combinateur enfant comme ceci :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <p>First</p>
  <div>
    <p>Second</p>
  </div>
  <div>
    <p>Third</p>
  </div>
</div>
```

```css
.container > p {
  color: blue;
}
```

:::

Dans l'exemple ci-dessus, on cible uniquement l'enfant direct de la classe `container`. Cela donnera à l'enfant direct la couleur de texte `blue`.

Parce que les deux autres éléments paragraphe sont imbriqués dans des éléments `div`, ils ne sont pas considérés comme enfants directs de la classe `container` et ne recevront pas la couleur de texte bleue.

Un autre combinateur courant serait le combinateur de frère adjacent.

Le combinateur de frère adjacent (`+`) en CSS sélectionne un élément qui suit immédiatement un élément frère spécifié. Ce combinateur est utile quand tu veux appliquer des styles à un élément qui suit directement un autre élément, permettant une mise en forme ciblée basée sur la position de l'élément par rapport à ses frères.

Regardons l'exemple HTML suivant :

:::interactive_editor

```html
<figure>
  <img
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
    alt="A cute orange cat lying on its back."
  />
  <figcaption>A cute orange cat lying on its back.</figcaption>
</figure>
```

:::

Ici, on a un élément `figure` contenant un élément `img` suivi d'un élément `figcaption`. L'élément `figcaption` est le frère immédiat de l'élément `img`.

Si tu voulais appliquer une bordure noire autour de l'élément `figcaption`, tu peux utiliser le combinateur de frère adjacent comme ceci :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<figure>
  <img
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
    alt="A cute orange cat lying on its back."
  />
  <figcaption>A cute orange cat lying on its back.</figcaption>
</figure>
```

```css
img + figcaption {
  border: 4px solid black;
}
```

:::

Dans cet exemple, le combinateur de frère adjacent (`+`) sélectionne l'élément `figcaption` qui suit immédiatement l'élément `img`. La règle CSS appliquée ajoute une `bordure noire continue de 4px` autour du `figcaption`.

Un autre combinateur courant est le combinateur de frères suivants.

Le combinateur de frères suivants (`~`) en CSS sélectionne tous les frères d'un élément spécifié qui viennent après lui.

Contrairement au combinateur de frère adjacent, qui cible uniquement le frère immédiatement suivant, le combinateur de frères suivants (`~`) peut cibler n'importe quel frère qui suit l'élément spécifié, offrant une plus grande flexibilité dans la mise en forme.

Regardons l'exemple HTML suivant :

:::interactive_editor

```html
<div class="container">
  <h2>Subheading</h2>
  <p>First paragraph.</p>
  <p>Second paragraph.</p>
  <p>Third paragraph.</p>
  <p>Another paragraph element</p>
</div>
```

:::

Dans cette structure HTML, on a un élément `h2` suivi de quatre éléments paragraphe. Les éléments paragraphe sont des frères de l'élément `h2`.

Si tu veux styliser tous les éléments paragraphe qui viennent après l'élément `h2`, alors tu peux utiliser le combinateur de frères suivants comme ceci :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <h2>Subheading</h2>
  <p>First paragraph.</p>
  <p>Second paragraph.</p>
  <p>Third paragraph.</p>
  <p>Another paragraph element</p>
</div>
```

```css
h2 ~ p {
  color: green;
}
```

:::

Dans cet exemple, tous les éléments paragraphe suivant l'élément `h2` auront la couleur de texte verte.

Le combinateur de frères suivants (`~`) cible tous les frères paragraphe qui apparaissent après l'élément `h2`, qu'ils soient ou non des frères immédiats.

En conclusion, comprendre et utiliser divers combinateurs CSS te permet d'appliquer des styles précis à tes éléments HTML, améliorant le contrôle et la flexibilité de ton design.

En maîtrisant ces sélecteurs, tu peux créer des règles de mise en forme plus sophistiquées et ciblées qui améliorent à la fois l'apparence et la fonctionnalité de tes pages web.
