# --interactive--

Margin et padding sont des propriétés essentielles en CSS pour créer des pages web bien structurées, lisibles et visuellement attrayantes.

Les margins contrôlent l'espace à l'extérieur d'un élément, aidant à le séparer d'autres éléments et à définir la structure de la mise en page, tandis que le padding contrôle l'espace à l'intérieur d'un élément, améliorant la lisibilité du contenu et l'attrait esthétique.

Pour mieux comprendre les différences entre `margin` et `padding`, regardons quelques exemples.

Voici un exemple HTML de trois éléments paragraphe sur la page :

:::interactive_editor

```html
<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

:::

Pour appliquer un espacement au sommet de chaque élément paragraphe, tu peux utiliser la propriété `margin-top` comme ceci :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  margin-top: 20px;
  border: 2px solid black;
}
```

:::

Dans cet exemple, on applique `20px` de margin uniquement au sommet de chaque élément paragraphe.

On a aussi une bordure noire continue sur les quatre côtés pour que tu puisses mieux voir la margin.

Les quatre différentes propriétés `margin` sont `margin-top`, `margin-right`, `margin-bottom` et `margin-left`.

Voici un exemple utilisant les quatre propriétés :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
  border: 2px solid black;
}
```

:::

Dans cet exemple, on assigne des valeurs d'espacement sur les quatre côtés de l'élément paragraphe. Une bordure noire continue a aussi été ajoutée pour que tu puisses mieux voir les margins.

Une autre façon d'utiliser la propriété `margin` est avec la notation raccourcie. Tu peux spécifier une, deux, trois ou quatre valeurs en même temps. Explorons ces options ensemble.

Quand on utilise une valeur unique sur le raccourci `margin`, cette valeur exacte sera appliquée aux quatre côtés de l'élément cible.

Voici un exemple d'utilisation d'une valeur unique sur le raccourci `margin` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px;
}
```

:::

Cet exemple de code appliquera `10px` de `margin` également aux quatre côtés de l'élément paragraphe.

Quand on utilise deux valeurs, la première valeur s'applique aux côtés `top` et `bottom`, tandis que la seconde valeur s'applique aux côtés `left` et `right` de l'élément.

Voici un exemple d'utilisation de deux valeurs pour le raccourci `margin` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px;
}
```

:::

Ça définit les margins `top` et `bottom` à `10px`, et `20px` pour les margins `left` et `right` pour l'élément paragraphe.

Si trois valeurs sont fournies, la première valeur s'applique à la margin `top`, la deuxième valeur aux margins `left` et `right`, et la troisième valeur à la margin `bottom`.

Voici un exemple pour mieux comprendre :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px;
}
```

:::

Ça définit la margin à `10px` pour le `top`, `20px` pour les `left` et `right`, et `30px` pour le `bottom`.

Quand on utilise quatre valeurs, ça te donne plus de contrôle, car tu peux spécifier indépendamment chaque valeur de margin pour chaque côté de l'élément cible.

La première valeur cible le `top`, la deuxième valeur cible le `right`, la troisième valeur cible le `bottom`, et la quatrième valeur cible le `left`.

Voici un exemple d'utilisation du raccourci margin avec quatre valeurs :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px 40px;
}
```

:::

Ça définit la margin à `10px` pour le `top`, `20px` pour le `right`, `30px` pour le `bottom`, et `40px` pour le `left`.

La propriété `padding` est utilisée pour appliquer de l'espace à l'intérieur de l'élément, entre le contenu et sa bordure.

Comme la propriété `margin`, les quatre propriétés `padding` sont `padding-top`, `padding-right`, `padding-bottom` et `padding-left`.

Voici un exemple sur comment définir le `padding` pour un élément paragraphe :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
  border: 2px solid black;
}
```

:::

Ça définit le padding à `10px` pour le `top`, `20px` pour le `right`, `30px` pour le `bottom`, et `40px` pour le `left`.

Comme tu peux le voir, `padding` est appliqué au contenu qui se trouve à l'intérieur de la bordure, contrairement à `margin` qui s'applique à l'extérieur de la bordure.

Tout comme la propriété `margin`, tu peux aussi choisir d'utiliser le raccourci pour la propriété `padding`.

Tu peux également spécifier une, deux, trois ou quatre valeurs sur la propriété raccourcie `padding`.

Voici un exemple d'utilisation du raccourci `padding` pour l'élément paragraphe d'avant :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding: 10px 20px 30px 40px;
  border: 2px solid black;
}
```

:::

Dans l'exemple, en utilisant le raccourci, le code définira le padding à `10px` pour le `top`, `20px` pour le `right`, `30px` pour le `bottom`, et `40px` pour le `left` de l'élément paragraphe.
