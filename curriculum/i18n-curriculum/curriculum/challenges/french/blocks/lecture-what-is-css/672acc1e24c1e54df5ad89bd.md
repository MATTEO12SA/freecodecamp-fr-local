# --interactive--

En HTML et CSS, les éléments sont classés comme des **éléments inline** ou des **éléments block-level**, et cette classification dicte leur comportement dans la mise en page du document.

Comprendre cette différence est essentiel pour contrôler comment ton contenu est affiché sur une page web.

Les éléments block-level sont des éléments qui prennent par défaut toute la largeur disponible, s'étendant sur toute la largeur de leur conteneur.

Ces éléments commencent toujours sur une nouvelle ligne et poussent les autres contenus à la ligne suivante, créant un « bloc » de contenu.

Les éléments block-level ont la propriété CSS `display: block;` appliquée par défaut. Cette propriété garantit que l'élément s'étend pour remplir la largeur du conteneur et apparaisse sur une nouvelle ligne.

Quelques éléments block-level courants sont les éléments `div`, les paragraphes, les titres, les listes ordonnées, les listes non ordonnées et les éléments section.

Voici un exemple de code d'un élément block-level :

:::interactive_editor

```html
<p style="border: 2px solid red;">
  First paragraph
</p>
<p>Second paragraph</p>
```

:::

Dans cet exemple, on a deux éléments paragraphe où le premier a une bordure rouge autour de lui.

Les deux éléments paragraphe ne partagent pas la même ligne parce qu'ils sont des éléments block-level par défaut.

Donc, les deux éléments paragraphe prendront toute la largeur de leur conteneur, qui dans ce cas est l'élément `body`.

Les éléments block-level sont idéaux quand tu veux que le contenu s'empile verticalement, comme des paragraphes, des sections ou de plus gros blocs de contenu. Ils sont couramment utilisés pour définir la mise en page et la structure d'une page web.

Les éléments inline, contrairement aux éléments block-level, ne prennent que la largeur dont ils ont besoin et ne commencent pas sur une nouvelle ligne. Ces éléments s'écoulent dans le contenu, permettant au texte et à d'autres éléments inline d'apparaître à côté d'eux.

Les éléments inline ont la propriété CSS `display: inline;` appliquée par défaut. Cette propriété garantit que l'élément reste dans le flux du contenu et ne saute pas à une nouvelle ligne.

Les éléments inline courants sont les éléments `span`, `anchor` et `img`.

Voici un exemple pour mieux comprendre les éléments inline :

:::interactive_editor

```html
<p>This is a
  <span style="color: red; border: 2px solid green;">red</span>
  word inside a paragraph.
</p>
```

:::

Dans cet exemple, on a un élément `span` imbriqué à l'intérieur d'un élément paragraphe. L'élément `span` a une couleur de texte `red` avec une bordure `green` autour de lui pour que tu puisses mieux voir le mot mis en évidence.

Comme tu peux le voir, l'élément `span` ne prend que l'espace nécessaire pour le mot « red » et ne pousse pas le contenu suivant sur une nouvelle ligne.

Les éléments inline sont les mieux utilisés pour styliser de plus petites portions de texte ou de contenu dans une ligne, comme mettre l'accent sur un mot, créer des hyperliens ou appliquer des styles spécifiques à des parties d'un paragraphe.
