## Comprendre les bases de l'ordinateur, d'internet et des outils de développement

- **Carte mère** : contient toute la mémoire, les connecteurs et les disques durs nécessaires au fonctionnement de l'ordinateur. Elle sert de circuit imprimé principal de l'ordinateur.
- **Unité centrale de traitement (CPU)** : un processeur chargé d'exécuter les instructions et d'effectuer les calculs.
- **Mémoire vive (RAM)** : un emplacement de stockage temporaire pour le CPU de l'ordinateur.
- **Disque dur (HDD)** : un emplacement de stockage permanent utilisé pour stocker les données même quand l'ordinateur est éteint.
- **Disque SSD** : mémoire flash non volatile qui peut être utilisée à la place d'un disque dur.
- **Bloc d'alimentation (PSU)** : chargé de convertir l'électricité de la prise murale en une forme utilisable par l'ordinateur.
- **Unité de traitement graphique (GPU)** : chargée d'afficher les visuels sur l'écran de l'ordinateur.
- **Différents types de fournisseurs d'accès internet** : Un fournisseur d'accès internet (FAI) est une entreprise qui fournit l'accès à internet. Il existe différents types de FAI, dont le dial-up, le DSL, le câble, la fibre optique et le satellite.
- **Façons sûres de se connecter à ton ordinateur** : Des exemples de façons sûres de se connecter à ton ordinateur incluent l'utilisation d'un mot de passe fort, l'activation de l'authentification à deux facteurs et l'utilisation d'un gestionnaire de mots de passe.
- **Environnement de développement intégré (IDE)** : un outil qui aide les développeurs à écrire, tester et déboguer du code de manière efficace.
- **Éditeur de code** : un outil que les développeurs utilisent pour écrire et déboguer du code.
- **Git** : un système de contrôle de version populaire qui permet aux développeurs de suivre les changements dans leur code et de collaborer avec d'autres.
- **Services d'hébergement cloud pour les dépôts** : Un dépôt est un emplacement de stockage pour les fichiers de projet et l'historique des versions. Les services d'hébergement cloud populaires pour les dépôts incluent GitHub, GitLab et Bitbucket.
- **Gestionnaires de paquets** : outils qui aident les développeurs à simplifier le processus d'ajout, de mise à jour et de suppression des bibliothèques et des dépendances du projet. Des exemples incluent npm, pip et Maven.
- **Bibliothèques et frameworks de test** : Le test est effectué dans les logiciels pour s'assurer que le code fonctionne comme prévu. Des exemples de bibliothèques et frameworks de test incluent Jest, PHPUnit et JUnit.

## Travailler avec les fichiers, les systèmes de fichiers et les formats de média

- **Bonnes pratiques pour nommer les fichiers** : Tu voudras nommer tes fichiers d'une manière facile à comprendre et à maintenir. Par exemple, `about-us.html` est un nom plus descriptif que `page1.html`.
- **répertoire racine** : répertoire de plus haut niveau dans un système de fichiers. Répertoire est un autre nom pour dossier.
- **Markdown** : un langage de balisage couramment utilisé pour la documentation et les fichiers `README`. Un fichier `README` est un fichier qui contient des informations sur un projet, comme comment l'installer et l'utiliser.
- **`index.html`** : représente la page par défaut affichée quand un utilisateur visite un site web.
- **Créer, déplacer et supprimer des fichiers et dossiers avec l'Explorateur/Finder** : L'Explorateur est le gestionnaire de fichiers de Windows, et le Finder est le gestionnaire de fichiers de macOS. Tu peux utiliser ces outils pour créer, déplacer et supprimer des fichiers et dossiers.
- **Rechercher des fichiers et dossiers** : Tu peux utiliser la fonctionnalité de recherche de l'Explorateur ou du Finder pour trouver des fichiers et dossiers sur ton ordinateur.
- **Types de fichiers HTML, CSS et JS** : L'extension de fichier `.html` est utilisée pour les fichiers HTML, `.css` pour les fichiers CSS et `.js` pour les fichiers JavaScript.
- **Formats d'image et graphiques courants** : `JPEG` et `PNG` sont des formats de fichiers d'image courants. `GIF` est un autre format de fichier d'image courant qui prend en charge l'animation. `SVG` est un format de fichier pour les graphiques vectoriels.
- **Formats audio et vidéo courants** : Le format `MP3` est couramment utilisé pour les fichiers audio. Le format `MP4` est couramment utilisé pour les fichiers vidéo. Le format `MOV` a été développé par Apple et est couramment utilisé pour les fichiers vidéo.
- **Formats de police courants** : Le format `TTF` est couramment utilisé pour les polices TrueType. Le format `WOFF` est couramment utilisé pour les polices web. Le successeur de WOFF est `WOFF2`, qui offre une meilleure compression.
- **ZIP** : un format de fichier utilisé pour compresser des fichiers et dossiers.

## Naviguer sur le web efficacement

- **Qu'est-ce qu'un navigateur web ?** : une application logicielle qui permet aux utilisateurs d'accéder et de consulter des sites web sur internet.
- **Qu'est-ce qu'un moteur de recherche** : un outil qui permet aux utilisateurs de rechercher des informations sur internet. Des exemples incluent Google, Bing et Yahoo.
- **Navigateurs web courants** : Quelques exemples de navigateurs web courants incluent Google Chrome, Mozilla Firefox et Microsoft Edge.
- **Stratégies de recherche courantes** : Tu peux utiliser `site:` suivi de l'URL d'un site web pour rechercher du contenu sur ce site web. Tu peux utiliser `filetype:` suivi d'une extension de fichier pour rechercher des fichiers de ce type. Tu peux préfixer un terme de recherche avec un signe moins pour exclure les résultats contenant ce terme. Tu peux préfixer un terme de recherche avec un signe plus pour inclure les résultats contenant ce terme.
