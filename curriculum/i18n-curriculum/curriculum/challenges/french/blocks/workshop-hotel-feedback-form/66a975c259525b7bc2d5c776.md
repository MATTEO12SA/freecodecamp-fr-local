# --description--

Ton formulaire de retour pour hôtel doit aussi permettre aux utilisateurs de noter la nourriture.

Commence par ajouter un élément `label` avec le texte `How was the food?`. Cet élément `label` doit avoir un attribut `for` défini sur `"food"`.

Sous ton élément `label`, ajoute un élément `select` avec un `id` et un `name` définis sur `"food"`.

# --hints--

Tu dois avoir un élément `label` avec l'attribut `for` défini sur `"food"`.

```js
assert.exists(document.querySelector('label[for="food"]'));
```

Ton `label` doit avoir le texte `How was the food?`.

```js
assert.strictEqual(document.querySelector('label[for="food"]')?.textContent.trim(), 'How was the food?');
```

Tu dois avoir un élément `select` avec l'`id` défini sur `"food"` sous ton `label`.

```js
assert.exists(document.querySelector('label + select#food'));
```

Ton élément `select` doit avoir l'attribut `name` défini sur `"food"`.

```js
assert.exists(document.querySelector('label + select[name="food"]'));
```
