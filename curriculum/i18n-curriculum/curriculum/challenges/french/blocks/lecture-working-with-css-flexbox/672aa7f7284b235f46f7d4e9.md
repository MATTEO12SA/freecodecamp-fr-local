# --interactive--

CSS Flexbox est un modèle de mise en page unidimensionnel qui te permet d'organiser des éléments en lignes et en colonnes dans un conteneur. Tu peux aussi contrôler leur ordre et leur orientation. Les développeurs web l'utilisent pour créer des sites web et des applications web responsives qui s'adaptent à différentes tailles et orientations d'écran. On parle de Flexbox comme d'un modèle de mise en page unidimensionnel, car il se concentre sur l'organisation des éléments le long d'un seul axe à la fois. Cet axe peut être horizontal ou vertical.

Il y a deux concepts clés que tu devrais connaître avant de commencer à travailler avec Flexbox : le conteneur flex et l'élément flex.

Un conteneur flex est un élément HTML avec une mise en page flex. Tu peux organiser et aligner des éléments de différentes façons dans un conteneur flex. Pour transformer un élément HTML en conteneur flex, tu dois ajouter `display: flex` à ses styles CSS.

Les éléments flex sont les enfants directs d'un conteneur flex. Ces éléments peuvent être organisés et alignés dans un conteneur flex selon ses propriétés. Ils peuvent aussi rétrécir ou s'agrandir pour occuper l'espace disponible.

Voici un exemple avec un conteneur `main` et trois éléments enfants `div` :

```html
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

Si tu définis seulement la `width`, la `height` et la `background-color` de ces éléments `div` dans la feuille de style CSS, chaque élément enfant sera placé sur sa propre ligne, car le conteneur n'est pas flex par défaut.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Mais si tu ajoutes `display: flex` au conteneur `main`, les éléments `div` seront réorganisés pour tenir sur la même ligne et ils rétréciront si nécessaire (pour voir les aperçus, active l'éditeur interactif) :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Par défaut, un conteneur flex sera un élément de niveau bloc, donc le conteneur lui-même sera sur sa propre ligne par rapport aux autres éléments et conteneurs.

Maintenant que tu en sais plus sur les conteneurs flex et les éléments flex, tu devrais aussi connaître les propriétés flex. Ces propriétés déterminent comment les éléments flex seront organisés, redimensionnés et distribués dans le conteneur flex. Parmi les plus couramment utilisées, on trouve `flex-direction`, `justify-content`, `align-items` et `flex-wrap`.

Très bien. Parlons maintenant un peu du modèle flex. Ce modèle définit comment les éléments flex sont organisés dans un conteneur flex. Chaque conteneur flex a deux axes :

- L'axe principal.
- L'axe transversal.

L'orientation de ces axes détermine comment différentes propriétés affecteront la mise en page et la distribution des éléments flex. Par défaut, l'axe principal d'un conteneur flex est horizontal et l'axe transversal est vertical. Les éléments flex sont organisés dans la direction de l'axe principal. L'axe transversal est perpendiculaire à l'axe principal.

La propriété `flex-direction` définit la direction de l'axe principal. La valeur par défaut de `flex-direction` est `row`, ce qui place tous les éléments flex sur la même ligne, dans la direction de la langue par défaut de ton navigateur (de gauche à droite ou de droite à gauche) :

```css
flex-direction: row; /* Default */
```

Pour inverser les éléments dans la ligne, tu peux utiliser `flex-direction: row-reverse` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: row-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Cela inversera l'ordre des éléments flex. Si tu veux plutôt aligner les éléments flex verticalement, tu dois simplement définir `flex-direction` à `column` sur le conteneur flex. Cela changera la direction de l'axe principal :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Les éléments `div` seront maintenant alignés verticalement, car l'axe principal sera vertical et l'axe transversal sera horizontal.

Tu peux aussi inverser verticalement l'ordre des éléments flex avec `flex-direction: column-reverse` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

CSS Flexbox est un modèle de mise en page puissant qui fournit une manière flexible et efficace d'organiser des éléments dans un conteneur. En comprenant les concepts clés des conteneurs flex, des éléments flex et des différentes propriétés flex, tu peux créer des sites web dynamiques et responsives qui s'adaptent à différentes tailles et orientations d'écran.
