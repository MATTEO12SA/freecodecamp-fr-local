# --interactive--

Les propriétés flex sont des propriétés que tu peux appliquer aux conteneurs flex pour déterminer la distribution des éléments enfants. Nous allons couvrir certaines des plus couramment utilisées : `flex-wrap`, `justify-content` et `align-items`.

Commençons par `flex-wrap`. Cette propriété détermine comment les éléments flex passent à la ligne dans un conteneur flex pour tenir dans l'espace disponible. `flex-wrap` peut prendre trois valeurs possibles : `nowrap`, `wrap` et `wrap-reverse`. `nowrap` est la valeur par défaut : les éléments flex ne passeront pas sur une nouvelle ligne, même si leur largeur dépasse celle du conteneur.

Dans le code ci-dessous, nous avons trois éléments `div`. Concentrons-nous sur la `width`. Le conteneur `main` entouré d'une bordure rouge a une `width` de `200px`, tandis que ses trois éléments enfants `div` ont ensemble une `width` de `240px` (`80px` chacun) :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

La largeur des éléments `div` dépasse celle de leur conteneur, mais par défaut ils seront rétrécis pour tenir dans l'espace disponible. Si tu veux qu'ils passent à la ligne quand ils dépassent la largeur de leur conteneur, tu peux définir `flex-wrap: wrap` sur le conteneur flex :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-wrap: wrap;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Les éléments `div` seront réorganisés en lignes quand ils dépassent la largeur de leur conteneur. Tu peux faire passer les éléments flex à la ligne dans l'ordre inverse avec `flex-wrap: wrap-reverse`. La propriété `flex-flow` est une propriété raccourcie pour `flex-direction` et `flex-wrap`. Dans cet exemple, nous définissons `flex-direction` à `column` et `flex-wrap` à `wrap-reverse` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-flow: column wrap-reverse;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Très bien. Parlons maintenant de `justify-content`. `justify-content` aligne les éléments enfants le long de l'axe principal du conteneur flex. Si tu attribues la valeur `flex-start` à `justify-content`, les éléments flex seront alignés au début de l'axe principal. Cela peut être horizontal ou vertical :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: flex-start;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Avec `justify-content: flex-end`, les éléments flex sont alignés à la fin de l'axe principal, horizontalement ou verticalement.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: flex-end;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Pour centrer les éléments flex le long de l'axe principal, tu peux utiliser `justify-content: center`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: center;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Pour distribuer les éléments uniformément le long de l'axe principal, tu peux utiliser `justify-content: space-between`. Cela ajoutera de l'espace entre les éléments flex si nécessaire.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-between;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

`justify-content: space-around` distribue les éléments flex uniformément sur l'axe principal, en ajoutant un espace avant le premier élément et après le dernier élément. Cet espace supplémentaire correspond à la moitié de l'espace entre chaque paire d'éléments adjacents. S'il n'y a qu'un seul élément à distribuer, il sera centré.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-around;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Et enfin, nous avons `justify-content: space-evenly`, qui distribue les éléments uniformément le long de l'axe principal. L'espace entre les éléments et l'espace avant et après les premier et dernier éléments sont exactement identiques :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: space-evenly;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

Très bien. Tu sais maintenant comment distribuer les éléments flex le long de l'axe principal. Mais tu peux aussi vouloir les distribuer le long de l'axe transversal. Rappelle-toi que l'axe transversal est perpendiculaire à l'axe principal. Tu peux le faire avec la propriété `align-items`. Pour centrer les éléments le long de l'axe transversal, il suffit d'ajouter `align-items: center` au conteneur flex :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: center;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Dans cet exemple, les éléments flex sont centrés le long de l'axe transversal, qui est vertical par défaut. Si l'axe transversal est horizontal, ils seront plutôt centrés horizontalement. À l'inverse, `align-items: flex-start` aligne les éléments au début de l'axe transversal :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: flex-start;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

L'opposé est `align-items: flex-end`. Cela alignera les éléments flex à la fin de l'axe transversal, verticalement ou horizontalement.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: flex-end;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Pour étirer les éléments flex le long de l'axe transversal, tu peux utiliser `align-items: stretch`. Cela fonctionne aussi avec les éléments dont la taille est automatique, comme ceux sans `width` ou `height` définie, ou avec seulement une `width` ou une `height` minimale. Les éléments flex s'étireront pour remplir le conteneur dans la direction de l'axe transversal.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: stretch; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;           
  border: 1px solid #888;
  background-color: lightblue;
}
```

:::

Enfin, tu peux utiliser la propriété `align-self` pour attribuer un alignement différent sur l'axe transversal à un élément flex individuel. Par exemple, tu peux l'étirer avec `align-self: stretch`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box special">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: flex-start; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;
  border: 1px solid #888;
  background-color: lightblue;
  margin: 4px;
}

.special {
  align-self: stretch; 
  background-color: lightcoral;
}
```

:::

Tu peux le centrer avec `align-self: center`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box special">Three</div>
</div>

```

```css

  border: 1px solid #888;
  background-color: lightblue;
  margin: 4px;
}

.special {
  align-self: center; 
  background-color: lightcoral;
}
```

:::

Tu peux l'aligner au début de l'axe transversal avec `align-self: flex-start`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box special">Three</div>
</div>

```

```css

  align-items: flex-start; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;
  border: 1px solid #888;
  background-color: lightblue;
  margin: 4px;
}

.special {
  align-self: flex-start; 
  background-color: lightcoral;
}
```

:::

Ou tu peux l'aligner à la fin de l'axe transversal avec `align-self: flex-end`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box special">Three</div>
```

```css
  background-color: lightcoral;
```

:::

Il existe d'autres propriétés et valeurs flex que tu peux choisir pour créer la mise en page responsive que tu imagines, mais celles-ci sont les plus couramment utilisées. Avec ces propriétés flex CSS et tes nouvelles connaissances du modèle flex CSS, tu peux commencer à créer des mises en page responsives pour offrir une expérience utilisateur fluide et inclusive sur tous les appareils.
