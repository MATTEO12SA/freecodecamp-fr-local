# --interactive--

L'un des aspects les plus intéressants du CSS est l'utilisation des pseudo-éléments. Dans ce contexte, « pseudo » signifie « pas réel », donc les pseudo-éléments sont des éléments virtuels ou synthétiques qui ne correspondent pas directement à de vrais éléments HTML. Ils te permettent de styliser des parties spécifiques d'un élément ou d'insérer du contenu sans ajouter de HTML supplémentaire.

Pour appliquer un pseudo-élément, attache-le au sélecteur de l'élément original en utilisant un double deux-points (`::`). Note que le sélecteur peut être de n'importe quel type, comme un sélecteur de classe ou d'ID. Voici à quoi ressemble la syntaxe de base des pseudo-éléments :

```css
selector::pseudo-element {
  property: value;
}
```

Ce double deux-points est ce qui distingue les pseudo-éléments des pseudo-classes, qui utilisent un seul deux-points.

Les pseudo-éléments te permettent de styliser des parties spécifiques du contenu d'un élément ou d'insérer du contenu avant ou après lui, mais ils ne peuvent pas exister indépendamment. L'élément auquel un pseudo-élément est attaché est appelé son élément d'origine.

Commençons par regarder des exemples pour les pseudo-éléments `::before` et `::after`. Comme leurs noms le suggèrent, `::before` te permet d'insérer du contenu juste avant le contenu de l'élément tandis que `::after` te permet d'insérer du contenu après.

Voici un exemple d'un élément button. Dans le CSS, on va utiliser le positionnement absolu et le pseudo-élément `::before` pour ajouter une étoile avant le texte `Learn More` du bouton. Tu en apprendras plus sur le positionnement absolu dans les futures leçons.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: lightseagreen;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
  position: relative;
}

.cta-button::before {
  content: "⭐";
  position: absolute;
  left: 3px;
  top: 8px;
  font-size: 0.75rem;
}
```

:::

La propriété `content` est utilisée pour représenter le contenu que tu souhaites ajouter avant le texte du bouton. Dans cet exemple, on ajoute une étoile. Tu remarqueras que tu peux non seulement insérer du contenu mais aussi le styliser. Voici un exemple du pseudo-élément `::after` avec le même bouton :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: '➡️';
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}
```

:::

Dans le pseudo-élément `::after`, la propriété `transition` est utilisée pour animer les changements sur 0,3 seconde avec un effet d'assouplissement, créant une transformation douce et graduelle plutôt que soudaine. Tu en apprendras plus sur la propriété `transition` dans les futures leçons.

Tu peux aussi attacher une pseudo-classe au contenu que tu insères dans un autre contenu avec les pseudo-éléments `::before` et `::after`. Par exemple, un état hover pour le contenu. Voici un exemple :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: "➡️";
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}

.cta-button:hover::after {
  transform: translateX(2px);
}
```

:::

Avec `transform: translateX(2px)` dans l'état hover, le contenu est poussé vers la droite de `2px` à chaque fois que l'utilisateur survole le bouton. La propriété transition dans le `::after` lui-même garantit que le processus prend `0.3s`.

C'est ce que fait la propriété `transform` – elle te permet de faire pivoter, fausser, mettre à l'échelle ou translater un élément dans une direction particulière. Tu en apprendras plus sur ça dans les futures leçons.

Dans le prochain exemple, on va regarder le pseudo-élément `::first-letter`. Le pseudo-élément `::first-letter` cible la première lettre du contenu d'un élément, te permettant de la styliser. Voici un exemple d'un texte de paragraphe. Si on veut styliser la première lettre, on peut utiliser le pseudo-élément `::first-letter` comme ceci :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>freeCodeCamp lets you learn to code without having to pay.</p>
```

```css
p::first-letter {
  font-size: 4rem;
}
```

:::

Dans le dernier exemple, on va regarder le pseudo-élément `::marker`, qui te permet de sélectionner le marqueur, la puce ou la numérotation des éléments de liste pour les styliser. Le pseudo-élément `::marker` offre une façon d'améliorer l'identité de marque de ton site web en personnalisant les marqueurs de liste pour correspondre à ta palette de couleurs.

Voici un exemple d'une liste non ordonnée et d'une liste ordonnée. Pour changer la couleur et la taille du marqueur d'élément de liste, tu peux utiliser le pseudo-élément `::marker` comme ceci :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
  <li>Unordered list item 3</li>
  <li>Unordered list item 4</li>
</ul>

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
  <li>Ordered list item 3</li>
  <li>Ordered list item 4</li>
</ol>
```

```css
li::marker {
  color: crimson;
  font-size: 1.5em;
  font-weight: bold;
}
```

:::

Dans cette leçon, on n'a couvert que quelques pseudo-éléments. Mais il y en a beaucoup d'autres comme `::placeholder`, `::spelling-error` et `::selection` que je t'encourage à explorer par toi-même.
