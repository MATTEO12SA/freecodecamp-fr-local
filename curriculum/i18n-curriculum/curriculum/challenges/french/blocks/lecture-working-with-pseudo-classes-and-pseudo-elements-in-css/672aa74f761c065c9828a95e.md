# --interactive--

Les pseudo-classes sont des mots-clés CSS spéciaux qui te permettent de sélectionner un élément en fonction de son état ou de sa position spécifique. L'état ou la position de l'élément peut inclure :

- Quand il est actif.
- Quand il est survolé par une souris.
- Quand il est le premier enfant d'un parent.
- Quand il est le dernier enfant d'un parent.
- Quand un lien a été visité.
- Quand il est désactivé.

Pour utiliser une pseudo-classe, tu l'ajoutes au sélecteur en utilisant deux points (`:`) suivis du nom de la pseudo-classe :

```css
selector:pseudo-class {
  /* CSS properties */
}
```

Voyons comment tu peux utiliser une pseudo-classe pour représenter chacun des états et positions déjà mentionnés.

La pseudo-classe `:active` te permet de sélectionner l'état actif d'un élément, comme cliquer sur un bouton :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example Button</button>
```

```css
button:active {
  background: greenyellow;
}
```

:::

La pseudo-classe `:hover` définit l'état hover d'un élément. Un exemple serait de survoler un bouton ou un lien :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="#">Hover over me!</a>  
```

```css
a:hover {
  text-decoration: none;
  color: white;
  background: crimson;
}
```

:::

La pseudo-classe `:first-child` sélectionne un élément qui est le premier enfant de son élément parent. Voici un exemple ciblant le premier élément paragraphe dans un conteneur `div` :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:first-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

Le premier élément paragraphe dans ce `div` recevra la couleur d'arrière-plan `lightcoral` et un `padding` de `0.4rem` sur les quatre côtés.

La pseudo-classe `:last-child` cible le dernier élément qui est l'enfant de son parent. Voici un exemple ciblant le dernier élément paragraphe dans l'élément `div` container :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:last-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

La pseudo-classe `:visited` te permet de styliser un lien que l'utilisateur a déjà visité :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

La pseudo-classe `:disabled` te permet de styliser un élément interactif en mode désactivé :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button disabled>Disabled Button</button>
```

```css
button:disabled {
  background-color: lightgray;  
}
```

:::

Comme tu peux le voir, les pseudo-classes te permettent de styliser les éléments en fonction des interactions de l'utilisateur, comme survoler ou visiter un lien. Cela rend ton site web plus interactif.

En plus des pseudo-classes déjà mentionnées, il y en a d'autres comme :

- `:focus`
- `:first-of-type`
- `:last-of-type`
- `:nth-of-type`
- `:modal`
- `:enabled`
- `:checked`
- `:required`, et plus.
