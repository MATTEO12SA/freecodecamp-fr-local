# --interactive--

Quand tu travailles avec JavaScript, tu peux rencontrer des situations où tu as besoin de répéter une chaîne un nombre précis de fois.

Que tu génères des motifs répétés ou que tu dupliques simplement du texte, la méthode `repeat()` offre un moyen simple et efficace d'y parvenir.

La méthode `repeat()` est une fonction intégrée de JavaScript qui te permet de répéter une chaîne un nombre de fois précis. Voici la syntaxe de base :

```js
string.repeat(count);
```

`string` est la chaîne que tu veux répéter, et `count` est le nombre de fois où tu veux que la chaîne soit répétée. Voici un exemple :

:::interactive_editor

```js
let word = "Hello!";
let repeatedWord = word.repeat(3);
console.log(repeatedWord);  // "Hello!Hello!Hello!"
```

:::

Dans ce cas, la chaîne `Hello!` est répétée trois fois, ce qui donne `Hello!Hello!Hello!`.

Même si la méthode `repeat()` est utile, il y a quelques exceptions et limitations à garder à l'esprit.

Le paramètre `count` doit être un nombre non négatif. Si tu passes un nombre négatif, JavaScript générera une `RangeError`.

```js
let word = "Test";
console.log(word.repeat(-1));  // Throws RangeError: Invalid count value
```

Le `count` doit être un nombre fini. Si tu essaies de répéter une chaîne un nombre infini de fois ou que tu utilises `Infinity` comme valeur, tu obtiendras également une `RangeError`.

En JavaScript, `Infinity` est une valeur spéciale qui représente une quantité infinie. Elle sert à désigner des nombres plus grands que n'importe quel nombre fini.

```js
let word = "Test";
console.log(word.repeat(Infinity));  // Throws RangeError: Invalid count value
```

Si le nombre n'est pas un entier (comme un décimal tel que `2.5`), la méthode `repeat()` l'arrondira à l'entier inférieur.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(2.5));  // "TestTest"
```

:::

Si tu passes `0` comme valeur, la méthode `repeat()` renverra une chaîne vide.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(0));  // ""
```

:::

La méthode `repeat()` peut simplifier les tâches qui impliquent la duplication de chaînes, rendant ton code plus concis et plus lisible.

Que tu génères des motifs de texte répétés ou que tu remplisses un espace avec des caractères, `repeat()` peut t'éviter d'écrire des boucles ou du code plus complexe.

Tu n'es pas obligé de passer un nombre directement dans la méthode `repeat()`. Tu peux aussi passer une variable qui stocke une valeur numérique.

:::interactive_editor

```js
let count = 4;
let word = "Test";
let repeatedWord = word.repeat(count);
console.log(repeatedWord); // TestTestTestTest
```

:::

Dans cet exemple, la variable `count` stocke le nombre de répétitions. Cela peut être utile quand le nombre de répétitions dépend de la saisie de l'utilisateur ou d'autres valeurs dynamiques de ton programme.
