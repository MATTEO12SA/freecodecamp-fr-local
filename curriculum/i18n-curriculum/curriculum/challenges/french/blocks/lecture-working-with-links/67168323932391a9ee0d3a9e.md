# --interactive--

Tu as peut-être déjà vu un lien sur une page web devenir violet après avoir cliqué dessus. C'est parce que l'état du lien a changé, donc un style différent est appliqué. Un lien peut avoir cinq états différents.

Le premier est l'état par défaut, `:link`. Cet état représente un lien que l'utilisateur n'a pas encore visité, cliqué ou utilisé. Tu peux voir cet état comme la base des styles pour tous les liens de ta page. Les autres états s'ajoutent par-dessus.

Active l'éditeur interactif et regarde le lien dans la fenêtre d'aperçu. Il devrait avoir la couleur bleue par défaut, qui représente l'état par défaut. Si tu cliques dessus, il deviendra violet.

:::interactive_editor

```html
<a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
```

:::

Le deuxième état est `:visited`, qui s'applique quand un utilisateur a déjà visité la page liée. Par défaut, cela rend le lien violet, mais tu peux utiliser CSS pour donner une indication visuelle différente à l'utilisateur. C'est utile pour indiquer à quelqu'un qu'il a déjà lu une partie de ta documentation, ou que le site est fiable parce qu'il l'a déjà utilisé.

Clique sur le lien dans la fenêtre d'aperçu et tu devrais voir que le lien visité devient marron. Pour voir les aperçus, tu dois activer l'éditeur interactif.

**NOTE** : du CSS a été fourni pour cet exemple interactif afin que tu puisses voir les changements de style des liens. Ne t'inquiète pas si tu ne comprends pas encore le code CSS, tu apprendras ce que tout cela signifie dans de futurs modules.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:visited {
  color: brown;
}
```

:::

Le troisième état est `:hover`. Cet état s'applique quand un utilisateur place son curseur sur un lien. Il est utile pour attirer davantage l'attention sur un lien et s'assurer que l'utilisateur a vraiment l'intention de cliquer dessus.

Active l'éditeur interactif et essaie de survoler le lien avec ta souris. Tu verras la couleur passer au rouge.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:hover {
  color: red;
}
```

:::

Ensuite, il y a `:focus`. Cet état s'applique quand on met le focus sur un lien.

Clique sur une zone vide de la fenêtre d'aperçu puis appuie sur `tab` sur ton clavier. Tu devrais voir le lien devenir vert. Pour voir les aperçus, tu dois activer l'éditeur interactif.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:focus {
  color: green;
}
```

:::

Enfin, il y a `:active`. Cet état s'applique aux liens qui sont activés par l'utilisateur. En général, cela signifie cliquer sur le lien avec le bouton principal de la souris, le plus souvent par un clic gauche. Cet état peut aider à montrer à l'utilisateur que l'élément sur lequel il a cliqué est interactif.

Active l'éditeur interactif et clique sur le lien. Tu devrais voir le lien devenir noir. Cela arrive très vite, donc tu devras peut-être cliquer plusieurs fois pour voir la couleur changer.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:active {
  color: black;
}
```

:::

Quand tu utilises ces états pour styliser tes liens, il y a un ordre précis dans lequel écrire ton CSS : `link`, `visited`, `hover`, `focus`, puis `active`.

Tu sais maintenant comment donner ton propre style aux liens de ta page tout en fournissant à l'utilisateur les informations importantes sur l'état de chaque lien.
