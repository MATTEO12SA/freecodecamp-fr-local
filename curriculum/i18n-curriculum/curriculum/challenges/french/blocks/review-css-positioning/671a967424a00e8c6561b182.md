# --interactive--

## Travailler avec les flottants

- **Définition** : Les flottants servent à retirer un élément de son flux normal sur la page et à le positionner soit à gauche, soit à droite de son conteneur. Quand cela se produit, le texte s'enroule autour de ce contenu flottant.

```css
float: left;
float: right;
```

- **Nettoyer les flottants** : La propriété `clear` sert à déterminer si un élément doit être déplacé sous le contenu flottant. Quand plusieurs éléments flottants sont empilés les uns à côté des autres, il peut y avoir des problèmes de chevauchement et d'effondrement dans les mises en page. Un hack `clearfix` a donc été créé pour corriger ce problème.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="clearfix">
  <div class="box left">Left</div>
  <div class="box right">Right</div>
</div>
```

```css
.box {
  width: 100px;
  height: 60px;
  color: white;
  text-align: center;
  line-height: 60px;
}

.left {
  float: left;
  background: teal;
}

.right {
  float: right;
  background: purple;
}

.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
```

:::

## Positionnements statique, relatif et absolu

- **Positionnement statique** : C'est le flux normal du document. Les éléments sont positionnés de haut en bas et de gauche à droite, les uns après les autres.
- **Positionnement relatif** : Il te permet d'utiliser les propriétés `top`, `left`, `right` et `bottom` pour positionner l'élément dans le flux normal du document. Tu peux aussi utiliser le positionnement relatif pour faire chevaucher des éléments avec d'autres éléments sur la page.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="relative">Relative Element</div>
```

```css
.relative {
    position: relative;
    top: 30px;
    left: 30px;
}
```

:::

- **Positionnement absolu** : Il te permet de retirer un élément du flux normal du document, ce qui le fait se comporter indépendamment des autres éléments.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  Parent
  <div class="positioned">Absolute</div>
</div>
```

```css
.parent {
  position: relative;
  height: 120px;
  background: #eee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

## Positionnements fixe et sticky

- **Positionnement fixe** : Quand un élément est positionné avec `position: fixed`, il est retiré du flux normal du document et placé par rapport au viewport, ce qui signifie qu'il reste à la même position même quand l'utilisateur fait défiler la page. C'est souvent utilisé pour des éléments comme des en-têtes ou des barres de navigation qui doivent rester visibles en permanence.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="navbar">Fixed Navbar</div>
<div style="height: 600px;"></div>
```

```css
.navbar {
  position: fixed;
  top: 0;
  width: 100%;
  background: #0077ff;
  color: white;
  padding: 8px;
}
```

:::

- **Positionnement sticky** : Ce type de positionnement agit comme un hybride entre les positionnements relatif et fixe. Au départ, l'élément se comporte comme s'il était positionné de manière relative, en restant dans le flux du document. Cependant, dès que l'utilisateur fait défiler l'élément au-delà d'un certain point, il « colle » au viewport (généralement en haut) et se comporte comme s'il était fixe. C'est idéal pour créer des éléments comme des barres de navigation sticky, qui ne deviennent fixes qu'une fois que l'utilisateur a fait défiler jusqu'à une certaine position.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Scroll down</p>
<div class="positioned">Sticky Element</div>
<div style="height: 500px;"></div>
```

```css
.positioned {
  position: sticky;
  top: 30px;
  left: 30px;
  background: #4caf50;
  color: white;
  padding: 6px;
}
```

:::

## Travailler avec la propriété `z-index`

- **Définition** : La propriété `z-index` en CSS sert à contrôler l'ordre d'empilement vertical des éléments positionnés qui se chevauchent sur la page.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box box1">1</div>
  <div class="box box2">2</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box {
  position: absolute;
  width: 100px;
  height: 100px;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

.box1 {
  background: lightcoral;
  top: 20px;
  left: 20px;
  z-index: 1;
}

.box2 {
  background: steelblue;
  top: 40px;
  left: 40px;
  z-index: 2;
}
```
