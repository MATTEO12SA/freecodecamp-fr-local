# --interactive--

Quand on travaille avec le CSS pour styliser une page web, l'une des méthodes les plus courantes pour définir les couleurs est d'utiliser les valeurs de couleur hex. Les valeurs de couleur hexadécimales, ou codes hex, sont une façon concise de représenter les couleurs dans le modèle de couleur RGB. Dans cette leçon, on va explorer ce que sont les codes hex, comment ils fonctionnent en CSS, et quand les utiliser.

Un code hex (abréviation de code hexadécimal) est une chaîne de six caractères utilisée pour représenter les couleurs dans le modèle de couleur RGB. Le « hex » fait référence au système de numérotation en base 16, qui utilise les chiffres de `0` à `9` et les lettres `A` à `F`.

Dans le contexte des couleurs, les codes hex spécifient les quantités de rouge, vert et bleu (RGB) qui composent une couleur particulière. Chaque code hex commence par un symbole dièse (`#`), suivi de six caractères qui définissent l'intensité du rouge, du vert et du bleu dans la couleur.

Par exemple, le code hex pour le noir est `#000000`, et le code hex pour le blanc est `#FFFFFF`. Les deux premiers caractères représentent la valeur du rouge, les deux suivants représentent le vert, et les deux derniers représentent le bleu.

En CSS, tu peux appliquer des couleurs aux éléments en utilisant les codes hex. Voici la syntaxe de base :

```css
element {
  color: #RRGGBB;
}
```

Ici, `#RRGGBB` est un placeholder pour le code hex réel. Les paires `RR`, `GG` et `BB` représentent chacune l'intensité du rouge, du vert et du bleu, respectivement.

Ces paires peuvent varier de `00` (l'intensité la plus basse) à `FF` (l'intensité la plus élevée). Plus le nombre est élevé, plus cette couleur sera présente dans le mélange final. Par exemple, appliquons quelques couleurs hex à quelques éléments :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>This is a heading.</h1>
<p>This is a paragraph.</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}
p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

Dans le premier cas, `#FF5733` est une couleur rouge-orangé. Voici la décomposition :

- `FF` est la valeur la plus élevée pour le rouge (entièrement saturé).
- `57` est une quantité modérée de vert.
- `33` est une petite quantité de bleu.

Dans le second cas, `#4CAF50` est une nuance de vert :

- `4C` est une faible valeur pour le rouge.
- `AF` est une forte valeur pour le vert.
- `50` est une valeur modérée pour le bleu.

Les codes hex sont populaires parce qu'ils sont précis et faciles à utiliser. Ils te permettent d'affiner tes couleurs, ce qui les rend idéaux pour les designs où la cohérence est importante.

La plupart des logiciels de design, comme Adobe Photoshop ou Figma, fournissent des codes hex pour les couleurs que tu sélectionnes, ce qui rend facile de copier-coller ces valeurs directement dans tes fichiers CSS.

Dans le développement web, les codes hex sont souvent utilisés pour définir les couleurs de texte, les arrière-plans, les bordures et d'autres éléments visuels. Par exemple :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example</button>
```

```css
body {
  background-color: #F0F0F0; /* Light gray background */
}

button {
  background-color: #007BFF; /* A shade of blue */
  color: #FFFFFF; /* White text */
}
```

:::

Cela garantit que tes couleurs restent cohérentes sur tout ton site web ou application.

Dans certains cas, tu peux remarquer des codes hex écrits au format raccourci, en utilisant seulement trois caractères au lieu de six. C'est possible quand les deux caractères de chaque paire de couleur sont les mêmes. Par exemple, `#FF5733` ne peut pas être raccourci, mais `#FFFFFF` (blanc) peut être écrit comme `#FFF`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h2>This is a heading.</h2>
```

```css
body {
  background-color: #000;
}

h2 {
  color: #FFF; /* White in shorthand */
}
```

:::

Le raccourci `#FFF` équivaut à `#FFFFFF`. De même, `#000` représenterait le noir qui est `#000000` en forme complète.

Les codes hex sont populaires pour leur précision et leur compatibilité, permettant plus de 16 millions de couleurs sur tous les navigateurs. Ils sont idéaux quand tu as besoin de couleurs de marque spécifiques, quand tu veux contrôler les nuances exactes qui ne sont pas facilement définies par les couleurs nommées ou HSL, ou quand tu copies des couleurs d'un logiciel de design qui fournit des codes hex par défaut.
