# --interactive--

Dans une leçon précédente, tu as appris comment commencer à travailler avec les dégradés linéaires et radiaux. Dans cette leçon, on va plonger plus profondément dans comment travailler avec les propriétés CSS pour créer des designs plus visuellement attrayants et complexes.

Les dégradés en CSS te permettent de créer des transitions douces entre deux couleurs ou plus. Ils sont souvent utilisés pour ajouter de la profondeur, de la texture et de l'intérêt visuel aux designs web sans avoir besoin de fichiers image.

Le CSS prend en charge deux principaux types de dégradés : les dégradés linéaires et les dégradés radiaux.

Les dégradés linéaires créent un mélange graduel entre les couleurs le long d'une ligne droite. Tu peux contrôler la direction de cette ligne et les couleurs utilisées. Voici un exemple basique de dégradé linéaire :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

Cela crée un dégradé qui passe du rouge à gauche au bleu à droite. Le paramètre `to right` spécifie la direction du dégradé. Tu peux utiliser d'autres directions comme `to left`, `to top`, `to bottom`, ou même spécifier des angles comme `45deg`.

Tu peux aussi utiliser plusieurs arrêts de couleur dans un dégradé linéaire :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="multi-color-gradient"></div>
```

```css
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  height: 40vh;
}
```

:::

Cela crée un dégradé qui passe du rouge au jaune au bleu, espacés uniformément sur l'élément.

Les dégradés radiaux, par contre, créent des dégradés circulaires ou elliptiques qui rayonnent à partir d'un point central. Voici un exemple :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::

Cela crée un dégradé circulaire qui commence avec du rouge au centre et passe au bleu aux bords. Tu peux aussi spécifier la forme et la taille des dégradés radiaux :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="custom-radial-gradient"></div>
```

```css
.custom-radial-gradient {
  background: radial-gradient(ellipse at top left, red, blue);
  height: 40vh;
}
```

:::

Cela crée un dégradé elliptique commençant depuis le coin supérieur gauche de l'élément. Les dégradés linéaires et radiaux te permettent tous deux de spécifier des arrêts de couleur à des positions précises. Par exemple :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="precise-gradient"></div>
```

```css
.precise-gradient {
  background: linear-gradient(to right, red 20%, yellow 40%, blue 80%);
  height: 40vh;
}
```

:::

Cela crée un dégradé où le rouge prend les premiers `20%`, puis passe au jaune à `40%`, et finalement au bleu à `80%`.

Les dégradés peuvent être utilisés partout où tu utiliserais une image en CSS. Cela les rend polyvalents pour créer des arrière-plans, des boutons, ou même des effets de texte combinés avec d'autres propriétés CSS.

Rappelle-toi, bien que les dégradés puissent améliorer l'attrait visuel de ton site web, il est important d'assurer un contraste suffisant pour l'accessibilité, surtout quand on utilise des dégradés derrière le texte.

## --text--

Quelle est la différence entre les dégradés linéaires et radiaux en CSS ?

## --answers--

Les dégradés linéaires utilisent seulement deux couleurs, tandis que les dégradés radiaux utilisent plusieurs couleurs.
