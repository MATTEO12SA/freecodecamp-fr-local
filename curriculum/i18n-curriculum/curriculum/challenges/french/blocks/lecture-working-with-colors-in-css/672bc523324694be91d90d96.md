# --interactive--

Quand on stylise des sites web, l'une des façons les plus polyvalentes et intuitives de travailler avec la couleur est d'utiliser le modèle de couleur HSL. HSL signifie Hue, Saturation et Lightness — trois composants clés qui définissent une couleur.

Le modèle de couleur HSL représente les couleurs d'une manière plus alignée avec la façon dont les humains perçoivent la couleur. Dans cette leçon, on va explorer ce qu'est le modèle de couleur HSL et comment la fonction `hsl()` est utilisée en CSS.

La teinte (hue) est le type de couleur, représenté comme un angle sur la roue chromatique, qui varie de `0` à `360` degrés. Par exemple, `0` degrés représente le rouge, `120` degrés est le vert, et `240` degrés est le bleu. En ajustant la teinte, tu te déplaces à travers différentes couleurs du spectre.

La saturation fait référence à l'intensité ou à la pureté de la couleur. Elle est mesurée en pourcentage de `0%` (une couleur complètement désaturée, en niveaux de gris) à `100%` (la forme la plus vibrante de cette couleur). Une couleur avec `100%` de saturation est entièrement vive, tandis qu'une couleur avec `0%` de saturation apparaît comme une nuance de gris.

La luminosité (lightness) détermine à quel point la couleur est claire ou sombre, à nouveau mesurée en pourcentage. Une valeur de luminosité de `0%` produira du noir, `50%` te donnera la teinte normale de la teinte, et `100%` donnera du blanc.

En CSS, la fonction `hsl()` est utilisée pour définir les couleurs en utilisant le modèle de couleur HSL. Voici la syntaxe de base :

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

Décomposons cela avec un exemple :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsla(0, 0%, 1%, 1.00);
}

p {
  color: hsl(120, 100%, 50%);
}
```

:::

Dans ce cas, la teinte est `120` degrés, ce qui correspond au vert. La saturation est `100%`, donc le vert sera complètement vif. La luminosité est `50%`, donc elle est à sa teinte normale — ni trop sombre ni trop claire. Comme résultat, la couleur du texte du paragraphe sera un vert vif.

L'un des principaux avantages du modèle de couleur HSL est sa nature intuitive. Il rend facile d'ajuster l'éclat ou la luminosité d'une couleur en modifiant les valeurs de saturation et de luminosité sans avoir à modifier la couleur de base (teinte).

Par exemple, si tu veux créer différentes nuances ou teintes de la même couleur, tu peux simplement ajuster la valeur de luminosité.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240, 100%, 80%);
}

div.dark {
  background-color: hsl(240, 100%, 20%);
  color: hsl(0, 0%, 100%);
}
```

:::

Ici, les deux éléments `div` utilisent la même teinte (`240` degrés, qui est le bleu), mais l'un a une luminosité de `80%` (une nuance plus claire de bleu), et l'autre a une luminosité de `20%` (une nuance plus sombre de bleu).

Tout comme le modèle RGB a une fonction `rgba()` pour inclure la transparence, le modèle HSL a une fonction `hsla()`. Le quatrième paramètre dans cette fonction représente la valeur alpha, qui contrôle l'opacité de la couleur. Voici la syntaxe de base :

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

Regardons un exemple :

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

Ce code donnerait au `div` un arrière-plan rouge semi-transparent, où la teinte est définie à `0` degrés (rouge), la saturation est `100%`, la luminosité est `50%`, et l'alpha est `0.5` (50% d'opacité). Tu peux aussi spécifier l'alpha en pourcentage — par exemple, `hsla(0, 100%, 50%, 50%)` équivaut à `hsla(0, 100%, 50%, 0.5)`.

Le modèle de couleur HSL est particulièrement utile quand tu as besoin de créer des schémas de couleurs et d'ajuster les nuances ou teintes facilement.

Parce que HSL sépare la teinte de la saturation et de la luminosité, tu peux maintenir un thème de couleur cohérent tout en modifiant la luminosité pour les ombres ou les surbrillances. Ce niveau de flexibilité fait de HSL un favori parmi les designers et développeurs.

De plus, HSL rend les valeurs de couleur plus lisibles et compréhensibles comparées à d'autres méthodes comme les codes hex ou RGB. Tu peux voir plus clairement l'intention derrière la couleur et l'ajuster en conséquence, surtout quand tu expérimentes avec différents designs.

Avec le modèle de couleur HSL, tu gagnes plus de flexibilité dans le contrôle des tons, de la saturation et de la luminosité, ce qui en fait un outil puissant pour créer des designs riches et dynamiques en CSS.
