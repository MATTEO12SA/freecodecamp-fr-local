# --description--

Le protocole Open Graph te permet de contrôler la façon dont le contenu de ton site apparaît sur différentes plateformes de réseaux sociaux, comme Facebook, LinkedIn et beaucoup d'autres. En définissant ces propriétés Open Graph, tu peux donner envie aux utilisateurs de cliquer et d'interagir avec ton contenu. Tu peux définir ces propriétés avec une collection d'éléments `meta` dans la section `head` de ton HTML.

La première propriété OG importante à inclure est `title`. Voici un exemple qui définit le `title` OG pour la page d'accueil de freeCodeCamp :

```html
<meta content="freeCodeCamp.org" property="og:title" />
```

Pour l'attribut `property`, tu dois préciser que sa valeur est `og:title`. L'attribut `content` est l'endroit où tu écris le titre que tu veux afficher sur les sites de réseaux sociaux.

La propriété OG importante suivante est `type`. Voici un exemple d'utilisation du `type` OG pour la page d'accueil de freeCodeCamp :

```html
<meta property="og:type" content="website" />
```

La propriété `type` sert à représenter le type de contenu partagé sur les réseaux sociaux. Ce contenu peut être, par exemple, un article, un site web, une vidéo ou de la musique.

La troisième propriété OG importante est `image`. Voici un exemple qui définit l'`image` OG pour la page d'accueil de freeCodeCamp :

```html
<meta
  content="https://cdn.freecodecamp.org/platform/universal/fcc_meta_1920X1080-indigo.png"
  property="og:image"
/>
```

Dans cet exemple, l'image Open Graph pointe vers le logo freeCodeCamp. Toutes ces images doivent être de haute qualité, avec de bonnes dimensions et de bons ratios. La plupart des plateformes de réseaux sociaux donnent des critères pour les images afin de t'aider à vérifier que ton contenu s'affiche bien sur leur site. Par exemple, la documentation developers.facebook.com indique :

"use images that are at least 1200 by 630 pixels for the best display on high resolution devices. At the minimum, you should use images that are 600 by 315 pixels to display link page posts with larger images."

La quatrième propriété OG importante est `url`. Voici un exemple qui définit l'`url` OG pour la page d'accueil de freeCodeCamp :

```html
<meta property="og:url" content="https://www.freecodecamp.org" />
```

Il existe beaucoup d'autres propriétés OG que tu peux définir, comme `description`, `audio`, `video` et `locale`. Cependant, les propriétés Open Graph `url`, `image`, `type` et `title` sont les plus importantes à inclure.

Alors, quel effet ces propriétés Open Graph ont-elles sur le Search Engine Optimization ? Quand ton contenu est partagé sur les réseaux sociaux, des propriétés OG bien rédigées peuvent améliorer l'apparence de ton contenu dans les fils des utilisateurs. Cela peut augmenter le taux de clics, ce qui peut signaler aux moteurs de recherche que ton contenu est pertinent et intéressant.
