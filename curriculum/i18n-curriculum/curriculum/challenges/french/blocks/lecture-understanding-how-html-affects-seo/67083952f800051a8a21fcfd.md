# --description--

Le SEO, ou Search Engine Optimization, est une pratique qui optimise les pages web pour qu'elles soient plus visibles et mieux classées dans les moteurs de recherche. Une façon d'améliorer le SEO de ton site consiste à fournir une courte description de la page web avec l'élément `meta`. Voici un exemple d'utilisation de l'élément `meta` pour définir la description d'une page de jardinage :

```html
<meta
  name="description"
  content="Discover expert tips and techniques for gardening in small spaces, choosing the right plants, and maintaining a thriving garden."
/>
```

En définissant l'attribut `name` sur `description`, tu t'assures que les navigateurs, les moteurs de recherche et les autres outils web interprètent correctement cette métadonnée. L'attribut `content` est l'endroit où tu places ta description. Il est recommandé de garder tes descriptions courtes et concises, car les moteurs de recherche les coupent souvent selon la mise en page des résultats.

Comme les autres types d'éléments `meta`, la meta description ne sera pas visible sur la page web elle-même. Un endroit où la description de la page peut apparaître est l'extrait affiché dans les résultats de recherche. Voici quelques exemples d'extraits de résultats pour le subreddit et les dépôts GitHub de freeCodeCamp :

```sh
r/FreeCodeCamp: This is the official subreddit for the freeCodeCamp.org community. Learn to
code for free together with millions of other people...
```

```sh
Our full-stack web development and machine learning curriculum is completely free and self-
paced. We have thousands of interactive coding challenges to help you...
```

Dans ces exemples, chaque description de page se trouve juste sous les liens du site. En quelques secondes, les utilisateurs peuvent comprendre globalement de quoi parle la page et décider de cliquer sur les liens pour en savoir plus.

Même si les meta descriptions n'affectent pas directement le classement d'un site dans un moteur de recherche, une description solide peut attirer plus de trafic vers ton site web.
